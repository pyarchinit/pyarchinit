# s3dgraphy bridge

Bidirectional bridge between PyArchInit and the s3dgraphy Extended Matrix data model. Includes the `sync/` layer (Phase 2: AI03 → AI08-F2) covering GraphProjector, GraphIngestor, ParadataStore, GroupStore, vocabularies, edge styling and per-dimension visual rules.
