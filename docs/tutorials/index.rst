About PyArchInit
================

**PyArchInit** is a comprehensive QGIS plugin for archaeological data management. It provides tools for:

- **Stratigraphic Documentation**: US/USM forms with Harris Matrix generation
- **Finds Management**: Complete inventory system with image handling
- **Spatial Analysis**: GIS integration with SAM segmentation
- **Reporting**: PDF generation for all record types
- **Multi-user Support**: PostgreSQL/PostGIS for collaborative work
- **AI Integration**: Text-to-SQL queries and automated analysis

System Requirements
-------------------

- QGIS 3.22 or later
- Python 3.8+
- PostgreSQL/PostGIS or SQLite/Spatialite database
- For AI features: OpenAI API key or Ollama installation

Getting Started
---------------

1. Install PyArchInit from QGIS Plugin Manager
2. Follow Tutorial 01: Configuration
3. Set up your database connection
4. Start documenting your archaeological site!
