## Generazione PDF / PDF Generation

### Requisiti / Requirements

1. **Python dependencies**:
   ```bash
   pip install -r requirements.txt
   # oppure / or
   python build_docs.py install
   ```

2. **LaTeX** (per PDF):
   - macOS: `brew install --cask mactex`
   - Ubuntu: `sudo apt-get install texlive-xetex texlive-latex-extra latexmk`
   - Windows: Install [MiKTeX](https://miktex.org/)

### Build Commands

**Using Python script (cross-platform)**:
```bash
# Build HTML documentation
python build_docs.py html

# Build all PDFs
python build_docs.py pdf

# Build specific language PDF
python build_docs.py pdf-it    # Italian
python build_docs.py pdf-en    # English
python build_docs.py pdf-de    # German
python build_docs.py pdf-fr    # French
python build_docs.py pdf-es    # Spanish
python build_docs.py pdf-ar    # Arabic
python build_docs.py pdf-ca    # Catalan

# Clean build directory
python build_docs.py clean
```

**Using Make (Linux/macOS)**:
```bash
make html        # HTML documentation
make pdf         # PDF documentation
make pdf-it      # Italian PDF only
make clean       # Clean build
```

**Using make.bat (Windows)**:
```batch
make.bat html
make.bat pdf
make.bat clean
```

### Output

- HTML: `_build/html/index.html`
- PDF: `_build/latex/[language]/PyArchInit_Tutorials_[LANG].pdf`
