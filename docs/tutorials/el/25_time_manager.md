# Οδηγός 25: Time Manager (GIS Time Controller)

## Εισαγωγή

Ο **Time Manager** (GIS Time Controller) είναι ένα προηγμένο εργαλείο για την οπτικοποίηση της στρωματογραφικής ακολουθίας σε χρονική εξέλιξη. Επιτρέπει την «πλοήγηση» μέσα στα στρωματογραφικά επίπεδα χρησιμοποιώντας ένα χρονικό στοιχείο ελέγχου, εμφανίζοντας σταδιακά τις ΣΜ από τις πιο πρόσφατες προς τις παλαιότερες.

### Κύρια Χαρακτηριστικά

- Σταδιακή οπτικοποίηση στρωματογραφικών επιπέδων
- Έλεγχος μέσω επιλογέα/ρυθμιστή
- Λειτουργία αθροιστική ή μεμονωμένου επιπέδου
- Αυτόματη δημιουργία εικόνων/βίντεο
- Ολοκλήρωση με Harris Matrix

## Πρόσβαση

### Από το Μενού
**PyArchInit** → **Time Manager**

### Προαπαιτούμενα

- Επίπεδο με πεδίο `order_layer` (στρωματογραφικός δείκτης)
- ΣΜ με συμπληρωμένο order_layer
- Επίπεδα φορτωμένα στο QGIS

### Στοιχεία Ελέγχου

| Στοιχείο Ελέγχου | Λειτουργία |
|-------------------|-----------|
| Layer Checkbox | Επιλογή επιπέδων προς έλεγχο |
| Dial | Πλοήγηση μεταξύ επιπέδων (περιστροφή) |
| SpinBox | Άμεση εισαγωγή επιπέδου |
| Cumulative Mode | Εμφάνιση όλων των επιπέδων μέχρι το επιλεγμένο |
| Show Matrix | Εμφάνιση συγχρονισμένου Harris Matrix |

## Πεδίο order_layer

### Τι είναι το order_layer;

Το πεδίο `order_layer` ορίζει τη σειρά εμφάνισης της στρωματογραφίας:
- **1** = Πιο πρόσφατο επίπεδο (επιφάνεια)
- **N** = Παλαιότερο επίπεδο (βάθος)

### Συμπλήρωση του order_layer

Στη Φόρμα ΣΜ, πεδίο **"Stratigraphic Index"**:
1. Αντιστοιχίστε αυξανόμενες τιμές από την επιφάνεια
2. Σύγχρονες ΣΜ μπορούν να έχουν την ίδια τιμή
3. Ακολουθήστε την ακολουθία του Matrix

### Παράδειγμα

| ΣΜ | order_layer | Περιγραφή |
|----|-------------|-----------|
| SU001 | 1 | Επιφανειακό χούμος |
| SU002 | 2 | Οργωμένο στρώμα |
| SU003 | 3 | Κατάρρευση |
| SU004 | 4 | Δάπεδο χρήσης |
| SU005 | 5 | Θεμελίωση |

## Λειτουργίες Οπτικοποίησης

### Λειτουργία Μεμονωμένου Επιπέδου

Πλαίσιο ελέγχου **ΜΗ** ενεργοποιημένο:
- Εμφανίζει ΜΟΝΟ τις ΣΜ του επιλεγμένου επιπέδου
- Χρήσιμο για απομόνωση μεμονωμένων στρωμάτων
- Οπτικοποίηση «τομής»

### Αθροιστική Λειτουργία

Πλαίσιο ελέγχου **ΕΝΕΡΓΟ**:
- Εμφανίζει όλες τις ΣΜ μέχρι το επιλεγμένο επίπεδο
- Προσομοιώνει σταδιακή ανασκαφή
- Πιο ρεαλιστική οπτικοποίηση

## Ολοκλήρωση με Matrix

### Συγχρονισμένη Οπτικοποίηση

Με ενεργοποιημένο το πλαίσιο ελέγχου **"Show Matrix"**:
- Το Harris Matrix εμφανίζεται στον πίνακα
- Ενημερώνεται συγχρονισμένα με το επίπεδο
- Επισημαίνει τις ΣΜ του τρέχοντος επιπέδου

### Δημιουργία Εικόνων

Ο Time Manager μπορεί να δημιουργήσει:
- Στιγμιότυπο οθόνης για κάθε επίπεδο
- Ακολουθία εικόνων
- Βίντεο time-lapse

## Δημιουργία Βίντεο/Εικόνων

### Διαδικασία

1. Επιλέξτε τα επίπεδα προς συμπερίληψη
2. Ρυθμίστε το εύρος επιπέδων (ελάχ.-μέγ.)
3. Κάντε κλικ στο **"Generate Video"**
4. Περιμένετε την επεξεργασία
5. Η έξοδος αποθηκεύεται στον καθορισμένο φάκελο

### Έξοδος

- Εικόνες PNG για κάθε επίπεδο
- Προαιρετικά: συμπιεσμένο βίντεο MP4

## Τυπική Ροή Εργασίας

```

```

### 2. Επιλογή Επιπέδων

```

### 3. Πλοήγηση

```
1. Χρησιμοποιήστε τον επιλογέα ή το spinbox

## Βέλτιστες Πρακτικές

### 1. order_layer

- Συμπληρώστε ΠΡΙΝ χρησιμοποιήσετε τον Time Manager
- Χρησιμοποιήστε διαδοχικές τιμές
- Σύγχρονες ΣΜ = ίδια τιμή

### 2. Οπτικοποίηση

- Ξεκινήστε από το επίπεδο 1 (επιφάνεια)
- Προχωρήστε σε αύξουσα σειρά
- Χρησιμοποιήστε αθροιστική λειτουργία για παρουσιάσεις

### 3. Τεκμηρίωση

- Αποτυπώστε στιγμιότυπα στα σημαντικά επίπεδα
- Τεκμηριώστε τις μεταβάσεις φάσεων
- Δημιουργήστε βίντεο για αναφορές

## Αντιμετώπιση Προβλημάτων

### Τα Επίπεδα Δεν Εμφανίζονται στη Λίστα

**Αιτία**: Επίπεδο χωρίς πεδίο order_layer

**Λύση**:
- Προσθέστε πεδίο order_layer στο επίπεδο
- Συμπληρώστε με τις κατάλληλες τιμές

### Καμία Οπτική Αλλαγή

**Αιτίες**:
- Το order_layer δεν είναι συμπληρωμένο
- Το φίλτρο δεν εφαρμόστηκε

**Λύσεις**:
- Επαληθεύστε τις τιμές order_layer στις ΣΜ
- Ελέγξτε ότι το επίπεδο είναι επιλεγμένο

### Ο Επιλογέας Δεν Ανταποκρίνεται

**Αιτία**: Δεν έχει επιλεγεί κανένα επίπεδο

**Λύση**: Επιλέξτε τουλάχιστον ένα επίπεδο από τη λίστα
