# Οδηγός 23: Αναζήτηση Εικόνων

## Εισαγωγή

Η λειτουργία **Αναζήτηση Εικόνων** σας επιτρέπει να αναζητάτε γρήγορα εικόνες στη βάση δεδομένων του PyArchInit, φιλτράροντας κατά θέση, τύπο οντότητας και άλλα κριτήρια. Αποτελεί ένα συμπληρωματικό εργαλείο του Media Manager για καθολική αναζήτηση.

## Πρόσβαση

### Από το Μενού
**PyArchInit** → **Image Search**

## Διεπαφή

### Πίνακας Αναζήτησης

```
+--------------------------------------------------+
|           Image Search                            |
+--------------------------------------------------+
| Φίλτρα:                                           |
|   Site: [ComboBox]                               |
|   Entity Type: [-- All -- | SU | Pottery | ...]  |
|   [ ] Μόνο εικόνες χωρίς ετικέτα                 |
+--------------------------------------------------+
| [Search]  [Clear]                                |
+--------------------------------------------------+
| Αποτελέσματα:                                     |
|  +------+  +------+  +------+                    |
|  | img  |  | img  |  | img  |                    |
|  | info |  | info |  | info |                    |
|  +------+  +------+  +------+                    |
+--------------------------------------------------+
| [Open Image] [Export] [Go to Record]             |
+--------------------------------------------------+
```

### Διαθέσιμα Φίλτρα

| Φίλτρο | Περιγραφή |
|--------|-----------|
| Site | Επιλογή συγκεκριμένης θέσης ή όλων |
| Entity Type | SU, Pottery, Materials, Grave, Structure, TU |
| Only untagged | Εμφάνιση μόνο εικόνων χωρίς συνδέσεις |

### Τύποι Οντοτήτων

| Τύπος | Περιγραφή |
|-------|-----------|
| -- All -- | Όλες οι οντότητες |
| SU | Στρωματογραφικές Μονάδες |
| Pottery | Κεραμική |
| Materials | Ευρήματα/Κατάλογος |
| Grave | Ταφές |
| Structure | Κατασκευές |
| TU | Τοπογραφικές Μονάδες |

## Λειτουργίες

### Βασική Αναζήτηση

1. Επιλέξτε τα επιθυμητά φίλτρα
2. Κάντε κλικ στο **"Search"**
3. Δείτε τα αποτελέσματα στο πλέγμα

### Ενέργειες στα Αποτελέσματα

| Κουμπί | Λειτουργία |
|--------|-----------|
| Open Image | Προβολή εικόνας σε πλήρες μέγεθος |
| Export | Εξαγωγή επιλεγμένης εικόνας |
| Go to Record | Άνοιγμα της φόρμας συνδεδεμένης οντότητας |
| Open Media Manager | Άνοιγμα του Media Manager με την επιλεγμένη εικόνα |

### Μενού Περιβάλλοντος (Δεξί κλικ)

- **Άνοιγμα εικόνας**
- **Εξαγωγή εικόνας...**
- **Μετάβαση στην εγγραφή**

### Αναζήτηση Εικόνων χωρίς Ετικέτα

Πλαίσιο ελέγχου **"Only untagged images"**:
- Εντοπίζει εικόνες στη βάση δεδομένων χωρίς συνδέσεις
- Χρήσιμο για καθαρισμό και οργάνωση
- Επιτρέπει τον εντοπισμό εικόνων προς καταλογογράφηση

## Τυπική Ροή Εργασίας

### 1. Εύρεση Εικόνων Θέσης

```
1. Επιλέξτε θέση από το ComboBox
2. Αφήστε "-- All --" για τον τύπο οντότητας
3. Κάντε κλικ στο Search
4. Περιηγηθείτε στα αποτελέσματα
```

### 2. Εύρεση Εικόνων Συγκεκριμένης ΣΜ

```
1. Επιλέξτε θέση
2. Επιλέξτε "SU" ως τύπο οντότητας
3. Κάντε κλικ στο Search
4. Διπλό κλικ για άνοιγμα εικόνας
```

### 3. Εντοπισμός Μη Καταλογογραφημένων Εικόνων

```
1. Επιλέξτε θέση (ή όλες)
2. Ενεργοποιήστε "Only untagged images"
3. Κάντε κλικ στο Search
4. Για κάθε αποτέλεσμα:
   - Ανοίξτε την εικόνα
   - Αναγνωρίστε το περιεχόμενο
   - Συνδέστε μέσω Media Manager
```

## Εξαγωγή

### Εξαγωγή Μεμονωμένης Εικόνας

1. Επιλέξτε εικόνα στα αποτελέσματα
2. Κάντε κλικ στο **"Export"** ή στο μενού περιβάλλοντος
3. Επιλέξτε προορισμό
4. Αποθηκεύστε

### Μαζική Εξαγωγή

Για εξαγωγή πολλαπλών εικόνων, χρησιμοποιήστε την ειδική λειτουργία **Εξαγωγή Εικόνων** (Οδηγός 24).

## Βέλτιστες Πρακτικές

### 1. Αποδοτική Αναζήτηση

- Χρησιμοποιήστε συγκεκριμένα φίλτρα για στοχευμένα αποτελέσματα
- Ξεκινήστε με ευρεία φίλτρα, στη συνέχεια περιορίστε
- Χρησιμοποιήστε περιοδικά την αναζήτηση χωρίς ετικέτα

### 2. Οργάνωση

- Καταλογογραφήστε τακτικά τις εικόνες χωρίς ετικέτα
- Επαληθεύστε τις συνδέσεις μετά την εισαγωγή
- Διατηρήστε συνεπή ονοματολογία

## Αντιμετώπιση Προβλημάτων

### Κανένα Αποτέλεσμα

**Αιτίες**:
- Υπερβολικά περιοριστικά φίλτρα
- Δεν υπάρχουν εικόνες για τα κριτήρια

**Λύσεις**:
- Διευρύνετε τα φίλτρα
- Επαληθεύστε ότι υπάρχουν δεδομένα

### Η Εικόνα Δεν Εμφανίζεται

**Αιτίες**:
- Το αρχείο δεν βρέθηκε
- Μη υποστηριζόμενη μορφή

**Λύσεις**:
- Επαληθεύστε τη διαδρομή αρχείου
- Ελέγξτε τη μορφή εικόνας
