## Εισαγωγη

Το module **Διαχειριση Εργοταξιου** (Site Management) του PyArchInit παρεχει ενα ολοκληρωμενο συνολο εργαλειων για τη διαχειριση των λειτουργικων πτυχων ενος αρχαιολογικου εργοταξιου. Σε αντιθεση με τα αλλα δελτια που εστιαζουν στην επιστημονικη τεκμηριωση (ΣΜ, κατασκευες, ευρηματα), αυτο το module καλυπτει τη διαχειριση πορων: προσωπικο, παρουσιες, εξοπλισμο και προυπολογισμο.

Το module αποτελειται απο **5 συστατικα**:

| Συστατικο | Περιγραφη |
|-----------|-----------|
| **Πινακας Εργοταξιου** | Κεντρικος πινακας ελεγχου (dashboard) με συνοπτικη εικονα ολων των πορων |
| **Προσωπικο** | Δελτιο CRUD για τη διαχειριση μελων του προσωπικου |
| **Παρουσιες** | Δελτιο CRUD για την καταγραφη ημερησιων παρουσιων και ωρων εργασιας |
| **Εξοπλισμος** | Δελτιο CRUD για τη διαχειριση εξοπλισμου και συντηρησεων |
| **Προυπολογισμος** | Δελτιο CRUD για τον ελεγχο κονδυλιων και δαπανων |

## Προσβαση στο module

### Απο τη γραμμη εργαλειων

Το module ειναι προσβασιμο απο τη γραμμη εργαλειων **Διαχειριση Εργοταξιου** του PyArchInit, η οποια περιεχει 5 εικονιδια:

| Εικονιδιο | Λειτουργια | Περιγραφη |
|-----------|------------|-----------|
| Πινακας | **Πινακας Εργοταξιου** | Ανοιγει τον κεντρικο πινακα ελεγχου |
| Ατομο | **Προσωπικο** | Ανοιγει τη φορμα προσωπικου |
| Ημερολογιο | **Παρουσιες** | Ανοιγει τη φορμα παρουσιων |
| Εργαλειο | **Εξοπλισμος** | Ανοιγει τη φορμα εξοπλισμου |
| Πορτοφολι | **Προυπολογισμος** | Ανοιγει τη φορμα προυπολογισμου |

<!-- IMAGE: Γραμμη εργαλειων Διαχειρισης Εργοταξιου με τα 5 εικονιδια σημειωμενα -->
> **Εικ. 2**: Η γραμμη εργαλειων Διαχειρισης Εργοταξιου στο PyArchInit

### Προαπαιτουμενα

Πριν χρησιμοποιησετε το module, βεβαιωθειτε οτι:

1. Εχετε ρυθμισει τη συνδεση βασης δεδομενων (SQLite η PostgreSQL)
2. Εχετε δημιουργησει τουλαχιστον μια **Θεση** (Site) στο δελτιο θεσης
3. Τα απαραιτητα πεδια εχουν δημιουργηθει στη βαση δεδομενων (αυτοματο κατα την πρωτη εκτελεση)

## Πινακας Εργοταξιου (Dashboard)

Ο **Πινακας Εργοταξιου** ειναι ο κεντρικος πινακας ελεγχου που συγκεντρωνει πληροφοριες απο ολες τις υπο-φορμες σε μια ενιαια επισκοπηση.

### Επιλογη θεσης και ετους

Στο επανω μερος του πινακα υπαρχουν δυο επιλογεις:

| Πεδιο | Περιγραφη |
|-------|-----------|
| **Θεση** (comboBox_sito) | Επιλογη αρχαιολογικης θεσης απο τις καταχωρημενες στη βαση |
| **Ετος** (comboBox_anno) | Επιλογη ετους αναφορας |

Αφου επιλεξετε θεση και ετος, πατηστε το κουμπι **Ανανεωση** για να ενημερωθει ο πινακας.

<!-- IMAGE: Επιλογεις θεσης και ετους στο επανω μερος του dashboard -->
> **Εικ. 3**: Επιλογεις θεσης/ετους και κουμπι ανανεωσης

### Συνοψη προυπολογισμου

Η ενοτητα προυπολογισμου εμφανιζει:

```
+---------------------------------------------------+
| Συνοψη Προυπολογισμου                               |
+---------------------------------------------------+
| Προβλεπομενο ποσο:   EUR 150.000,00                |
| Πραγματικο ποσο:     EUR 112.500,00                |
| Ποσοστο:             75%                           |
| [=================------] 75%                     |
|                                                    |
|        [Κυκλικο γραφημα κατανομης]                  |
|                                                    |
+---------------------------------------------------+
```

| Στοιχειο | Περιγραφη |
|----------|-----------|
| **Προβλεπομενο ποσο** | Συνολο προυπολογισμενων κονδυλιων για τη θεση/ετος |
| **Πραγματικο ποσο** | Συνολο πραγματικων δαπανων |
| **Μπαρα προοδου** | Ποσοστιαια απεικονιση χρησης προυπολογισμου |
| **Κυκλικο γραφημα** | Κατανομη δαπανων ανα κατηγορια (pie chart) |

<!-- IMAGE: Ενοτητα συνοψης προυπολογισμου με μπαρα προοδου και κυκλικο γραφημα -->

### Συνοψη προσωπικου

Η ενοτητα προσωπικου εμφανιζει:

| Στοιχειο | Περιγραφη |
|----------|-----------|
| **Παροντες** | Αριθμος μελων σε ενεργη υπηρεσια σημερα |
| **Αδεια** | Αριθμος μελων σε αδεια |
| **Ασθενεια** | Αριθμος μελων σε ασθενεια |
| **Μηνιαιες ωρες** | Συνολο ωρων εργασιας τρεχοντος μηνα |
| **Μηνιαιο κοστος** | Συνολικο κοστος προσωπικου τρεχοντος μηνα |

<!-- IMAGE: Ενοτητα συνοψης προσωπικου με αριθμους και πινακα -->
> **Εικ. 5**: Συνοψη προσωπικου -- κατασταση παρουσιων και μηνιαιο κοστος

### Συνοψη εξοπλισμου

Η ενοτητα εξοπλισμου εμφανιζει:

| Στοιχειο | Περιγραφη |
|----------|-----------|
| **Συνολο** | Αριθμος καταχωρημενων ειδων εξοπλισμου |
| **Σε χρηση** | Ειδη με κατασταση "σε_χρηση" |
| **Συντηρηση** | Ειδη με κατασταση "συντηρηση" |
| **Ειδοποιησεις καθυστερησης** | Ειδη με εκκρεμη η ληξιπροθεσμη συντηρηση |

Οι **ειδοποιησεις καθυστερησης** εμφανιζονται ως προειδοποιητικα μηνυματα οταν η προγραμματισμενη ημερομηνια επομενης συντηρησης εχει παρελθει.

<!-- IMAGE: Ενοτητα εξοπλισμου με δεικτες κατασταση και ειδοποιησεις -->
> **Εικ. 6**: Συνοψη εξοπλισμου -- κατασταση και ειδοποιησεις καθυστερησης

### Computo Metrico

Η ενοτητα Computo Metrico παρεχει εργαλεια υπολογισμου ογκων εκσκαφης:

| Λειτουργια | Περιγραφη |
|------------|-----------|
| **Διαφορα DEM** | Υπολογιζει τη διαφορα ογκου μεταξυ δυο DEM (πριν/μετα την εκσκαφη) |
| **DEM + Πολυγωνο** | Υπολογιζει τον ογκο περιορισμενο σε μια πολυγωνικη περιοχη |

1. Επιλεξτε τα raster DEM εισοδου (πριν και μετα)
2. (Προαιρετικα) Επιλεξτε ενα πολυγωνο περιορισμου
3. Πατηστε **Υπολογισμος**
4. Τα αποτελεσματα (ογκος σε m3) εμφανιζονται στον πινακα
5. Πατηστε **Αποθηκευση Computo** για να αποθηκευσετε το αποτελεσμα

<!-- IMAGE: Ενοτητα Computo Metrico με επιλογη DEM και αποτελεσματα -->
> **Εικ. 7**: Computo Metrico -- υπολογισμος ογκων εκσκαφης απο DEM

### Ιστορικο

Στο κατω μερος του πινακα, η ενοτητα **Ιστορικο** εμφανιζει χρονολογικα τις τελευταιες ενεργειες (προσθηκες, τροποποιησεις, διαγραφες) σε ολα τα υπο-δελτια της επιλεγμενης θεσης.

## Φορμα Προσωπικου

Η φορμα **Προσωπικο** (Personnel) επιτρεπει τη διαχειριση των μελων που εργαζονται στο αρχαιολογικο εργοταξιο.

### Πεδια δελτιου

| Πεδιο | Τυπος | Περιγραφη | Υποχρεωτικο |
|-------|-------|-----------|-------------|
| **Τοποθεσια** | ComboBox | Αρχαιολογικη θεση αναφορας | Ναι |
| **Ονομα** | Κειμενο | Ονομα εργαζομενου | Ναι |
| **Επωνυμο** | Κειμενο | Επωνυμο εργαζομενου | Ναι |
| **Ειδικοτητα** | ComboBox | Επαγγελματικη ειδικοτητα (αρχαιολογος, τοπογραφος, εργατης κ.λπ.) | Οχι |
| **Ρολος** | ComboBox | Ρολος στο εργοταξιο (υπευθυνος, βοηθος, ειδικος κ.λπ.) | Οχι |
| **Ημ. Προσληψης** | Ημερομηνια | Ημερομηνια εναρξης εργασιας | Οχι |
| **Ημ. Ληξης** | Ημερομηνια | Ημερομηνια ληξης συμβασης | Οχι |
| **Ημερησιο κοστος** | Αριθμος | Κοστος ανα ημερα εργασιας (EUR) | Οχι |
| **Σημειωσεις** | Κειμενο | Ελευθερο πεδιο σημειωσεων | Οχι |

<!-- IMAGE: Φορμα προσωπικου με ολα τα πεδια συμπληρωμενα -->
> **Εικ. 9**: Η φορμα Προσωπικου με παραδειγμα συμπληρωμενων πεδιων

### Γραμμη εργαλειων ΣΔΒΔ

Η φορμα διαθετει την τυπικη γραμμη εργαλειων ΣΔΒΔ του PyArchInit:

| Κουμπι | Λειτουργια |
|--------|------------|
| **Νεα εγγραφη** | Δημιουργια νεου μελους προσωπικου |
| **Αποθηκευση** | Αποθηκευση τρεχουσας εγγραφης |
| **Διαγραφη** | Διαγραφη τρεχουσας εγγραφης |
| **Αναζητηση** | Αναζητηση με κριτηρια |
| **Πλοηγηση** | Πρωτη / Προηγουμενη / Επομενη / Τελευταια |
| **Ταξινομηση** | Ταξινομηση εγγραφων κατα πεδιο |

### Ροη εργασιας δημιουργιας

1. Πατηστε **Νεα εγγραφη** -- η κατασταση αλλαζει σε "New Record"
2. Επιλεξτε τη **Θεση** απο την πτυσσομενη λιστα
3. Εισαγετε **Ονομα** και **Επωνυμο**
4. Επιλεξτε **Ειδικοτητα** και **Ρολο**
5. Εισαγετε τις ημερομηνιες και το ημερησιο κοστος
6. Πατηστε **Αποθηκευση**

<!-- IMAGE: Βημα-βημα δημιουργια νεας εγγραφης προσωπικου -->
> **Εικ. 10**: Ροη εργασιας δημιουργιας νεου μελους προσωπικου

## Φορμα Παρουσιων

Η φορμα **Παρουσιες** (Attendance) χρησιμοποιειται για την ημερησια καταγραφη ωρων εργασιας, αδειων και ασθενειων.

### Πεδια δελτιου

| Πεδιο | Τυπος | Περιγραφη | Υποχρεωτικο |
|-------|-------|-----------|-------------|
| **Τοποθεσια** | ComboBox | Αρχαιολογικη θεση | Ναι |
| **ID Προσωπικου** | ComboBox | Αναφορα στο μελος προσωπικου | Ναι |
| **Ημερομηνια** | Ημερομηνια | Ημερομηνια παρουσιας | Ναι |
| **Τυπος ημερας** | ComboBox | Τυπος: εργασιμη / αδεια / ασθενεια / αργια | Ναι |
| **Κανονικες ωρες** | Αριθμος | Ωρες κανονικης εργασιας | Οχι |
| **Υπερωριες** | Αριθμος | Ωρες υπερωριακης εργασιας | Οχι |
| **Κοστος ημερας** | Αριθμος | Υπολογισμενο κοστος ημερας (EUR) | Οχι |
| **Σημειωσεις** | Κειμενο | Ελευθερο πεδιο σημειωσεων | Οχι |

### Τυποι ημερας

| Τυπος | Περιγραφη |
|-------|-----------|
| **Εργασιμη** | Κανονικη εργασιμη ημερα |
| **Αδεια** | Ημερα αδειας (κανονικη, αναρρωτικη κ.λπ.) |
| **Ασθενεια** | Ημερα ασθενειας |
| **Αργια** | Αργια η μη εργασιμη ημερα |

<!-- IMAGE: Φορμα παρουσιων με πεδια συμπληρωμενα -->
> **Εικ. 11**: Η φορμα Παρουσιων με παραδειγμα ημερησιας καταγραφης

### Υπολογισμος κοστους ημερας

Το πεδιο **Κοστος ημερας** μπορει να υπολογισθει αυτοματα απο:
- Το ημερησιο κοστος του μελους (απο τη φορμα Προσωπικου)
- Τις κανονικες ωρες και τις υπερωριες
- Τον τυπο ημερας (οι ημερες αδειας/ασθενειας μπορει να εχουν διαφορετικο κοστος)

### Ροη ημερησιας καταγραφης

1. Πατηστε **Νεα εγγραφη**
2. Επιλεξτε **Θεση** και **ID Προσωπικου**
3. Εισαγετε την **Ημερομηνια** (προεπιλογη: σημερινη)
4. Επιλεξτε τον **Τυπο ημερας**
5. Εισαγετε **Κανονικες ωρες** και **Υπερωριες**
6. Πατηστε **Αποθηκευση**

<!-- IMAGE: Βημα-βημα καταγραφη παρουσιας -->
> **Εικ. 12**: Ροη εργασιας ημερησιας καταγραφης παρουσιας

## Φορμα Προυπολογισμου

Η φορμα **Προυπολογισμος** (Budget) χρησιμοποιειται για τον ελεγχο του οικονομικου σχεδιασμου και της εκτελεσης δαπανων.

### Πεδια δελτιου

| Πεδιο | Τυπος | Περιγραφη | Υποχρεωτικο |
|-------|-------|-----------|-------------|
| **Τοποθεσια** | ComboBox | Αρχαιολογικη θεση | Ναι |
| **Ετος** | Αριθμος | Ετος αναφορας | Ναι |
| **Κατηγορια** | ComboBox | Κατηγορια δαπανης (προσωπικο, εξοπλισμος, αναλωσιμα, μεταφορες κ.λπ.) | Ναι |
| **Κονδυλι** | Κειμενο | Περιγραφη συγκεκριμενου κονδυλιου | Ναι |
| **Προβλεπομενο ποσο** | Αριθμος | Προυπολογισμενο ποσο (EUR) | Οχι |
| **Πραγματικο ποσο** | Αριθμος | Πραγματικη δαπανη (EUR) | Οχι |
| **Σημειωσεις** | Κειμενο | Ελευθερο πεδιο σημειωσεων | Οχι |

### Κατηγοριες δαπανων

Ενδεικτικες κατηγοριες δαπανων:

| Κατηγορια | Περιγραφη |
|-----------|-----------|
| **Προσωπικο** | Μισθοι, ασφαλιση, μετακινησεις προσωπικου |
| **Εξοπλισμος** | Αγορα, ενοικιαση, συντηρηση εξοπλισμου |
| **Αναλωσιμα** | Υλικα τεκμηριωσης, εργαλεια μιας χρησης |
| **Μεταφορες** | Μεταφορα υλικων, καυσιμα, ενοικιαση οχηματων |
| **Αναλυσεις** | Εργαστηριακες αναλυσεις, χρονολογησεις |
| **Αλλα** | Λοιπες δαπανες |

<!-- IMAGE: Φορμα προυπολογισμου με παραδειγμα κονδυλιων -->
> **Εικ. 15**: Η φορμα Προυπολογισμου -- παραδειγμα με πολλαπλα κονδυλια

### Σχεση με τον Πινακα Εργοταξιου

Τα δεδομενα προυπολογισμου τροφοδοτουν αυτοματα τη **Συνοψη Προυπολογισμου** στον Πινακα Εργοταξιου:
- Το **Προβλεπομενο ποσο** αθροιζεται για ολα τα κονδυλια της θεσης/ετους
- Το **Πραγματικο ποσο** αθροιζεται αντιστοιχα
- Η **μπαρα προοδου** δειχνει το ποσοστο (πραγματικο / προβλεπομενο)
- Το **κυκλικο γραφημα** δειχνει την κατανομη δαπανων ανα κατηγορια

## Ροη εργασιας

### Πληρης ροη εργασιας εργοταξιου

Ακολουθηστε αυτα τα βηματα για να ρυθμισετε τη διαχειριση ενος νεου εργοταξιου:

### Βημα 1: Δημιουργια θεσης

```
1. Ανοιξτε το Δελτιο Θεσης (Site form)
2. Δημιουργηστε μια νεα θεση για το εργοταξιο σας
3. Αποθηκευστε
```

### Βημα 2: Καταχωρηση προσωπικου

```
1. Ανοιξτε τη φορμα Προσωπικου απο τη γραμμη εργαλειων
2. Δημιουργηστε εγγραφη για καθε μελος
3. Εισαγετε ειδικοτητα, ρολο και ημερησιο κοστος
4. Αποθηκευστε
```

### Βημα 3: Καταχωρηση εξοπλισμου

```
1. Ανοιξτε τη φορμα Εξοπλισμου
2. Δημιουργηστε εγγραφη για καθε ειδος εξοπλισμου
3. Εισαγετε τυπο, μαρκα, κατασταση, ημερομηνιες συντηρησης
4. Αποθηκευστε
```

### Βημα 4: Ορισμος προυπολογισμου

```
1. Ανοιξτε τη φορμα Προυπολογισμου
2. Δημιουργηστε κονδυλια ανα κατηγορια
3. Εισαγετε τα προβλεπομενα ποσα
4. Αποθηκευστε
```

### Βημα 5: Ημερησια καταγραφη (κατα τη διαρκεια της εκσκαφης)

```
1. Ανοιξτε τη φορμα Παρουσιων
2. Καταγραψτε τις παρουσιες και ωρες εργασιας για καθε μελος
3. Ενημερωστε τον Προυπολογισμο με πραγματικες δαπανες
4. Ελεγξτε τον Πινακα Εργοταξιου για συνοπτικη εικονα
```

### Βημα 6: Παρακολουθηση μεσω Πινακα

```
1. Ανοιξτε τον Πινακα Εργοταξιου
2. Επιλεξτε θεση και ετος
3. Πατηστε Ανανεωση
4. Ελεγξτε: προυπολογισμο, παρουσιες, κατασταση εξοπλισμου
5. Υπολογιστε ογκους εκσκαφης με το Computo Metrico
```

## Συχνες ερωτησεις (FAQ)

### Πως μπορω να δω ολο το προσωπικο μιας θεσης;

1. Ανοιξτε τη φορμα **Προσωπικου**
2. Πατηστε **Νεα αναζητηση**
3. Επιλεξτε τη θεση στο πεδιο **Τοποθεσια**
4. Πατηστε **Εκτελεση αναζητησης**
5. Πλοηγηθειτε στα αποτελεσματα

### Πως υπολογιζεται το μηνιαιο κοστος στον Πινακα;

Το μηνιαιο κοστος στον Πινακα Εργοταξιου ειναι το αθροισμα ολων των **Κοστος ημερας** απο τη φορμα Παρουσιων για τον τρεχοντα μηνα και τη συγκεκριμενη θεση.

### Τι γινεται αν σβησω ενα μελος προσωπικου;

Η διαγραφη ενος μελους προσωπικου **δεν** διαγραφει αυτοματα τις παρουσιες του. Πρεπει να διαγραψετε χειροκινητα τις σχετικες εγγραφες παρουσιων αν θελετε.

### Μπορω να χρησιμοποιησω το module χωρις PostgreSQL;

Ναι, το module λειτουργει και με **SQLite/Spatialite**. Ολες οι φορμες ειναι συμβατες και με τις δυο βασεις δεδομενων.

### Πως λειτουργει το Computo Metrico;

Το Computo Metrico χρησιμοποιει δυο DEM (Digital Elevation Models) -- ενα πριν και ενα μετα την εκσκαφη -- για να υπολογισει τη διαφορα ογκου. Αν οριστει πολυγωνο, ο υπολογισμος περιοριζεται στην εκταση του πολυγωνου.

### Γιατι δεν εμφανιζονται δεδομενα στον Πινακα;

- Βεβαιωθειτε οτι εχετε επιλεξει τη σωστη **Θεση** και **Ετος**
- Πατηστε **Ανανεωση** μετα την επιλογη
- Ελεγξτε οτι υπαρχουν καταχωρημενα δεδομενα στις αντιστοιχες φορμες

### Πως μπορω να παρακολουθω τις ληξιπροθεσμες συντηρησεις;

Στον **Πινακα Εργοταξιου**, η ενοτητα εξοπλισμου εμφανιζει αυτοματα ειδοποιησεις καθυστερησης για εξοπλισμο του οποιου η **Ημ. Επομ. Συντηρησης** εχει παρελθει.
