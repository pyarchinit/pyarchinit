# Εκπαιδευτικό 17: TMA - Πίνακας Αρχαιολογικών Υλικών

## Εισαγωγή

Το **Δελτίο TMA** (Πίνακας Αρχαιολογικών Υλικών) είναι η προηγμένη ενότητα του PyArchInit για τη διαχείριση υλικών ανασκαφής σύμφωνα με τα ιταλικά υπουργικά πρότυπα. Επιτρέπει λεπτομερή καταλογογράφηση σε συμμόρφωση με τους κανονισμούς του ICCD (Κεντρικό Ινστιτούτο Καταλόγου και Τεκμηρίωσης).

### Κύρια Χαρακτηριστικά

- Καταλογογράφηση σύμφωνα με το ICCD
- Διαχείριση υλικών ανά κιβώτιο/δοχείο
- Λεπτομερή χρονολογικά πεδία
- Πίνακας συσχετιζόμενων υλικών
- Ενσωματωμένη διαχείριση πολυμέσων
- Εξαγωγή ετικετών και δελτίων PDF

---

## Πρόσβαση στο Δελτίο

### Μέσω Μενού
1. Μενού **PyArchInit** στη γραμμή μενού του QGIS
2. Επιλέξτε **TMA Form**

### Μέσω Γραμμής Εργαλείων
1. Εντοπίστε τη γραμμή εργαλείων του PyArchInit
2. Κάντε κλικ στο εικονίδιο **TMA**

---

Το δελτίο παρουσιάζει μια σύνθετη διεπαφή με πολλά πεδία.

## Ροή Εργασίας

### Καταχώρηση Νέου TMA

1. **Άνοιγμα δελτίου**
   - Μέσω μενού ή γραμμής εργαλείων

2. **Νέα εγγραφή**
   - Κάντε κλικ στο «New Record»

3. **Στοιχεία ταυτοποίησης**
   ```
   Site: Roman Villa
   Area: 1000
   SU: 150
   Box: C-001
   ```

4. **Τοποθεσία**
   ```
   LDCT: deposit
   LDCN: Rome Superintendency Deposit
   SCAN: Roman Villa Excavations 2024
   ```

5. **Χρονολογία**
   ```
   DTZG: Roman Age
   DTZS: Imperial
   ```

6. **Υλικά** (πίνακας)
   ```
   | Inv | Cat | Class | Def | Qty |
   |-----|-----|-------|-----|-----|
   | 001 | ceramic | common | jar | 5 |
   | 002 | ceramic | sigillata | plate | 3 |
   | 003 | glass | - | unguentarium | 1 |
   ```

7. **Αποθήκευση**
   - Κάντε κλικ στο «Save»

## Βέλτιστες Πρακτικές

### Πρότυπα ICCD

- Χρησιμοποιήστε ελεγχόμενα λεξιλόγια ICCD
- Ακολουθήστε τις επίσημες συντομογραφίες
- Διατηρήστε ορολογική συνέπεια

### Οργάνωση Κιβωτίων

- Μοναδική αύξουσα αρίθμηση
- Ένα TMA ανά φυσικό κιβώτιο
- Διαχωρισμός ανά ΣΕ όταν είναι δυνατόν

### Τεκμηρίωση

- Πάντα συσχέτιση φωτογραφιών και σχεδίων
- Χρήση μοναδικών κωδικών για πολυμέσα
- Καταγραφή συντάκτη και ημερομηνίας

---

## Αντιμετώπιση Προβλημάτων

### Πρόβλημα: Τα λεξιλόγια ICCD δεν είναι διαθέσιμα

**Αιτία**: Μη ρυθμισμένος θησαυρός.

**Λύση**:
1. Εισαγάγετε τυπικά λεξιλόγια ICCD
2. Επαληθεύστε τη ρύθμιση θησαυρού

### Πρόβλημα: Τα υλικά δεν αποθηκεύονται

**Αιτία**: Ο πίνακας υλικών δεν συγχρονίζεται.

**Λύση**:
1. Επαληθεύστε ότι όλα τα υποχρεωτικά πεδία είναι συμπληρωμένα
2. Αποθηκεύστε το κύριο δελτίο πριν από την προσθήκη υλικών
