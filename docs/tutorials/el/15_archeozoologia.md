# Εκπαιδευτικό 15: Δελτίο Αρχαιοζωολογίας (Πανίδα)

## Εισαγωγή

Το **Δελτίο Αρχαιοζωολογίας/Πανίδας** (FAUNA RECORD - FR) είναι η ενότητα του PyArchInit αφιερωμένη στην ανάλυση και τεκμηρίωση οστεολογικών καταλοίπων πανίδας. Επιτρέπει την καταγραφή λεπτομερών αρχαιοζωολογικών δεδομένων για τη μελέτη αρχαίων οικονομιών επιβίωσης.

### Βασικές Έννοιες

**Αρχαιοζωολογία:**
- Μελέτη ζωικών καταλοίπων από αρχαιολογικά συμφραζόμενα
- Ανάλυση σχέσεων ανθρώπου-ζώου στο παρελθόν
- Ανασύσταση διατροφής, κτηνοτροφίας, κυνηγίου

**Καταγραφόμενα δεδομένα:**
- Ταξινομική αναγνώριση (είδος)
- Σκελετικά μέρη παρόντα
- ΕΑΑ (Ελάχιστος Αριθμός Ατόμων)
- Κατάσταση διατήρησης
- Ταφονομικά ίχνη
- Σημάδια τεμαχισμού

---

## Πρόσβαση στο Δελτίο

### Μέσω Μενού
1. Μενού **PyArchInit** στη γραμμή μενού του QGIS
2. Επιλέξτε **Fauna Form** (ή **Fauna form**)

### Μέσω Γραμμής Εργαλείων
1. Εντοπίστε τη γραμμή εργαλείων του PyArchInit
2. Κάντε κλικ στο εικονίδιο **Fauna** (στυλιζαρισμένο οστό)

---

## Ροή Εργασίας

### Καταγραφή Καταλοίπων Πανίδας

1. **Άνοιγμα δελτίου**
   - Μέσω μενού ή γραμμής εργαλείων

2. **Νέα εγγραφή**
   - Κάντε κλικ στο «New Record»

3. **Στοιχεία ταυτοποίησης**
   ```
   Site: Roman Villa
   Area: 1000
   SU: 150
   Responsible: Γ. Βέρντι
   Date: 20/07/2024
   ```

4. **Αρχαιοζωολογικά δεδομένα** (Καρτέλα 2)
   ```
   Context: Dump/Waste
   Species: Bos taurus
   MNI: 3

   Σκελετικά μέρη:
   - Βραχιόνιο / Δεξιά / 2 / 1
   - Κνήμη / Αριστερή / 3 / 2
   - Μεταποδιαίο / - / 5 / 1
   ```

5. **Ταφονομικά δεδομένα** (Καρτέλα 3)
   ```
   Fragmentation: Θραυσματοποιημένο
   Conservation: Καλή
   Combustion: Απουσία
   Taphonomic signs: Σημάδια ριζών
   ```

6. **Ερμηνεία**
   ```
   Απορριπτήρας τροφικών αποβλήτων.
   Παρουσία σημαδιών τεμαχισμού
   σε ορισμένα μακρά οστά.
   ```

7. **Αποθήκευση**
   - Κάντε κλικ στο «Save»

## Βέλτιστες Πρακτικές

### Αναγνώριση

- Χρησιμοποιήστε συγκριτικές συλλογές
- Δηλώστε βαθμό βεβαιότητας αναγνώρισης
- Καταγράψτε και τα μη αναγνωρίσιμα κατάλοιπα

### ΕΑΑ

- Υπολογίστε για κάθε είδος ξεχωριστά
- Λάβετε υπόψη πλευρά και ηλικία ευρημάτων
- Τεκμηριώστε τη μέθοδο υπολογισμού

### Ταφονομία

- Παρατηρήστε συστηματικά κάθε δείγμα
- Τεκμηριώστε ίχνη πριν από το πλύσιμο
- Φωτογραφίστε σημαντικές περιπτώσεις

### Πλαίσιο

- Πάντα σύνδεση με τη ΣΕ προέλευσης
- Καταγράψτε τη μέθοδο ανάκτησης
- Σημειώστε σημαντικές συσχετίσεις

## Αντιμετώπιση Προβλημάτων

### Πρόβλημα: Το είδος δεν είναι διαθέσιμο

**Αιτία**: Ελλιπής κατάλογος ειδών.

**Λύση**:
1. Ελέγξτε τον θησαυρό πανίδας
2. Προσθέστε τα ελλείποντα είδη
3. Επικοινωνήστε με τον διαχειριστή

### Πρόβλημα: Ο ΕΑΑ δεν υπολογίζεται

**Αιτία**: Τα σκελετικά μέρη δεν έχουν καταγραφεί.

**Λύση**:
1. Συμπληρώστε τον πίνακα σκελετικών μερών
2. Δηλώστε πλευρά και ποσότητα
3. Το σύστημα θα υπολογίσει αυτόματα
