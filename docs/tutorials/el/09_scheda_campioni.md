# Εκπαιδευτικό 09: Δελτίο Δειγμάτων

## Εισαγωγή

Το **Δελτίο Δειγμάτων** είναι η ενότητα του PyArchInit αφιερωμένη στη διαχείριση αρχαιολογικών δειγμάτων. Σας επιτρέπει να καταγράψετε και να παρακολουθήσετε όλους τους τύπους δειγμάτων που λαμβάνονται κατά τη διάρκεια της ανασκαφής: χώμα, κάρβουνα, σπόρους, οστά, κονιάματα, μέταλλα και άλλα υλικά που προορίζονται για εξειδικευμένες αναλύσεις.

### Τύποι Δειγμάτων

Τα αρχαιολογικά δείγματα περιλαμβάνουν:
- **Ιζήματα**: για ιζηματολογικές, κοκκομετρικές αναλύσεις
- **Κάρβουνα**: για χρονολόγηση ραδιοάνθρακα (C14)
- **Σπόροι/Γύρη**: για αρχαιοβοτανικές αναλύσεις
- **Οστά**: για αρχαιοζωολογικές, ισοτοπικές, αναλύσεις DNA
- **Κονιάματα/Επιχρίσματα**: για αρχαιομετρικές αναλύσεις
- **Μέταλλα/Σκουριές**: για μεταλλουργικές αναλύσεις
- **Κεραμική**: για αναλύσεις πηλού και προέλευσης

---

## Πρόσβαση στο Δελτίο

### Μέσω Μενού
1. Μενού **PyArchInit** στη γραμμή μενού του QGIS
2. Επιλέξτε **Samples Form**

### Μέσω Γραμμής Εργαλείων
1. Εντοπίστε τη γραμμή εργαλείων PyArchInit
2. Κάντε κλικ στο εικονίδιο **Samples**

---

## Επισκόπηση Διεπαφής

Το δελτίο παρουσιάζει μια απλοποιημένη διάταξη για ταχεία διαχείριση δειγμάτων.

### Κύριες Περιοχές

| # | Περιοχή | Περιγραφή |
|---|---------|-----------|
| 1 | Γραμμή Εργαλείων ΣΔΒΔ | Πλοήγηση, αναζήτηση, αποθήκευση |
| 2 | Πληροφορίες ΒΔ | Κατάσταση εγγραφής, ταξινόμηση, μετρητής |
| 3 | Πεδία Ταυτοποίησης | Θέση, Αρ. Δείγματος, Τύπος |
| 4 | Περιγραφικά Πεδία | Περιγραφή, σημειώσεις |
| 5 | Πεδία Αποθήκευσης | Κιβώτιο, Τοποθεσία |

---

## Γραμμή Εργαλείων ΣΔΒΔ

### Κουμπιά Πλοήγησης

| Εικονίδιο | Λειτουργία | Περιγραφή |
|-----------|------------|-----------|
| First rec | Μετάβαση στην πρώτη εγγραφή |
| Prev rec | Μετάβαση στην προηγούμενη εγγραφή |
| Next rec | Μετάβαση στην επόμενη εγγραφή |
| Last rec | Μετάβαση στην τελευταία εγγραφή |

### Κουμπιά CRUD

| Εικονίδιο | Λειτουργία | Περιγραφή |
|-----------|------------|-----------|
| New record | Δημιουργία νέας εγγραφής δείγματος |
| Save | Αποθήκευση αλλαγών |
| Delete | Διαγραφή τρέχουσας εγγραφής |

### Κουμπιά Αναζήτησης

| Εικονίδιο | Λειτουργία | Περιγραφή |
|-----------|------------|-----------|
| New search | Εκκίνηση νέας αναζήτησης |
| Search!!! | Εκτέλεση αναζήτησης |
| Order by | Ταξινόμηση αποτελεσμάτων |
| View all | Προβολή όλων των εγγραφών |

## Πεδία Δελτίου

### Θέση

**Πεδίο**: `comboBox_sito`
**Βάση δεδομένων**: `sito`

Επιλέξτε την αρχαιολογική θέση στην οποία ανήκει.

### Αριθμός Δείγματος

**Πεδίο**: `lineEdit_nr_campione`
**Βάση δεδομένων**: `nr_campione`

Προοδευτικός αριθμός ταυτοποίησης δείγματος.

### Τύπος Δείγματος

**Πεδίο**: `comboBox_tipo_campione`
**Βάση δεδομένων**: `tipo_campione`

Τυπολογική κατάταξη του δείγματος. Οι τιμές προέρχονται από θησαυρό.

**Συνήθεις τύποι:**
| Τύπος | Περιγραφή |
|-------|-----------|
| Ίζημα | Δείγμα εδάφους |
| Κάρβουνο | Για χρονολόγηση C14 |
| Σπόροι | Καρπολογικά κατάλοιπα |
| Οστά | Πανιδικά κατάλοιπα |
| Κονίαμα | Οικοδομικά συνδετικά |
| Κεραμική | Για ανάλυση πηλού |
| Μέταλλο | Για μεταλλουργική ανάλυση |
| Γύρη | Για παλυνολογική ανάλυση |

### Περιγραφή

**Πεδίο**: `textEdit_descrizione`
**Βάση δεδομένων**: `descrizione`

Λεπτομερής περιγραφή του δείγματος.

**Προτεινόμενο περιεχόμενο:**
- Φυσικά χαρακτηριστικά δείγματος
- Ποσότητα που λήφθηκε
- Μέθοδος συλλογής
- Λόγος δειγματοληψίας
- Σχεδιαζόμενες αναλύσεις

### Τομέας

**Πεδίο**: `comboBox_area`
**Βάση δεδομένων**: `area`

Τομέας ανασκαφής προέλευσης.

### ΣΜ

**Πεδίο**: `comboBox_us`
**Βάση δεδομένων**: `us`

Στρωματογραφική Μονάδα προέλευσης.

### Αριθμός Απογραφής Υλικού

**Πεδίο**: `lineEdit_nr_inv_mat`
**Βάση δεδομένων**: `numero_inventario_materiale`

Αν το δείγμα συνδέεται με απογεγραμμένο εύρημα, αναφέρετε αριθμό απογραφής.

### Αριθμός Κιβωτίου

**Πεδίο**: `lineEdit_nr_cassa`
**Βάση δεδομένων**: `nr_cassa`

Κιβώτιο ή περιέκτης αποθήκευσης.

### Τοποθεσία Αποθήκευσης

**Πεδίο**: `comboBox_luogo_conservazione`
**Βάση δεδομένων**: `luogo_conservazione`

Τοποθεσία αποθήκευσης του δείγματος.

**Παραδείγματα:**
- Εργαστήριο ανασκαφής
- Αποθήκη μουσείου
- Εξωτερικό εργαστήριο αναλύσεων
- Πανεπιστήμιο

## Ροή Εργασίας

### Δημιουργία Νέου Δείγματος

1. **Άνοιγμα δελτίου** - Μέσω μενού ή γραμμής εργαλείων

2. **Νέα εγγραφή** - Κάντε κλικ στο "New Record"

3. **Δεδομένα ταυτοποίησης**
   ```
   Site: Ρωμαϊκή Έπαυλη Settefinestre
   Sample No.: C-2024-001
   Sample type: Κάρβουνο
   ```

4. **Προέλευση**
   ```
   Area: 1000
   SU: 150
   ```

5. **Περιγραφή**
   ```
   Δείγμα κάρβουνου ληφθέν από
   καμένη επιφάνεια ΣΜ 150.
   Ποσότητα: περίπου 50 γρ.
   Συλλέχθηκε για χρονολόγηση C14.
   ```

6. **Αποθήκευση**
   ```
   Box No.: Samp-1
   Location: Εργαστήριο ανασκαφής
   ```

7. **Αποθήκευση** - Κάντε κλικ στο "Save"

### Αναζήτηση Δειγμάτων

1. Κάντε κλικ στο "New Search"
2. Συμπληρώστε κριτήρια:
   - Θέση
   - Τύπος δείγματος
   - ΣΜ
3. Κάντε κλικ στο "Search"
4. Πλοηγηθείτε στα αποτελέσματα

## Βέλτιστες Πρακτικές

### Ονοματοδοσία

- Χρησιμοποιήστε μοναδικούς και σημαντικούς κωδικούς
- Προτεινόμενη μορφή: `ΘΕΣΗ-ΕΤΟΣ-ΠΡΟΟΔΕΥΤΙΚΟΣ`
- Παράδειγμα: `VRS-2024-C001`

### Συλλογή

- Πάντα τεκμηριώνετε τις συντεταγμένες συλλογής
- Φωτογραφίστε το σημείο συλλογής
- Σημειώστε βάθος και πλαίσιο

### Αποθήκευση

- Χρησιμοποιήστε κατάλληλους περιέκτες ανά τύπο
- Σημάνετε σαφώς κάθε δείγμα
- Διατηρήστε κατάλληλες συνθήκες

### Τεκμηρίωση

- Πάντα συνδέστε με ΣΜ προέλευσης
- Αναφέρετε σχεδιαζόμενες αναλύσεις
- Καταγράψτε αποστολή σε εξωτερικά εργαστήρια

---

## Αντιμετώπιση Προβλημάτων

### Πρόβλημα: Τύπος δείγματος μη διαθέσιμος

**Αιτία**: Θησαυρός μη ρυθμισμένος.

**Λύση**:
1. Ανοίξτε το Δελτίο Θησαυρού
2. Προσθέστε τον ελλείποντα τύπο για `campioni_table`
3. Αποθηκεύστε και ανοίξτε ξανά το Δελτίο Δειγμάτων

### Πρόβλημα: Η ΣΜ δεν εμφανίζεται

**Αιτία**: Η ΣΜ δεν είναι καταχωρημένη για την επιλεγμένη θέση.

**Λύση**:
1. Επαληθεύστε ότι η ΣΜ υπάρχει στο Δελτίο ΣΜ
2. Ελέγξτε ότι ανήκει στην ίδια θέση
