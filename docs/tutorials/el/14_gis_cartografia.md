# Εκπαιδευτικό 14: GIS και Χαρτογραφία

## Εισαγωγή

Το PyArchInit είναι βαθιά ενσωματωμένο με το **QGIS**, αξιοποιώντας όλες τις δυνατότητες GIS για τη χωρική διαχείριση αρχαιολογικών δεδομένων. Αυτό το εκπαιδευτικό καλύπτει τη χαρτογραφική ενσωμάτωση, τα προκαθορισμένα στρώματα και τις προηγμένες λειτουργίες όπως η **αυτόματη τμηματοποίηση SAM**.

### Κύριες Λειτουργίες GIS

- Απεικόνιση ΣΕ στον χάρτη
- Προκαθορισμένα διανυσματικά στρώματα
- Προσαρμοσμένη μορφοποίηση QML
- Υψόμετρα και μετρήσεις GIS
- Αυτόματη τμηματοποίηση (SAM)
- Χαρτογραφική εξαγωγή

## Προκαθορισμένα Στρώματα PyArchInit

### Κύρια Διανυσματικά Στρώματα

| Στρώμα | Γεωμετρία | Περιγραφή |
|--------|-----------|-----------|
| `pyunitastratigrafiche` | MultiPolygon | ΣΕ αποθέσεων |
| `pyunitastratigrafiche_usm` | MultiPolygon | ΣΕ τοίχων |
| `pyarchinit_quote` | Point | Σημεία υψομέτρων |
| `pyarchinit_siti` | Point | Θέσεις αρχαιολογικών χώρων |
| `pyarchinit_ripartizioni_spaziali` | Polygon | Περιοχές ανασκαφής |
| `pyarchinit_strutture_ipotesi` | Polygon | Υποθετικές δομές |
| `pyarchinit_documentazione` | Point | Αναφορές τεκμηρίωσης |

### Ιδιότητες Στρώματος ΣΕ

| Πεδίο | Τύπος | Περιγραφή |
|-------|-------|-----------|
| `gid` | Integer | Μοναδικό αναγνωριστικό |
| `scavo_s` | Text | Όνομα θέσης |
| `area_s` | Text | Αριθμός τομέα |
| `us_s` | Text | Αριθμός ΣΕ |
| `stratigraph_index_us` | Integer | Στρωματογραφικός δείκτης |
| `tipo_us_s` | Text | Τύπος ΣΕ |
| `rilievo_originale` | Text | Πρωτότυπη αποτύπωση |
| `disegnatore` | Text | Σχεδιαστής αποτύπωσης |
| `data` | Date | Ημερομηνία αποτύπωσης |

## Απεικόνιση ΣΕ στον Χάρτη

### Από την Καρτέλα «Map» στο Δελτίο ΣΕ

1. Ανοίξτε ένα δελτίο ΣΕ
2. Επιλέξτε την καρτέλα **Map**
3. Διαθέσιμες λειτουργίες:

| Κουμπί | Λειτουργία |
|--------|------------|
| View US | Εμφάνιση τρέχουσας ΣΕ στον χάρτη |
| View All | Εμφάνιση όλων των ΣΕ στον τομέα |
| New Record | Δημιουργία νέας γεωμετρίας |
| Pan to | Κεντράρισμα χάρτη στη ΣΕ |

### Απεικόνιση από Αναζήτηση

1. Εκτελέστε αναζήτηση ΣΕ
2. Κουμπί **«View Record»** → εμφάνιση μεμονωμένης
3. Κουμπί **«View All»** → εμφάνιση όλων των αποτελεσμάτων

## Μορφοποίηση Στρωμάτων

### Αρχεία QML

Το PyArchInit περιλαμβάνει προκαθορισμένες μορφοποιήσεις σε μορφή QML:
```
pyarchinit/styles/
├── pyunitastratigrafiche.qml
├── pyunitastratigrafiche_usm.qml
├── pyarchinit_quote.qml
└── ...
```

### Εφαρμογή Μορφοποίησης

1. Επιλέξτε το στρώμα στο υπόμνημα
2. Δεξί κλικ → **Properties**
3. Καρτέλα **Style**
4. **Load style** → επιλέξτε QML

### Προσαρμογή

Οι μορφοποιήσεις μπορούν να προσαρμοστούν για:
- Χρώματα βάσει τύπου ΣΕ
- Ετικέτες με αριθμό ΣΕ
- Διαφάνεια
- Περιγράμματα και γεμίσματα

## Υψόμετρα και Μετρήσεις

### Στρώμα Υψομέτρων

Το στρώμα `pyarchinit_quote` αποθηκεύει:
- Συντεταγμένες X, Y
- Υψόμετρο Z (απόλυτο)
- Τύπο σημείου υψομέτρου
- Αναφορά ΣΕ
- Αναφορά τομέα

### Αυτόματος Υπολογισμός Υψομέτρων

Από το Δελτίο ΣΕ, τα ελάχ./μέγ. υψόμετρα υπολογίζονται:
1. Ερώτημα σημείων υψομέτρου που συσχετίζονται με τη ΣΕ
2. Εξαγωγή ελάχιστης και μέγιστης τιμής
3. Εμφάνιση στην αναφορά

### Εισαγωγή Υψομέτρων

1. Στρώμα υψομέτρων σε λειτουργία επεξεργασίας
2. Σχεδιάστε σημείο στον χάρτη
3. Συμπληρώστε τις ιδιότητες:
   - `sito_q`
   - `area_q`
   - `us_q`
   - `quota`
   - `unita_misura_q`

## Αυτόματη Τμηματοποίηση SAM

### Τι είναι το SAM;

Το **SAM (Segment Anything Model)** είναι ένα μοντέλο τεχνητής νοημοσύνης που αναπτύχθηκε από τη Meta για αυτόματη τμηματοποίηση εικόνων. Το PyArchInit το ενσωματώνει για:
- Αυτόματη ψηφιοποίηση λίθων/στοιχείων
- Τμηματοποίηση ορθοφωτογραφιών
- Επιτάχυνση αποτυπώσεων

### Πρόσβαση στη Λειτουργία

1. **PyArchInit** → **SAM Segmentation**
2. Ή από τη γραμμή εργαλείων: εικονίδιο **SAM**

### Διεπαφή SAM

```
+--------------------------------------------------+
|        SAM Stone Segmentation                     |
+--------------------------------------------------+
| Input:                                            |
|   Raster Layer: [ComboBox orthophoto]            |
+--------------------------------------------------+
| Target Layer:                                     |
|   [o] pyunitastratigrafiche                      |
|   [ ] pyunitastratigrafiche_usm                  |
+--------------------------------------------------+
| Default Attributes:                               |
|   Site (sito): [automatic field]                 |
|   Area: [area input]                             |
|   Stratigraphic Index: [1-10]                    |
|   Type US: [stone|layer|accumulation|cut]        |
+--------------------------------------------------+
| Segmentation Mode:                                |
|   [o] Automatic (detect all stones)              |
|   [ ] Click mode (click on each stone)           |
|   [ ] Box mode (draw rectangle)                  |
|   [ ] Polygon mode (draw freehand)               |
|   [ ] From layer (use existing polygon)          |
+--------------------------------------------------+
| Model:                                            |
|   [ComboBox model]                               |
|   API Key: [******]                              |
+--------------------------------------------------+
|        [Start Segmentation]  [Cancel]             |
+--------------------------------------------------+
```

### Λειτουργίες Τμηματοποίησης

#### 1. Αυτόματη Λειτουργία
- Τμηματοποιεί αυτόματα όλα τα ορατά αντικείμενα
- Ιδανική για περιοχές με πολλούς λίθους
- Απαιτεί ορθοφωτογραφία καλής ποιότητας

#### 2. Λειτουργία Κλικ
- Κάντε κλικ σε κάθε αντικείμενο για τμηματοποίηση
- Δεξί κλικ ή Enter για ολοκλήρωση
- Escape για ακύρωση
- Ακριβέστερη για συγκεκριμένα αντικείμενα

#### 3. Λειτουργία Πλαισίου
- Σχεδιάστε ορθογώνιο στην περιοχή
- Τμηματοποιεί μόνο την επιλεγμένη περιοχή
- Χρήσιμη για οριοθετημένες ζώνες

#### 4. Λειτουργία Πολυγώνου
- Σχεδιάστε πολύγωνο ελεύθερου χεριού
- Κλικ για προσθήκη κορυφών
- Δεξί κλικ για ολοκλήρωση
- Για ακανόνιστες περιοχές

#### 5. Λειτουργία από Στρώμα
- Χρήση υπάρχοντος πολυγώνου ως μάσκα
- Επιλέξτε στρώμα πολυγώνων
- Τμηματοποιεί μόνο εντός πολυγώνου

### Διαθέσιμα Μοντέλα

| Μοντέλο | Τύπος | Απαιτήσεις | Ποιότητα |
|---------|-------|------------|----------|
| Replicate SAM-2 | Cloud API | Κλειδί API | Εξαιρετική |
| Roboflow SAM-3 | Cloud API | Κλειδί API | Εξαιρετική + Text Prompt |
| SAM vit_b | Τοπικό | 375MB VRAM | Καλή |
| SAM vit_l | Τοπικό | 1.2GB VRAM | Πολύ καλή |
| SAM vit_h | Τοπικό | 2.5GB VRAM | Εξαιρετική |
| OpenCV | Τοπικό | Καμία | Βασική |

### SAM-3 με Text Prompt

Η έκδοση SAM-3 (Roboflow) υποστηρίζει **κειμενικές εντολές**:
- «stones» - λίθοι
- «pottery fragments» - θραύσματα κεραμικής
- «bones» - οστά
- Οποιαδήποτε κειμενική περιγραφή

### Ρύθμιση API

#### Replicate API (SAM-2)
1. Εγγραφείτε στο [replicate.com](https://replicate.com)
2. Αποκτήστε κλειδί API
3. Εισαγάγετε στη ρύθμιση

#### Roboflow API (SAM-3)
1. Εγγραφείτε στο [roboflow.com](https://roboflow.com)
2. Αποκτήστε κλειδί API
3. Εισαγάγετε στη ρύθμιση

### Τοπική Εγκατάσταση SAM

Για τοπική χρήση χωρίς API:
```bash
# Δημιουργία εικονικού περιβάλλοντος
cd ~/pyarchinit/bin
python -m venv sam_venv

# Ενεργοποίηση περιβάλλοντος
source sam_venv/bin/activate

# Εγκατάσταση εξαρτήσεων
pip install segment-anything torch torchvision

# Λήψη μοντέλων (προαιρετικά)
# vit_b: https://dl.fbaipublicfiles.com/segment_anything/sam_vit_b_01ec64.pth
# vit_l: https://dl.fbaipublicfiles.com/segment_anything/sam_vit_l_0b3195.pth
# vit_h: https://dl.fbaipublicfiles.com/segment_anything/sam_vit_h_4b8939.pth
```

### Ροή Εργασίας SAM

1. **Προετοιμασία**
   - Φόρτωση ορθοφωτογραφίας ως στρώμα ψηφιδωτού
   - Επαλήθευση συστήματος αναφοράς συντεταγμένων
   - Προετοιμασία στρώματος προορισμού

2. **Ρύθμιση**
   - Επιλογή ψηφιδωτού εισόδου
   - Ορισμός προεπιλεγμένων ιδιοτήτων
   - Επιλογή λειτουργίας και μοντέλου

3. **Εκτέλεση**
   - Κλικ στο «Start Segmentation»
   - Αναμονή επεξεργασίας
   - Επαλήθευση αποτελεσμάτων

4. **Μεταεπεξεργασία**
   - Έλεγχος παραγόμενων πολυγώνων
   - Αντιστοίχιση αριθμών ΣΕ
   - Διόρθωση τυχόν σφαλμάτων

## Χαρτογραφική Ενσωμάτωση

### Εξαγωγή Δεδομένων GIS

Από το Δελτίο ΣΕ, καρτέλα Map:
- **Export GeoPackage**: Στρώμα ως GPKG
- **Export Shapefile**: Στρώμα ως SHP
- **Export GeoJSON**: Στρώμα ως JSON

### Εισαγωγή Δεδομένων GIS

Εισαγωγή υπαρχουσών γεωμετριών:
1. Φόρτωση στρώματος στο QGIS
2. Επιλογή χαρακτηριστικών
3. Χρήση λειτουργίας εισαγωγής

### Γεωεπεξεργασία

Διαθέσιμες χωρικές λειτουργίες:
- Ζώνη επιρροής (Buffer)
- Τομή (Intersection)
- Ένωση (Union)
- Διαφορά (Difference)
- Κεντροειδή (Centroids)

## Βέλτιστες Πρακτικές

### 1. Ορθοφωτογραφίες

- Ελάχιστη ανάλυση: 1-2 cm/pixel
- Μορφή: Γεωαναφερμένο GeoTIFF
- Σύστημα αναφοράς: συνεπές με το έργο

### 2. Ψηφιοποίηση

- Χρήση snap για ακρίβεια
- Επαλήθευση τοπολογίας
- Διατήρηση συνέπειας ιδιοτήτων

### 3. Τμηματοποίηση SAM

- Υψηλής ποιότητας ορθοφωτογραφία
- Ομοιόμορφος φωτισμός
- Επαρκής αντίθεση αντικειμένου/φόντου
- Πάντα απαραίτητη η εκ των υστέρων επαλήθευση

### 4. Οργάνωση Στρωμάτων

- Ομαδοποίηση κατά τύπο
- Χρήση συνεπών μορφοποιήσεων
- Διατήρηση τάξης στο υπόμνημα

## Αντιμετώπιση Προβλημάτων

### Τα Στρώματα δεν Εμφανίζονται

**Πιθανές αιτίες**:
- Λανθασμένη έκταση
- Διαφορετικό σύστημα αναφοράς
- Ενεργό φίλτρο

**Λύσεις**:
- Ζουμ στο στρώμα (Zoom to Layer)
- Επαλήθευση CRS
- Αφαίρεση φίλτρων

### Το SAM δεν Λειτουργεί

**Πιθανές αιτίες**:
- Μη έγκυρο κλειδί API
- Μη γεωαναφερμένο ψηφιδωτό
- Μη εγκατεστημένο τοπικό μοντέλο

**Λύσεις**:
- Επαλήθευση κλειδιού API
- Έλεγχος γεωαναφοράς
- Εγκατάσταση μοντέλου

### Κατεστραμμένες Γεωμετρίες

**Πιθανές αιτίες**:
- Σφάλματα ψηφιοποίησης
- Προβληματική εισαγωγή

**Λύσεις**:
- Χρήση «Fix Geometries»
- Επανασχεδίαση στοιχείου
