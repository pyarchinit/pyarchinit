# Οδηγός 28: Εξαγωγή GeoPackage

## Εισαγωγή

Η λειτουργία **Εξαγωγή GeoPackage** επιτρέπει τη συσκευασία διανυσματικών και ψηφιδωτών επιπέδων του PyArchInit σε ένα μοναδικό αρχείο GeoPackage (.gpkg). Αυτή η μορφή είναι ιδανική για κοινή χρήση, αρχειοθέτηση και φορητότητα δεδομένων.

### Πλεονεκτήματα GeoPackage

| Πτυχή | Πλεονέκτημα |
|-------|-------------|
| Ενιαίο αρχείο | Όλα τα δεδομένα σε ένα αρχείο |
| Φορητότητα | Εύκολη κοινή χρήση |
| Πρότυπο OGC | Καθολική συμβατότητα |
| Πολλαπλά επίπεδα | Διανύσματα και ψηφιδωτά μαζί |
| Βασισμένο σε SQLite | Ελαφρύ και γρήγορο |

## Πρόσβαση

### Από το Μενού
**PyArchInit** → **Package for GeoPackage**

## Διεπαφή

### Πίνακας Εξαγωγής

```
+--------------------------------------------------+
|        Import to GeoPackage                       |
+--------------------------------------------------+
| Προορισμός:                                       |
|   [____________________________] [Browse]        |
+--------------------------------------------------+
| [Export Vector Layers]                           |
| [Export Raster Layers]                           |
+--------------------------------------------------+
```

## Διαδικασία

### Εξαγωγή Διανυσματικών Επιπέδων

1. Επιλέξτε τα επίπεδα προς εξαγωγή στον πίνακα Επιπέδων του QGIS
2. Ανοίξτε το εργαλείο Εξαγωγής GeoPackage
3. Καθορίστε τη διαδρομή προορισμού και το όνομα αρχείου
4. Κάντε κλικ στο **"Export Vector Layers"**

### Εξαγωγή Ψηφιδωτών Επιπέδων

1. Επιλέξτε ψηφιδωτά επίπεδα στον πίνακα Επιπέδων
2. Καθορίστε τον προορισμό (ίδιο αρχείο ή νέο)
3. Κάντε κλικ στο **"Export Raster Layers"**

### Συνδυασμένη Εξαγωγή

Για να συμπεριλάβετε διανύσματα και ψηφιδωτά στο ίδιο GeoPackage:
1. Πρώτα εξάγετε τα διανύσματα
2. Στη συνέχεια εξάγετε τα ψηφιδωτά στο ίδιο αρχείο
3. Το σύστημα προσθέτει επίπεδα στο υπάρχον GeoPackage

## Επιλογή Επιπέδων

### Μέθοδος

1. Στον πίνακα Επιπέδων του QGIS, επιλέξτε τα επιθυμητά επίπεδα
   - Ctrl+κλικ για πολλαπλή επιλογή
   - Shift+κλικ για εύρος
2. Ανοίξτε την Εξαγωγή GeoPackage
3. Τα επιλεγμένα επίπεδα θα εξαχθούν

### Συνιστώμενα Επίπεδα

| Επίπεδο | Τύπος | Σημειώσεις |
|---------|-------|-----------|
| pyunitastratigrafiche | Διανυσματικό | ΣΜ αποθέσεων |
| pyunitastratigrafiche_usm | Διανυσματικό | ΣΜ τοιχοποιίας |
| pyarchinit_quote | Διανυσματικό | Υψομετρικά σημεία |
| pyarchinit_siti | Διανυσματικό | Θέσεις |
| Ορθοφωτογραφία | Ψηφιδωτό | Ορθοφωτογραφία ανασκαφής |

## Έξοδος

## Επιλογές Εξαγωγής

### Διανυσματικά Επίπεδα

- Διατηρεί τις αρχικές γεωμετρίες
- Διατηρεί όλα τα χαρακτηριστικά
- Μετατρέπει αυτόματα ονόματα με κενά (χρησιμοποιεί κάτω παύλα)

### Ψηφιδωτά Επίπεδα

- Υποστηρίζει κοινές μορφές (GeoTIFF κ.λπ.)
- Διατηρεί τη γεωαναφορά
- Διατηρεί το σύστημα αναφοράς συντεταγμένων

## Τυπικές Χρήσεις

### Κοινή Χρήση Έργου

```
1. Επιλέξτε όλα τα επίπεδα του έργου
2. Εξάγετε σε GeoPackage
3. Μοιραστείτε το αρχείο .gpkg
4. Ο παραλήπτης ανοίγει απευθείας στο QGIS
```

### Αρχειοθέτηση Ανασκαφικής Περιόδου

```
1. Στο τέλος της ανασκαφής, επιλέξτε τα τελικά επίπεδα
2. Εξάγετε σε GeoPackage με ημερομηνία
3. Αρχειοθετήστε μαζί με την τεκμηρίωση
```

### Αντίγραφο Ασφαλείας GIS

```
1. Περιοδικά εξάγετε την τρέχουσα κατάσταση
2. Διατηρήστε εκδόσεις με ημερομηνία
3. Χρησιμοποιήστε για ανάκτηση μετά από καταστροφή
```

## Βέλτιστες Πρακτικές

### 1. Πριν την Εξαγωγή

- Επαληθεύστε την πληρότητα των επιπέδων
- Ελέγξτε το σύστημα αναφοράς συντεταγμένων
- Αποθηκεύστε το έργο QGIS

### 2. Ονοματοδοσία

- Χρησιμοποιήστε περιγραφικά ονόματα αρχείων
- Συμπεριλάβετε ημερομηνία στο όνομα
- Αποφύγετε ειδικούς χαρακτήρες

### 3. Επαλήθευση

- Ανοίξτε το δημιουργημένο GeoPackage
- Επαληθεύστε ότι υπάρχουν όλα τα επίπεδα
- Ελέγξτε τα χαρακτηριστικά

## Αντιμετώπιση Προβλημάτων

### Αποτυχία Εξαγωγής

**Αιτίες**:
- Μη έγκυρο επίπεδο
- Μη εγγράψιμη διαδρομή
- Ανεπαρκής χώρος δίσκου

**Λύσεις**:
- Επαληθεύστε την εγκυρότητα του επιπέδου
- Ελέγξτε τα δικαιώματα φακέλου
- Ελευθερώστε χώρο δίσκου

### Ελλείποντα Επίπεδα

**Αιτία**: Το επίπεδο δεν ήταν επιλεγμένο

**Λύση**: Επαληθεύστε την επιλογή στον πίνακα Επιπέδων

### Το Ψηφιδωτό Δεν Εξήχθη

**Αιτίες**:
- Μη υποστηριζόμενη μορφή
- Μη προσβάσιμο αρχείο πηγής

**Λύσεις**:
- Μετατρέψτε το ψηφιδωτό σε GeoTIFF
- Επαληθεύστε τη διαδρομή αρχείου πηγής
