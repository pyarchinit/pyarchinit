## Εισαγωγή

Το παράθυρο ρυθμίσεων του PyArchInit σας επιτρέπει να ορίσετε όλες τις παραμέτρους που είναι απαραίτητες για τη σωστή λειτουργία του πρόσθετου. Πριν ξεκινήσετε την τεκμηρίωση μιας αρχαιολογικής ανασκαφής, πρέπει να ρυθμίσετε σωστά τη σύνδεση με τη βάση δεδομένων και τις διαδρομές πόρων.

> **Εκπαιδευτικό Βίντεο**: [Εισαγωγή βίντεο για τη ρύθμιση]

## Πρόσβαση στις Ρυθμίσεις

Για να αποκτήσετε πρόσβαση στις ρυθμίσεις:
1. Ανοίξτε το QGIS
2. Μενού **PyArchInit** → **Config**

Ή από τη γραμμή εργαλείων του PyArchInit κάντε κλικ στο εικονίδιο **Ρυθμίσεις**.

## Καρτέλα Παραμέτρων Σύνδεσης

Αυτή είναι η κύρια καρτέλα για τη ρύθμιση της σύνδεσης με τη βάση δεδομένων.

![Καρτέλα Παραμέτρων Σύνδεσης](images/01_configurazione/03_tab_parametri_connessione.png)
*Εικόνα 3: Καρτέλα Παραμέτρων Σύνδεσης - Πλήρης προβολή*

### Ενότητα Ρυθμίσεων Βάσης Δεδομένων

| Πεδίο | Περιγραφή |
|-------|-----------|
| **Database** | Επιλέξτε τον τύπο βάσης δεδομένων: `sqlite` (τοπική) ή `postgres` (διακομιστής) |
| **Host** | Διεύθυνση διακομιστή PostgreSQL (π.χ., `localhost` ή IP διακομιστή) |
| **DBname** | Όνομα βάσης δεδομένων (π.χ., `pyarchinit`) |
| **Port** | Θύρα σύνδεσης (προεπιλογή: `5432` για PostgreSQL) |
| **User** | Όνομα χρήστη για τη σύνδεση |
| **Password** | Κωδικός πρόσβασης χρήστη |
| **SSL Mode** | Λειτουργία SSL για PostgreSQL: `allow`, `prefer`, `require`, `disable` |

![Ρυθμίσεις Βάσης Δεδομένων](images/01_configurazione/04_db_settings.png)
*Εικόνα 4: Ενότητα Ρυθμίσεων Βάσης Δεδομένων*

#### Επιλογή Βάσης Δεδομένων

**SQLite/Spatialite** (Συνιστάται για μεμονωμένο χρήστη):
- Τοπική βάση δεδομένων, δεν απαιτείται διακομιστής
- Ιδανική για ατομικά ή μικρά έργα
- Το αρχείο `.sqlite` αποθηκεύεται στον φάκελο `pyarchinit_DB_folder`

![Ρύθμιση SQLite](images/01_configurazione/05_config_sqlite.png)
*Εικόνα 5: Παράδειγμα ρύθμισης SQLite*

**PostgreSQL/PostGIS** (Συνιστάται για ομάδες):
- Βάση δεδομένων σε διακομιστή, πρόσβαση πολλαπλών χρηστών
- Απαιτεί εγκατεστημένο PostgreSQL με επέκταση PostGIS
- Υποστηρίζει διαχείριση χρηστών και δικαιωμάτων
- Ιδανική για μεγάλα έργα με πολλούς χειριστές

![Ρύθμιση PostgreSQL](images/01_configurazione/06_config_postgres.png)
*Εικόνα 6: Παράδειγμα ρύθμισης PostgreSQL*

> **Εκπαιδευτικό Βίντεο**: [Εισαγωγή βίντεο για τη ρύθμιση βάσης δεδομένων]

### Ενότητα Ρυθμίσεων Διαδρομών

| Πεδίο | Περιγραφή | Κουμπί |
|-------|-----------|--------|
| **Thumbnail path** | Διαδρομή αποθήκευσης μικρογραφιών εικόνων | `...` για αναζήτηση |
| **Image resize** | Διαδρομή για αλλαγή μεγέθους εικόνων | `...` για αναζήτηση |
| **Logo path** | Διαδρομή προσαρμοσμένου λογότυπου για αναφορές | `...` για αναζήτηση |

![Ρυθμίσεις Διαδρομών](images/01_configurazione/07_path_settings.png)
*Εικόνα 7: Ενότητα Ρυθμίσεων Διαδρομών*

#### Υποστηριζόμενες Απομακρυσμένες Διαδρομές

Το PyArchInit υποστηρίζει επίσης απομακρυσμένη αποθήκευση:
- **Google Drive**: `gdrive://folder/path/`
- **Dropbox**: `dropbox://folder/path/`
- **Amazon S3**: `s3://bucket/path/`
- **Cloudinary**: `cloudinary://cloud_name/folder/`
- **WebDAV**: `webdav://server/path/`
- **HTTP/HTTPS**: `https://server/path/`

![Απομακρυσμένη Αποθήκευση](images/01_configurazione/08_remote_storage.png)
*Εικόνα 8: Παράδειγμα ρύθμισης απομακρυσμένης αποθήκευσης*

### Ενότητα Ρυθμίσεων Θέσης

| Πεδίο | Περιγραφή |
|-------|-----------|
| **Site** | Προεπιλεγμένο όνομα θέσης για νέες εγγραφές |
| **Experimental features** | Ενεργοποίηση πειραματικών λειτουργιών (Ναι/Όχι) |

## Καρτέλα Εγκατάστασης Βάσης Δεδομένων

Αυτή η καρτέλα σας επιτρέπει να εγκαταστήσετε ή να αναβαθμίσετε τη βάση δεδομένων του PyArchInit.

![Καρτέλα Εγκατάστασης Βάσης Δεδομένων](images/01_configurazione/08_tab_installazione_db.png)
*Εικόνα 8: Καρτέλα Εγκατάστασης Βάσης Δεδομένων*

### Νέα Εγκατάσταση

1. Επιλέξτε **sqlite** ή **postgres** ως τύπο βάσης δεδομένων
2. Κάντε κλικ στο **Connect**
3. Αν η βάση δεδομένων δεν υπάρχει, κάντε κλικ στο **Install Database**
4. Το σύστημα θα δημιουργήσει όλους τους απαραίτητους πίνακες

### Αναβάθμιση Βάσης Δεδομένων

Για υπάρχουσες εγκαταστάσεις:
1. Κάντε κλικ στο **Database Check/Update**
2. Το σύστημα ελέγχει τη δομή και εφαρμόζει τυχόν ενημερώσεις

> **Προσοχή**: Δημιουργήστε πάντα αντίγραφο ασφαλείας της βάσης δεδομένων πριν από την αναβάθμιση!

## Αντιμετώπιση Προβλημάτων

### Αποτυχία Σύνδεσης

**Αιτίες**:
- Λανθασμένα διαπιστευτήρια
- Ο διακομιστής δεν εκτελείται
- Το τείχος προστασίας αποκλείει τη σύνδεση

**Λύσεις**:
- Επαληθεύστε όνομα χρήστη και κωδικό πρόσβασης
- Ελέγξτε αν η υπηρεσία PostgreSQL εκτελείται
- Ελέγξτε τους κανόνες του τείχους προστασίας

### Βάση Δεδομένων Δεν Βρέθηκε

**Αιτία**: Η βάση δεδομένων δεν έχει εγκατασταθεί ακόμα

**Λύση**: Μεταβείτε στην καρτέλα Εγκατάσταση Βάσης Δεδομένων και κάντε κλικ στο Install Database
