# Εκπαιδευτικό 12: Αναφορές και Εκτυπώσεις PDF

## Εισαγωγή

Το PyArchInit προσφέρει ένα ολοκληρωμένο σύστημα δημιουργίας **αναφορών PDF** για όλα τα αρχαιολογικά δελτία. Αυτή η λειτουργία επιτρέπει την εξαγωγή τεκμηρίωσης σε εκτυπώσιμη μορφή, σύμφωνη με τα υπουργικά πρότυπα και έτοιμη για αρχειοθέτηση.

### Διαθέσιμοι Τύποι Αναφορών

| Τύπος | Περιγραφή | Δελτίο Προέλευσης |
|-------|-----------|-------------------|
| Δελτία ΣΕ | Πλήρεις αναφορές ΣΕ/ΤΣΕ | Δελτίο ΣΕ |
| Ευρετήριο ΣΕ | Συνοπτικός κατάλογος ΣΕ | Δελτίο ΣΕ |
| Δελτία Περιοδολόγησης | Αναφορές περιόδων/φάσεων | Δελτίο Περιοδολόγησης |
| Δελτία Δομών | Αναφορές δομών | Δελτίο Δομής |
| Δελτία Ευρημάτων | Αναφορές καταγραφής υλικών | Δελτίο Καταγραφής |
| Δελτία Τάφων | Αναφορές ταφών | Δελτίο Τάφου |
| Δελτία Δειγμάτων | Αναφορές δειγμάτων | Δελτίο Δείγματος |
| Δελτία Ατόμων | Ανθρωπολογικές αναφορές | Δελτίο Ατόμου |

## Πρόσβαση στη Λειτουργία

### Από το Κύριο Μενού
1. **PyArchInit** στη γραμμή μενού
2. Επιλέξτε **Export PDF**

### Από τη Γραμμή Εργαλείων
Κάντε κλικ στο εικονίδιο **PDF** στη γραμμή εργαλείων του PyArchInit

## Διεπαφή Εξαγωγής

### Επισκόπηση Παραθύρου

Το παράθυρο εξαγωγής PDF παρουσιάζει:

```
+------------------------------------------+
|        PyArchInit - Export PDF            |
+------------------------------------------+
| Site: [Site selection ComboBox]    [v]   |
+------------------------------------------+
| Forms to export:                          |
| [x] SU Forms                              |
| [x] Periodization Forms                   |
| [x] Structure Forms                       |
| [x] Finds Forms                           |
| [x] Grave Forms                           |
| [x] Sample Forms                          |
| [x] Individual Forms                      |
+------------------------------------------+
| [Open Folder]  [Export PDF]  [Cancel]    |
+------------------------------------------+
```

### Επιλογή Θέσης

| Πεδίο | Περιγραφή |
|-------|-----------|
| Site ComboBox | Κατάλογος όλων των θέσεων στη βάση δεδομένων |

**Σημείωση**: Η εξαγωγή γίνεται ανά μεμονωμένη θέση. Για εξαγωγή πολλαπλών θέσεων, επαναλάβετε τη διαδικασία.

### Πλαίσια Ελέγχου Δελτίων

Κάθε πλαίσιο ελέγχου ενεργοποιεί την εξαγωγή ενός συγκεκριμένου τύπου:

| Πλαίσιο Ελέγχου | Παράγει |
|-----------------|---------|
| SU Forms | Πλήρη δελτία + Ευρετήριο ΣΕ |
| Periodization Forms | Δελτία περιόδων + Ευρετήριο |
| Structure Forms | Δελτία δομών + Ευρετήριο |
| Finds Forms | Δελτία υλικών + Ευρετήριο |
| Grave Forms | Δελτία ταφών + Ευρετήριο |
| Sample Forms | Δελτία δειγμάτων + Ευρετήριο |
| Individual Forms | Ανθρωπολογικά δελτία + Ευρετήριο |

## Διαδικασία Εξαγωγής

### Βήμα 1: Επιλογή Δεδομένων

```python
# Το σύστημα εκτελεί για κάθε επιλεγμένο τύπο:
1. Ερώτημα βάσης δεδομένων για την επιλεγμένη θέση
2. Ταξινόμηση δεδομένων (κατά αριθμό, τομέα κ.λπ.)
3. Προετοιμασία καταλόγου για δημιουργία
```

### Βήμα 2: Δημιουργία PDF

Για κάθε τύπο δελτίου:
1. **Μεμονωμένο δελτίο**: Λεπτομερές PDF για κάθε εγγραφή
2. **Ευρετήριο**: Συνοπτικό PDF με όλες τις εγγραφές

### Βήμα 3: Αποθήκευση

Εξαγωγή στον φάκελο:
```
~/pyarchinit/pyarchinit_PDF_folder/
```

## Περιεχόμενα Αναφορών

### Δελτίο ΣΕ

Πληροφορίες που περιλαμβάνονται:
- **Στοιχεία ταυτοποίησης**: Θέση, Τομέας, Αριθμός ΣΕ, Τύπος μονάδας
- **Ορισμοί**: Στρωματογραφικός, Ερμηνευτικός
- **Περιγραφή**: Πλήρες περιγραφικό κείμενο
- **Ερμηνεία**: Ερμηνευτική ανάλυση
- **Περιοδολόγηση**: Αρχική/Τελική Περίοδος/Φάση
- **Φυσικά χαρακτηριστικά**: Χρώμα, συνεκτικότητα, σχηματισμός
- **Μετρήσεις**: Ελάχ./μέγ. υψόμετρα, διαστάσεις
- **Σχέσεις**: Κατάλογος στρωματογραφικών σχέσεων
- **Τεκμηρίωση**: Γραφικές και φωτογραφικές αναφορές
- **Δεδομένα ΤΣΕ**: (εάν ισχύει) Τεχνική δόμησης, υλικά

### Ευρετήριο ΣΕ

Συνοπτικός πίνακας με στήλες:
| Θέση | Τομέας | ΣΕ | Στρωματ. Ορισμ. | Ερμηνευτ. Ορισμ. | Περίοδος |

### Δελτίο Περιοδολόγησης

- Θέση
- Αριθμός Περιόδου
- Αριθμός Φάσης
- Αρχική/τελική χρονολόγηση
- Εκτεταμένη χρονολόγηση
- Περιγραφή περιόδου

### Δελτίο Δομής

- Στοιχεία ταυτοποίησης (Θέση, Κωδικός, Αριθμός)
- Κατηγορία, Τύπος, Ορισμός
- Περιγραφή και Ερμηνεία
- Περιοδολόγηση
- Χρησιμοποιούμενα υλικά
- Δομικά στοιχεία
- Σχέσεις δομών
- Μετρήσεις και υψόμετρα

### Δελτίο Ευρημάτων

- Θέση, Αριθμός καταγραφής
- Τύπος ευρήματος, Ορισμός
- Περιγραφή
- Προέλευση (Τομέας, ΣΕ)
- Κατάσταση διατήρησης
- Χρονολόγηση
- Στοιχεία και μετρήσεις
- Βιβλιογραφία

### Δελτίο Τάφου

- Στοιχεία ταυτοποίησης
- Τελετουργία (ενταφιασμός/αποτέφρωση)
- Τύπος ταφής και εναπόθεσης
- Περιγραφή και ερμηνεία
- Κτερίσματα (παρουσία, τύπος, περιγραφή)
- Περιοδολόγηση
- Υψόμετρα δομής και ατόμου
- Συσχετιζόμενες ΣΕ

### Δελτίο Δείγματος

- Θέση, Αριθμός δείγματος
- Τύπος δείγματος
- Περιγραφή
- Προέλευση (Τομέας, ΣΕ)
- Τοποθεσία αποθήκευσης
- Αριθμός κιβωτίου

### Δελτίο Ατόμου

- Στοιχεία ταυτοποίησης
- Φύλο, Ηλικία (ελάχ./μέγ.), Ηλικιακές κατηγορίες
- Θέση σκελετού
- Προσανατολισμός (άξονας, αζιμούθιο)
- Κατάσταση διατήρησης
- Παρατηρήσεις

## Υποστηριζόμενες Γλώσσες

Το σύστημα δημιουργεί αναφορές βάσει της γλώσσας του συστήματος:

| Γλώσσα | Κωδικός | Πρότυπο |
|--------|---------|---------|
| Ιταλικά | IT | `build_*_sheets()` |
| Γερμανικά | DE | `build_*_sheets_de()` |
| Αγγλικά | EN | `build_*_sheets_en()` |

Η γλώσσα ανιχνεύεται αυτόματα από τις ρυθμίσεις του QGIS.

## Προτεινόμενη Ροή Εργασίας

```
1. Συμπληρώστε όλα τα δελτία της θέσης

3. Ελέγξτε την περιοδολόγηση
4. Επαληθεύστε τις στρωματογραφικές σχέσεις
```

### 2. Εξαγωγή

```
1. Ανοίξτε το Export PDF
2. Επιλέξτε θέση
3. Επιλέξτε τύπους δελτίων
4. Κάντε κλικ στο «Export PDF»
5. Περιμένετε την ολοκλήρωση
```

### 3. Επαλήθευση

```
1. Ανοίξτε τον φάκελο εξόδου
2. Ελέγξτε τα παραγόμενα PDF
3. Επαληθεύστε τη μορφοποίηση
4. Εκτυπώστε ή αρχειοθετήστε
```

## Αντιμετώπιση Προβλημάτων

### Σφάλμα: «No form to print»

**Αιτία**: Δεν βρέθηκαν εγγραφές για τον επιλεγμένο τύπο

**Λύση**:
- Επαληθεύστε ότι υπάρχουν δεδομένα για την επιλεγμένη θέση
- Ελέγξτε τη βάση δεδομένων

### Κενά ή Ελλιπή PDF

**Πιθανές αιτίες**:
1. Τα υποχρεωτικά πεδία δεν έχουν συμπληρωθεί
2. Προβλήματα κωδικοποίησης χαρακτήρων
3. Απουσία γραμματοσειρών

**Λύσεις**:
- Συμπληρώστε τα υποχρεωτικά πεδία
- Επαληθεύστε την εγκατάσταση της γραμματοσειράς Cambria

### Σφάλμα Γραμματοσειράς

**Αιτία**: Η γραμματοσειρά Cambria δεν είναι εγκατεστημένη

**Λύση**:
- Το πρόσθετο επιχειρεί αυτόματη εγκατάσταση
- Σε περίπτωση προβλημάτων, εγκαταστήστε χειροκίνητα

### Αργή Εξαγωγή

**Αιτία**: Πολλές εγγραφές προς εξαγωγή

**Λύση**:
- Εξαγωγή κατά τύπο ξεχωριστά
- Περιμένετε την ολοκλήρωση

## Βέλτιστες Πρακτικές

### 1. Οργάνωση

- Εξαγωγή τακτικά κατά τη διάρκεια της ανασκαφής
- Δημιουργία αντιγράφων ασφαλείας των παραγόμενων PDF
- Οργάνωση ανά ανασκαφική περίοδο/έτος

### 2. Πληρότητα Δεδομένων

- Συμπληρώστε όλα τα πεδία πριν από την εξαγωγή
- Επαληθεύστε τα υψόμετρα από μετρήσεις GIS
- Ελέγξτε τις στρωματογραφικές σχέσεις

### 3. Αρχειοθέτηση

- Αποθηκεύστε τα PDF σε ασφαλή αποθηκευτικό χώρο
- Συμπεριλάβετέ τα στην τελική τεκμηρίωση
- Επισυνάψτε τα στην αναφορά ανασκαφής

### 4. Εκτύπωση

- Χρησιμοποιήστε χαρτί χωρίς οξέα για αρχειοθέτηση
- Εκτυπώστε σε μορφή A4
- Βιβλιοδεσία ανά ανασκαφική περίοδο

## Ενσωμάτωση με Άλλες Λειτουργίες

Το σύστημα ανακτά αυτόματα:
- Ελάχιστα και μέγιστα υψόμετρα από τις γεωμετρίες

- Συνδεδεμένες φωτογραφίες
- Σχέδια και αποτυπώσεις

### Περιοδολόγηση

- Εκτεταμένη χρονολόγηση από την αντιστοιχισμένη περίοδο/φάση
- Χρονολογικές αναφορές
