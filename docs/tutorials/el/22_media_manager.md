# Εκπαιδευτικό 22: Διαχειριστής Πολυμέσων

## Εισαγωγή

Ο **Διαχειριστής Πολυμέσων** είναι το κεντρικό εργαλείο του PyArchInit για τη διαχείριση εικόνων και πολυμεσικού περιεχομένου που συσχετίζεται με αρχαιολογικές εγγραφές. Επιτρέπει τη σύνδεση φωτογραφιών, σχεδίων, βίντεο και άλλων πολυμέσων με ΣΕ, ευρήματα, τάφους, δομές και άλλες οντότητες.

### Κύρια Χαρακτηριστικά

- Κεντρική διαχείριση όλων των πολυμέσων
- Σύνδεση με αρχαιολογικές οντότητες (ΣΕ, Ευρήματα, Κεραμική, Τάφοι, Δομές, ΤΜ)
- Προβολή μικρογραφιών και εικόνων πλήρους μεγέθους
- Ετικετοποίηση και κατηγοριοποίηση
- Προχωρημένη αναζήτηση
- Ενσωμάτωση GPT για ανάλυση εικόνων

## Πρόσβαση

### Από το Μενού
**PyArchInit** → **Media Manager**

### Από τη Γραμμή Εργαλείων
Εικονίδιο **Media Manager** στη γραμμή εργαλείων του PyArchInit

### Φίλτρα Αναζήτησης

| Πεδίο | Περιγραφή |
|-------|-----------|
| Site | Φίλτρο κατά αρχαιολογική θέση |
| Area | Φίλτρο κατά τομέα ανασκαφής |
| SU | Φίλτρο κατά Στρωματογραφική Ενότητα |
| Structure Code | Φίλτρο κατά κωδικό δομής |
| Structure No. | Φίλτρο κατά αριθμό δομής |

### Στοιχεία Ελέγχου Μικρογραφιών

| Στοιχείο Ελέγχου | Λειτουργία |
|-------------------|------------|
| Ρυθμιστικό μεγέθους | Προσαρμογή μεγέθους μικρογραφιών |
| Διπλό κλικ | Άνοιγμα εικόνας σε αρχικό μέγεθος |
| Πολλαπλή επιλογή | Ctrl+κλικ για επιλογή πολλαπλών εικόνων |

## Διαχείριση Πολυμέσων

### Προσθήκη Νέων Εικόνων

1. Ανοίξτε τον Διαχειριστή Πολυμέσων
2. Επιλέξτε θέση προορισμού
3. Κάντε κλικ στο **«New Record»** ή χρησιμοποιήστε μενού περιβάλλοντος
4. Επιλέξτε εικόνες προς προσθήκη
5. Συμπληρώστε τα μεταδεδομένα

### Σύνδεση Πολυμέσων με Οντότητες

1. Επιλέξτε εικόνα στο πλέγμα
2. Στον πίνακα Ετικετών, επιλέξτε:
   - **Τύπος οντότητας**: ΣΕ, Εύρημα, Κεραμική, Τάφος, Δομή, ΤΜ
   - **Αναγνωριστικό**: Αριθμός/κωδικός οντότητας
3. Κάντε κλικ στο **«Link»**

### Υποστηριζόμενοι Τύποι Οντοτήτων

| Τύπος | Πίνακας ΒΔ | Περιγραφή |
|-------|------------|-----------|
| SU | us_table | Στρωματογραφικές Ενότητες |
| FIND | inventario_materiali_table | Ευρήματα/Υλικά |
| POTTERY | pottery_table | Κεραμική |
| GRAVE | tomba_table | Ταφές |
| STRUCTURE | struttura_table | Δομές |
| TU | ut_table | Τοπογραφικές Μονάδες |

### Προβολή Αρχικής Εικόνας

- **Διπλό κλικ** στη μικρογραφία
- Ανοίγει προβολέας με:
  - Μεγέθυνση (τροχός ποντικιού)
  - Μετακίνηση (σύρσιμο)
  - Περιστροφή
  - Μέτρηση

## Προχωρημένες Λειτουργίες

### Προχωρημένη Αναζήτηση

Ο Διαχειριστής Πολυμέσων υποστηρίζει αναζήτηση κατά:
- Όνομα αρχείου
- Ημερομηνία καταχώρησης
- Συνδεδεμένη οντότητα
- Ετικέτες/κατηγορίες

### Ενσωμάτωση GPT

Κουμπί **«GPT Sketch»** για:
- Αυτόματη ανάλυση εικόνας
- Δημιουργία περιγραφής
- Προτάσεις ταξινόμησης

### Απομακρυσμένη Φόρτωση

Υποστήριξη εικόνων σε απομακρυσμένους εξυπηρετητές:
- Απευθείας URL
- Εξυπηρετητές FTP
- Αποθήκευση νέφους

## Βάση Δεδομένων

### Εμπλεκόμενοι Πίνακες

| Πίνακας | Περιγραφή |
|---------|-----------|
| `media_table` | Μεταδεδομένα πολυμέσων |
| `media_thumb_table` | Μικρογραφίες |
| `media_to_entity_table` | Συνδέσεις οντοτήτων |

### Κλάσεις Αντιστοίχισης

- `MEDIA` - Κύρια οντότητα πολυμέσων
- `MEDIA_THUMB` - Μικρογραφίες
- `MEDIATOENTITY` - Σχέση πολυμέσου-οντότητας

## Βέλτιστες Πρακτικές

### 1. Οργάνωση Αρχείων

- Χρησιμοποιήστε περιγραφικά ονόματα αρχείων
- Οργανώστε ανά θέση/τομέα/έτος
- Κρατήστε αντίγραφα ασφαλείας πρωτοτύπων

### 2. Μεταδεδομένα

- Πάντα συμπληρώνετε θέση και τομέα
- Προσθέστε ουσιαστικές περιγραφές
- Χρησιμοποιήστε συνεπείς ετικέτες

### 3. Ποιότητα Εικόνων

- Ελάχιστη συνιστώμενη ανάλυση: 1920x1080
- Μορφή: JPG για φωτογραφίες, PNG για σχέδια
- Μέτρια συμπίεση

### 4. Συνδέσεις

- Συνδέστε κάθε εικόνα με σχετικές οντότητες
- Επαληθεύστε τις συνδέσεις μετά από μαζική εισαγωγή
- Χρησιμοποιήστε αναζήτηση για ασύνδετες εικόνες

## Αντιμετώπιση Προβλημάτων

### Οι Μικρογραφίες δεν Εμφανίζονται

**Αιτίες**:
- Λανθασμένη διαδρομή εικόνας
- Ελλιπές αρχείο
- Προβλήματα δικαιωμάτων

**Λύσεις**:
- Επαληθεύστε τη διαδρομή στη βάση δεδομένων
- Ελέγξτε την ύπαρξη αρχείου
- Ελέγξτε τα δικαιώματα φακέλου

### Η Εικόνα δεν Μπορεί να Συνδεθεί

**Αιτίες**:
- Η οντότητα δεν υπάρχει
- Λανθασμένος τύπος οντότητας

**Λύσεις**:
- Επαληθεύστε ότι η εγγραφή υπάρχει
- Ελέγξτε τον επιλεγμένο τύπο οντότητας
