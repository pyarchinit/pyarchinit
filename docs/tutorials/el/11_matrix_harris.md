# Εκπαιδευτικό 11: Harris Matrix

## Εισαγωγή

Το **Harris Matrix** (ή στρωματογραφικό διάγραμμα) είναι θεμελιώδες εργαλείο στην αρχαιολογία για τη γραφική αναπαράσταση στρωματογραφικών σχέσεων μεταξύ διαφορετικών Στρωματογραφικών Μονάδων (ΣΜ). Το PyArchInit δημιουργεί αυτόματα το Harris Matrix από τις στρωματογραφικές σχέσεις που εισάγονται στα δελτία ΣΜ.

### Τι είναι το Harris Matrix;

Το Harris Matrix είναι ένα διάγραμμα που αναπαριστά:
- Τη **χρονική ακολουθία** των ΣΜ (από πιο πρόσφατη στην κορυφή μέχρι αρχαιότερη στη βάση)
- Τις **φυσικές σχέσεις** μεταξύ ΣΜ (καλύπτει/καλύπτεται από, τέμνει/τέμνεται από, δένει με)
- Την **περιοδολόγηση** της ανασκαφής (ομαδοποίηση κατά περιόδους και φάσεις)

### Τύποι Αναπαριστώμενων Σχέσεων

| Σχέση | Σημασία | Αναπαράσταση |
|-------|---------|--------------|
| Καλύπτει/Καλύπτεται από | Φυσική επικάλυψη | Συμπαγής γραμμή προς τα κάτω |
| Τέμνει/Τέμνεται από | Αρνητική ενέργεια (διεπιφάνεια) | Διακεκομμένη γραμμή |
| Δένει με/Ισοδυναμεί με | Σύγχρονη κατασκευή | Οριζόντια αμφίδρομη γραμμή |
| Πληρώνει/Πληρώνεται από | Πλήρωση τομής | Συμπαγής γραμμή |
| Εφάπτεται σε/Υποστηρίζει | Δομική στήριξη | Συμπαγής γραμμή |

## Πρόσβαση στη Λειτουργία

### Από το Κύριο Μενού
1. **PyArchInit** στη γραμμή μενού
2. Επιλέξτε **Harris Matrix**

### Από το Δελτίο ΣΜ
1. Ανοίξτε το Δελτίο ΣΜ
2. Καρτέλα **Map**
3. Κουμπί **"Export Matrix"** ή **"View Matrix"**

### Προαπαιτούμενα
- Βάση δεδομένων σωστά συνδεδεμένη
- ΣΜ με συμπληρωμένες στρωματογραφικές σχέσεις
- Καθορισμένη περιοδολόγηση (προαιρετικό αλλά συνιστάται)
- Graphviz εγκατεστημένο στο σύστημα

## Τύποι Εξαγωγής

### 1. Τυπική Εξαγωγή Matrix

Δημιουργεί βασικό matrix με:
- Όλες τις στρωματογραφικές σχέσεις
- Ομαδοποίηση περιόδου/φάσης
- Κατακόρυφη διάταξη (TB - Πάνω προς Κάτω)

**Αποτέλεσμα**: `pyarchinit_Matrix_folder/Harris_matrix.jpg`

### 2. Εκτεταμένη Εξαγωγή Matrix (2ED)

Εκτεταμένη έκδοση με:
- Πρόσθετες πληροφορίες κόμβων (ΣΜ + ορισμός + χρονολόγηση)
- Ειδικές συνδέσεις (>, >>)
- Εξαγωγή μορφής GraphML

**Αποτέλεσμα**: `pyarchinit_Matrix_folder/Harris_matrix2ED.jpg`

### 3. Προβολή Matrix (Γρήγορη Απεικόνιση)

Για γρήγορη προβολή χωρίς επιλογές ρύθμισης:
- Χρησιμοποιεί προεπιλεγμένες ρυθμίσεις
- Ταχύτερη δημιουργία
- Ιδανικό για γρήγορους ελέγχους

## Διαδικασία Δημιουργίας

### Βήμα 1: Συλλογή Δεδομένων

Το σύστημα συλλέγει αυτόματα:
```
Για κάθε ΣΜ στην επιλεγμένη θέση/τομέα:
  - Αριθμός ΣΜ
  - Τύπος μονάδας (SU/WSU)
  - Στρωματογραφικές σχέσεις
  - Αρχική περίοδος και φάση
  - Ερμηνευτικός ορισμός
```

### Βήμα 2: Κατασκευή Γραφήματος

Δημιουργία σχέσεων:
```
Ακολουθία (Πριν/Μετά):
  US1 -> US2 (US1 καλύπτει US2)

Αρνητική (Τομές):
  US3 -> US4 (US3 τέμνει US4)

Σύγχρονη:
  US5 <-> US6 (US5 δένει με US6)
```

### Βήμα 3: Ομαδοποίηση Περιόδων

Ιεραρχική ομαδοποίηση:
```
Θέση
  └── Τομέας
      └── Περίοδος 1 : Φάση 1 : "Ρωμαϊκή Εποχή"
          ├── US101
          ├── US102
          └── US103
      └── Περίοδος 1 : Φάση 2 : "Ύστερη Αρχαιότητα"
          ├── US201
          └── US202
```

### Βήμα 4: Μεταβατική Μείωση (tred)

Η εντολή `tred` του Graphviz αφαιρεί πλεονάζουσες σχέσεις:
- Αν US1 -> US2 και US2 -> US3, αφαιρεί US1 -> US3
- Απλοποιεί το διάγραμμα
- Διατηρεί μόνο άμεσες σχέσεις

### Βήμα 5: Τελική Απόδοση

Δημιουργία εικόνας σε πολλαπλές μορφές:
- DOT (πηγαίο αρχείο Graphviz)
- JPG (συμπιεσμένη εικόνα)
- PNG (εικόνα χωρίς απώλειες)

## Ερμηνεία Matrix

### Κατακόρυφη Ανάγνωση

```
     [Πιο πρόσφατη ΣΜ]
           ↓
        US 001
           ↓
        US 002
           ↓
        US 003
           ↓
     [Αρχαιότερη ΣΜ]
```

### Ανάγνωση Ομάδων

Τα χρωματιστά πλαίσια αντιπροσωπεύουν περιόδους/φάσεις:
- **Γαλάζιο**: Ομάδα περιόδου
- **Κίτρινο**: Ομάδα φάσης
- **Γκρι**: Υπόβαθρο θέσης

### Τύποι Συνδέσεων

```
─────────→  Συμπαγής γραμμή = Καλύπτει/Πληρώνει/Εφάπτεται
- - - - →  Διακεκομμένη γραμμή = Τέμνει
←────────→  Αμφίδρομη = Σύγχρονο/Ισοδυναμεί
```

### Χρώματα Κόμβων

| Χρώμα | Τυπική Σημασία |
|-------|----------------|
| Λευκό | Κανονική ΣΜ απόθεσης |
| Γκρι | Αρνητική ΣΜ (τομή) |
| Κίτρινο | Σύγχρονη ΣΜ |
| Μπλε | ΣΜ με ειδικές σχέσεις |

## Αντιμετώπιση Προβλημάτων

### Σφάλμα: "Loop Detected"

**Αιτία**: Υπάρχουν κύκλοι στις σχέσεις (Α καλύπτει Β, Β καλύπτει Α)

**Λύση**:
1. Ανοίξτε το Δελτίο ΣΜ
2. Επαληθεύστε τις σχέσεις των υποδεικνυόμενων ΣΜ
3. Διορθώστε τις κυκλικές σχέσεις
4. Αναδημιουργήστε το matrix

### Σφάλμα: "tred command not found"

**Αιτία**: Το Graphviz δεν είναι εγκατεστημένο

**Λύση**:
- **Windows**: Εγκαταστήστε Graphviz από graphviz.org
- **macOS**: `brew install graphviz`
- **Linux**: `sudo apt install graphviz`

### Το Matrix Δεν Δημιουργήθηκε

**Πιθανές αιτίες**:
1. Δεν έχουν εισαχθεί στρωματογραφικές σχέσεις
2. ΣΜ χωρίς αποδοθείσα περίοδο/φάση
3. Προβλήματα δικαιωμάτων στον φάκελο εξόδου

**Έλεγχος**:
1. Επαληθεύστε ότι οι ΣΜ έχουν σχέσεις
2. Επαληθεύστε περιοδολόγηση
3. Ελέγξτε δικαιώματα στον `pyarchinit_Matrix_folder`

### Πολύ Μεγάλο Matrix

**Πρόβλημα**: Μη αναγνώσιμη εικόνα με πολλές ΣΜ

**Λύσεις**:
1. Μειώστε DPI (100-150)
2. Φιλτράρετε κατά συγκεκριμένο τομέα
3. Χρησιμοποιήστε View Matrix για μεμονωμένους τομείς
4. Εξαγάγετε σε διανυσματική μορφή (DOT) και ανοίξτε με yEd

## Βέλτιστες Πρακτικές

### 1. Πριν τη Δημιουργία

- Επαληθεύστε πληρότητα στρωματογραφικών σχέσεων
- Ελέγξτε απουσία κύκλων
- Αναθέστε περίοδο/φάση σε όλες τις ΣΜ
- Συμπληρώστε ερμηνευτικό ορισμό

### 2. Κατά τη Σύνταξη ΣΜ

- Εισαγάγετε σωστές αμφίδρομες σχέσεις
- Χρησιμοποιήστε συνεπή ορολογία
- Επαληθεύστε σωστό τομέα στις σχέσεις

### 3. Βελτιστοποίηση Αποτελεσμάτων

- Για εκτύπωση: DPI 300
- Για οθόνη: DPI 150
- Για πολύπλοκες ανασκαφές: διαιρέστε κατά τομείς

### 4. Ποιοτικός Έλεγχος

- Συγκρίνετε matrix με τεκμηρίωση ανασκαφής
- Επαληθεύστε λογικές ακολουθίες
- Ελέγξτε ομαδοποιήσεις περιόδων
