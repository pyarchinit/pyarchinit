## Εισαγωγη

Το **GeoArchaeo** ειναι η μοναδα γεωστατιστικης αναλυσης ενσωματωμενη στο PyArchInit. Παρεχει ενα ολοκληρωμενο συνολο εργαλειων για τη χωρικη αναλυση αρχαιολογικων δεδομενων, απο τη μοντελοποιηση βαριογραμματων εως τη παρεμβολη kriging, τις προβλεψεις μεσω μηχανικης μαθησης και τον σχεδιασμο στρατηγικων δειγματοληψιας.

<!-- VIDEO: Εισαγωγη στο GeoArchaeo -->
> **Εκπαιδευτικο βιντεο**: [Εισαγετε συνδεσμο βιντεο εισαγωγης GeoArchaeo]

### Γιατι γεωστατιστικη αναλυση στην αρχαιολογια;

Η γεωστατιστικη αναλυση επιτρεπει:

- **Παρεμβολη** τιμων μεταξυ γνωστων σημειων δειγματοληψιας, δημιουργωντας συνεχεις επιφανειες απο διακριτα δεδομενα
- **Ποσοτικοποιηση** της χωρικης συσχετισης στα αρχαιολογικα δεδομενα (π.χ. πυκνοτητα ευρηματων, παχος στρωματων)
- **Προβλεψη** χωρικων κατανομων σε περιοχες που δεν εχουν ανασκαφει ακομα
- **Βελτιστοποιηση** στρατηγικων δειγματοληψιας για μελλοντικες ερευνες
- **Δημιουργια** ολοκληρωμενων αναλυτικων αναφορων για επιστημονικη τεκμηριωση

### Επισκοπηση ροης εργασιας

```
1. Φορτωση δεδομενων    2. Βαριογραμμα         3. Kriging/ML
   (Καρτελα Δεδομενα)      (Καρτελα Βαριογρ.)     (Καρτελα Kriging/ML)
        |                      |                      |
   Επιλογη στρωματος     Υπολογισμος και        Παρεμβολη η
   και πεδιων            μοντελοποιηση          χωρικη προβλεψη
                          βαριογραμματος
                               |                      |
                          4. Δειγματοληψια      5. Αναφορα
                             (Καρτελα Δειγμ.)      (Καρτελα Αναφορα)
                                  |                      |
                             Σχεδιασμος           Δημιουργια
                             στρατηγικης          αναφορας αναλυσης
```

## Προσβαση στο εργαλειο

Το GeoArchaeo ειναι προσβασιμο απο τη γραμμη εργαλειων του PyArchInit μεσω του αναπτυσσομενου κουμπιου Εργαλεια Αναλυσης.

### Απο τη γραμμη εργαλειων

1. Εντοπιστε το κουμπι **Εργαλεια Αναλυσης** (αναπτυσσομενο εικονιδιο) στη γραμμη εργαλειων του PyArchInit
2. Κανε κλικ στο βελος του αναπτυσσομενου μενου
3. Επιλεξτε **GeoArchaeo** απο τη λιστα

<!-- IMAGE: Κουμπι Εργαλεια Αναλυσης στη γραμμη εργαλειων -->
> **Εικ. 1**: Το αναπτυσσομενο μενου Εργαλεια Αναλυσης στη γραμμη εργαλειων του PyArchInit

Ο πινακας GeoArchaeo εμφανιζεται ως **αγκυρωσιμο widget** στη διεπαφη του QGIS. Μπορει να μετακινηθει, να αλλαξει μεγεθος η να αποσπαστει οπως οποιοσδηποτε αλλος πινακας του QGIS.

<!-- IMAGE: Πινακας GeoArchaeo αγκυρωμενος στο QGIS -->
> **Εικ. 2**: Ο πινακας GeoArchaeo αγκυρωμενος στο παραθυρο του QGIS

### Επιλογεας γλωσσας

Ο πινακας GeoArchaeo περιλαμβανει εναν **επιλογεα γλωσσας** στο επανω μερος, που επιτρεπει την αλλαγη της γλωσσας της διεπαφης χωρις επανεκκινηση του QGIS. Οι υποστηριζομενες γλωσσες περιλαμβανουν ιταλικα, αγγλικα, γερμανικα, γαλλικα, ισπανικα, αραβικα, καταλανικα, ρουμανικα, πορτογαλικα και ελληνικα.

## Διεπαφη χρηστη

Ο πινακας GeoArchaeo ειναι οργανωμενος σε **6 κυριες καρτελες**, καθε μια αφιερωμενη σε μια φαση της ροης εργασιας γεωστατιστικης αναλυσης.

| Καρτελα | Εικονιδιο | Λειτουργια |
|---------|-----------|------------|
| **Δεδομενα** | Πινακας | Φορτωση και εξερευνηση χωρικων δεδομενων απο στρωματα QGIS |
| **Βαριογραμμα** | Γραφημα | Υπολογισμος και μοντελοποιηση πειραματικων βαριογραμματων |
| **Kriging** | Πλεγμα | Εκτελεση παρεμβολης kriging (συνηθες, καθολικο) |
| **ML** | Εγκεφαλος | Χωρικες προβλεψεις μεσω μηχανικης μαθησης |
| **Δειγματοληψια** | Στοχος | Σχεδιασμος στρατηγικων δειγματοληψιας για αρχαιολογικες ερευνες |
| **Αναφορα** | Εγγραφο | Δημιουργια αναφορων αναλυσης |

<!-- IMAGE: Επισκοπηση των 6 καρτελων του GeoArchaeo -->
> **Εικ. 3**: Οι εξι καρτελες του πινακα GeoArchaeo

### Γραμμη εργαλειων πινακα

Στο επανω μερος του πινακα θα βρειτε:

- **Επιλογεας γλωσσας**: Αναπτυσσομενο μενου για αλλαγη γλωσσας διεπαφης
- **Φορτωση δεδομενων παραδειγματος**: Κουμπι για φορτωση δοκιμαστικου συνολου δεδομενων
- **Βοηθεια**: Κουμπι για προσβαση στην τεκμηριωση

## Καρτελα Δεδομενα

Η καρτελα **Δεδομενα** ειναι το σημειο εκκινησης για οποιαδηποτε γεωστατιστικη αναλυση. Επιτρεπει τη φορτωση και την προβολη χωρικων δεδομενων που ειναι διαθεσιμα στα στρωματα QGIS.

### Φορτωση δεδομενων

1. Ανοιξτε την καρτελα **Δεδομενα**
2. Επιλεξτε ενα **διανυσματικο στρωμα** απο το αναπτυσσομενο μενου (εμφανιζονται ολα τα στρωματα σημειων του εργου QGIS)
3. Επιλεξτε το **πεδιο αναλυσης** (το αριθμητικο πεδιο για αναλυση)
4. Κανε κλικ στο **Φορτωση δεδομενων**

<!-- IMAGE: Καρτελα Δεδομενα με επιλεγμενο στρωμα και πεδιο -->
> **Εικ. 4**: Η καρτελα Δεδομενα με ενα στρωμα και πεδιο αναλυσης επιλεγμενα

### Δεδομενα παραδειγματος

Για να εξοικειωθειτε με το εργαλειο, μπορειτε να φορτωσετε ενα **συνολο δεδομενων παραδειγματος** κανοντας κλικ στο αντιστοιχο κουμπι. Το συνολο δεδομενων παραδειγματος περιεχει προσομοιωμενα αρχαιολογικα δεδομενα με συντεταγμενες και αριθμητικες τιμες καταλληλες για γεωστατιστικη αναλυση.

### Εξερευνηση δεδομενων

Μετα τη φορτωση, η καρτελα εμφανιζει:

| Πληροφορια | Περιγραφη |
|------------|-----------|
| **Αριθμος σημειων** | Συνολικα φορτωμενα σημεια |
| **Εκταση** | Πλαισιο οριων του συνολου δεδομενων (xmin, ymin, xmax, ymax) |
| **Στατιστικα** | Μεσος ορος, διαμεσος, τυπικη αποκλιση, ελαχιστο, μεγιστο |
| **Προεπισκοπηση** | Πινακας με τις πρωτες γραμμες του συνολου δεδομενων |

### Απαιτησεις δεδομενων

- Το στρωμα πρεπει να ειναι **διανυσματικο στρωμα σημειων**
- Το πεδιο αναλυσης πρεπει να περιεχει **αριθμητικες τιμες**
- Τα σημεια πρεπει να εχουν **εγκυρες συντεταγμενες** στο συστημα αναφορας του εργου
- Συνισταται τουλαχιστον **30 σημεια** για σημαντικη γεωστατιστικη αναλυση

## Καρτελα Βαριογραμμα

Η καρτελα **Βαριογραμμα** επιτρεπει τον υπολογισμο και τη μοντελοποιηση πειραματικων βαριογραμματων, τα οποια περιγραφουν τη δομη της χωρικης συσχετισης στα δεδομενα.

### Τι ειναι ενα βαριογραμμα;

Ενα βαριογραμμα ειναι ενα γραφημα που δειχνει πως η **διακυμανση** μεταξυ ζευγων σημειων αλλαζει ως συναρτηση της **αποστασης** που τα χωριζει. Οι βασικες παραμετροι ειναι:

| Παραμετρος | Περιγραφη |
|------------|-----------|
| **Nugget** | Διακυμανση σε μηδενικη αποσταση (σφαλμα μετρησης + μικροκλιμακη μεταβλητοτητα) |
| **Sill** | Μεγιστη επιτευχθεισα διακυμανση (οροπεδιο του βαριογραμματος) |
| **Range** | Αποσταση περαν της οποιας δεν υπαρχει πλεον χωρικη συσχετιση |

### Υπολογισμος πειραματικου βαριογραμματος

1. Βεβαιωθειτε οτι εχετε φορτωσει δεδομενα στην καρτελα Δεδομενα
2. Ανοιξτε την καρτελα **Βαριογραμμα**
3. Ρυθμιστε τις παραμετρους:
   - **Αριθμος lag**: Αριθμος διαστηματων αποστασης (προεπιλογη: 15)
   - **Μεγιστη αποσταση**: Μεγιστη αποσταση προς εξεταση (προεπιλογη: αυτοματη)
   - **Γωνιακη ανοχη**: Για κατευθυντικα βαριογραμματα (προεπιλογη: πανκατευθυντικο)
4. Κανε κλικ στο **Υπολογισμος βαριογραμματος**

<!-- IMAGE: Υπολογισμενο πειραματικο βαριογραμμα -->
> **Εικ. 5**: Ενα πειραματικο βαριογραμμα υπολογισμενο απο αρχαιολογικα δεδομενα

### Μοντελοποιηση βαριογραμματος

Μετα τον υπολογισμο του πειραματικου βαριογραμματος, μπορειτε να προσαρμοσετε ενα **θεωρητικο μοντελο**:

1. Επιλεξτε τον **τυπο μοντελου**:
   - **Σφαιρικο**: Το πιο κοινο μοντελο, φτανει το sill σε πεπερασμενη αποσταση
   - **Εκθετικο**: Φτανει το sill ασυμπτωτικα
   - **Γκαουσιανο**: Σταδιακη μεταβαση, καταλληλο για πολυ ομαλα φαινομενα
   - **Γραμμικο**: Βαριογραμμα χωρις καθορισμενο sill
2. Κανε κλικ στο **Προσαρμογη μοντελου**
3. Ελεγξτε τις εκτιμωμενες παραμετρους (nugget, sill, range) και την ποιοτητα προσαρμογης

<!-- IMAGE: Προσαρμοσμενο μοντελο βαριογραμματος -->
> **Εικ. 6**: Σφαιρικο μοντελο προσαρμοσμενο στο πειραματικο βαριογραμμα

### Κατευθυντικα βαριογραμματα

Για ελεγχο **ανισοτροπιας** (μεταβολη της χωρικης δομης σε διαφορετικες κατευθυνσεις):

1. Ρυθμιστε μια **γωνιακη ανοχη** (π.χ. 22,5 μοιρες)
2. Επιλεξτε τις **κατευθυνσεις** για αναλυση (0, 45, 90, 135 μοιρες)
3. Συγκρινετε τα βαριογραμματα στις διαφορετικες κατευθυνσεις

## Καρτελα Kriging

Η καρτελα **Kriging** επιτρεπει την εκτελεση παρεμβολης kriging, της μεθοδου αναφορας στη γεωστατιστικη για βελτιστη χωρικη προβλεψη.

### Διαθεσιμοι τυποι kriging

| Τυπος | Περιγραφη | Ποτε χρησιμοποιειται |
|-------|-----------|---------------------|
| **Συνηθες kriging** | Υποθετει σταθερο αλλα αγνωστο τοπικο μεσο ορο | Πιο κοινη περιπτωση, στασιμα δεδομενα |
| **Καθολικο kriging** | Λαμβανει υποψη μια χωρικη ταση (drift) | Οταν τα δεδομενα εμφανιζουν κατευθυντικη ταση |

### Εκτελεση kriging

1. Βεβαιωθειτε οτι εχετε μοντελοποιησει το βαριογραμμα στην καρτελα Βαριογραμμα
2. Ανοιξτε την καρτελα **Kriging**
3. Επιλεξτε τον **τυπο kriging** (συνηθες η καθολικο)
4. Ρυθμιστε τις παραμετρους του πλεγματος εξοδου:
   - **Αναλυση**: Μεγεθος κελιων πλεγματος (σε μοναδες CRS)
   - **Εκταση**: Αυτοματη απο το συνολο δεδομενων η προσαρμοσμενη
5. Ρυθμιστε τις παραμετρους kriging:
   - **Ελαχιστα σημεια**: Ελαχιστος αριθμος γειτονικων σημειων
   - **Μεγιστα σημεια**: Μεγιστος αριθμος γειτονικων σημειων
   - **Ακτινα αναζητησης**: Μεγιστη αποσταση για γειτονικα σημεια
6. Κανε κλικ στο **Εκτελεση kriging**

<!-- IMAGE: Αποτελεσμα παρεμβολης kriging -->
> **Εικ. 7**: Χαρτης παρεμβολης kriging με το πλεγμα προβλεψης

### Αποτελεσματα kriging

Η αναλυση παραγει δυο στρωματα raster:

- **Προβλεψη**: Οι παρεμβεβλημενες τιμες στο πλεγμα
- **Διακυμανση kriging**: Η αβεβαιοτητα της προβλεψης σε καθε κελι

Τα στρωματα προστιθενται αυτοματα στο εργο QGIS και εμφανιζονται στον χαρτη.

> **Σημειωση**: Η αναλυση εκτελειται σε **νημα παρασκηνιου**, ετσι η διεπαφη του QGIS παραμενει χρησιμοποιησιμη κατα τον υπολογισμο. Μια γραμμη προοδου δειχνει την κατασταση επεξεργασιας.

## Καρτελα Μηχανικη Μαθηση

Η καρτελα **ML** προσφερει μεθοδους μηχανικης μαθησης για χωρικες προβλεψεις, ως εναλλακτικη η συμπληρωμα στο kriging.

### Διαθεσιμοι αλγοριθμοι

| Αλγοριθμος | Περιγραφη | Πλεονεκτηματα |
|------------|-----------|--------------|
| **Random Forest** | Συνολο δεντρων αποφασεων | Ανθεκτικο, διαχειριζεται μη γραμμικες σχεσεις |
| **Gradient Boosting** | Διαδοχικα δεντρα αποφασεων | Υψηλη ακριβεια, καταλληλο για συνθετα μοτιβα |
| **SVR** | Παλινδρομηση διανυσματων υποστηριξης | Καλο με λιγα δεδομενα, ευελικτοι πυρηνες |

### Ροη εργασιας ML

1. Ανοιξτε την καρτελα **ML**
2. Επιλεξτε τον επιθυμητο **αλγοριθμο**
3. Ρυθμιστε τις **μεταβλητες προβλεψης**:
   - Συντεταγμενες (X, Y)
   - Προσθετα πεδια απο το στρωμα (π.χ. υψομετρο, κλιση, αποσταση απο ποταμο)
4. Ρυθμιστε τις **παραμετρους** του αλγοριθμου (η χρησιμοποιηστε τις προεπιλογες)
5. Επιλεξτε τη μεθοδο **επικυρωσης**:
   - Διασταυρωμενη επικυρωση k-fold (προεπιλογη: 5 folds)
   - Hold-out (ποσοστο ελεγχου)
6. Κανε κλικ στο **Εκπαιδευση μοντελου**

<!-- IMAGE: Ρυθμιση μοντελου ML -->
> **Εικ. 8**: Ρυθμιση μοντελου Random Forest στην καρτελα ML

### Αποτελεσματα ML

| Αποτελεσμα | Περιγραφη |
|------------|-----------|
| **Χαρτης προβλεψης** | Στρωμα raster με τις προβλεπομενες τιμες |
| **Σημαντικοτητα μεταβλητων** | Γραφημα σχετικης σημαντικοτητας μεταβλητων προβλεψης |
| **Μετρικες επικυρωσης** | R-τετραγωνο, RMSE, MAE απο τη διασταυρωμενη επικυρωση |
| **Γραφημα υπολειμματων** | Διαγραμμα διασπορας παρατηρηθεισων εναντι προβλεφθεισων τιμων |

### Συγκριση Kriging εναντι ML

Για συγκριση αποτελεσματων:

1. Εκτελεστε τοσο kriging οσο και ML στα ιδια δεδομενα
2. Συγκρινετε τις μετρικες επικυρωσης στην καρτελα Αναφορα
3. Οπτικοποιηστε χαρτες διαφορων

## Καρτελα Δειγματοληψια

Η καρτελα **Δειγματοληψια** επιτρεπει τον σχεδιασμο βελτιστων στρατηγικων δειγματοληψιας για μελλοντικες αρχαιολογικες ερευνες.

### Στρατηγικες δειγματοληψιας

| Στρατηγικη | Περιγραφη | Ποτε χρησιμοποιειται |
|------------|-----------|---------------------|
| **Απλη τυχαια** | Σημεια κατανεμημενα τυχαια στην περιοχη | Οταν δεν υπαρχει προηγουμενη πληροφορια |
| **Στρωματοποιημενη τυχαια** | Τυχαια σημεια σε καθορισμενα στρωματα | Οταν η περιοχη εχει ζωνες με διαφορετικα χαρακτηριστικα |
| **Κανονικο πλεγμα** | Σημεια σε κανονικο πλεγμα | Για ομοιομορφη καλυψη της περιοχης |
| **Βελτιστοποιημενη** | Σημεια τοποθετημενα για ελαχιστοποιηση της διακυμανσης kriging | Οταν υπαρχει βαριογραμμα |

### Σχεδιασμος πλανου δειγματοληψιας

1. Ανοιξτε την καρτελα **Δειγματοληψια**
2. Επιλεξτε τη **στρατηγικη** δειγματοληψιας
3. Ορισαν τον επιθυμητο **αριθμο σημειων**
4. Καθοριστε την **περιοχη μελετης**:
   - Απο την εκταση του τρεχοντος στρωματος
   - Απο ενα πολυγωνικο στρωμα
   - Σχεδιαζοντας χειροκινητα στον χαρτη
5. Ορισαν πιθανους **περιορισμους**:
   - Ελαχιστη αποσταση μεταξυ σημειων
   - Περιοχες αποκλεισμου
6. Κανε κλικ στο **Δημιουργια δειγματοληψιας**

<!-- IMAGE: Δημιουργημενα σημεια δειγματοληψιας -->
> **Εικ. 9**: Βελτιστοποιημενα σημεια δειγματοληψιας υπερτεθειμενα στον χαρτη της περιοχης μελετης

### Αποτελεσματα δειγματοληψιας

- Ενα **διανυσματικο στρωμα σημειων** με τα σημεια δειγματοληψιας προστιθεται στο εργο QGIS
- Ενας **πινακας χαρακτηριστικων** με συντεταγμενες και αναγνωριστικα σημειων
- Μια **αναφορα** με τα στατιστικα της στρατηγικης (καλυψη, αποστασεις, κλπ.)

## Καρτελα Αναφορα

Η καρτελα **Αναφορα** επιτρεπει τη δημιουργια ολοκληρωμενων αναφορων γεωστατιστικης αναλυσης.

### Περιεχομενο αναφορας

Η αναφορα περιλαμβανει αυτοματα ολες τις αναλυσεις που πραγματοποιηθηκαν κατα τη διαρκεια της συνεδριας:

| Ενοτητα | Περιεχομενο |
|---------|------------|
| **Περιληψη** | Επισκοπηση του συνολου δεδομενων και των αναλυσεων |
| **Δεδομενα** | Περιγραφικα στατιστικα, κατανομη, χαρτης σημειων |
| **Βαριογραμμα** | Πειραματικο βαριογραμμα, μοντελο, παραμετροι |
| **Παρεμβολη** | Χαρτης kriging/ML, μετρικες επικυρωσης |
| **Δειγματοληψια** | Στρατηγικη, χαρτης σημειων, στατιστικα |
| **Συμπερασματα** | Συνθετικη ερμηνεια αποτελεσματων |

### Δημιουργια αναφορας

1. Ανοιξτε την καρτελα **Αναφορα**
2. Επιλεξτε τις **ενοτητες** για συμπεριληψη (ολες εξ ορισμου)
3. Ρυθμιστε τη **μορφη εξοδου**:
   - PDF (συνισταται για τεκμηριωση)
   - HTML (για διαδραστικη προβολη)
   - Markdown (για μεταγενεστερη επεξεργασια)
4. Εισαγετε τυχον **προσθετες σημειωσεις** η σχολια
5. Κανε κλικ στο **Δημιουργια αναφορας**

## Επιχειρησιακη ροη εργασιας

Ακολουθει μια τυπικη ροη εργασιας για πληρη γεωστατιστικη αναλυση στο GeoArchaeo:

### Βημα 1: Προετοιμασια δεδομενων

1. Φορτωστε το διανυσματικο στρωμα σημειων στο QGIS
2. Βεβαιωθειτε οτι το αριθμητικο πεδιο για αναλυση ειναι παρον και σωστο
3. Ελεγξτε το συστημα αναφορας συντεταγμενων

### Βημα 2: Εξερευνηση δεδομενων

1. Ανοιξτε το GeoArchaeo απο τη γραμμη εργαλειων
2. Στην καρτελα **Δεδομενα**, επιλεξτε το στρωμα και το πεδιο
3. Εξετασαν τα περιγραφικα στατιστικα
4. Ελεγξτε την κατανομη δεδομενων (αναζητηστε ακραιες η ανωμαλες τιμες)

### Βημα 3: Αναλυση βαριογραμματος

1. Στην καρτελα **Βαριογραμμα**, υπολογιστε το πειραματικο βαριογραμμα
2. Δοκιμαστε διαφορετικα μοντελα (σφαιρικο, εκθετικο, γκαουσιανο)
3. Επιλεξτε το μοντελο με την καλυτερη προσαρμογη
4. Σημειωστε τις παραμετρους (nugget, sill, range)

### Βημα 4: Παρεμβολη

1. Στην καρτελα **Kriging**, ρυθμιστε τις παραμετρους πλεγματος
2. Εκτελεστε συνηθες kriging
3. Εξετασαν τον χαρτη προβλεψης και τη διακυμανση
4. Προαιρετικα, συγκρινετε με μοντελο ML στην καρτελα ML

### Βημα 5: Δειγματοληψια (προαιρετικο)

1. Στην καρτελα **Δειγματοληψια**, σχεδιαστε μια στρατηγικη για μελλοντικες ερευνες
2. Χρησιμοποιηστε το βαριογραμμα για βελτιστοποιημενη δειγματοληψια

### Βημα 6: Αναφορα

1. Στην καρτελα **Αναφορα**, δημιουργηστε την τελικη αναφορα
2. Εξαγωγη σε PDF για τεκμηριωση

## Αντιμετωπιση προβληματων

### Συνηθη προβληματα

| Προβλημα | Αιτια | Λυση |
|----------|-------|------|
| Δεν υπαρχουν διαθεσιμα στρωματα | Δεν υπαρχουν στρωματα σημειων στο εργο | Προσθεστε ενα διανυσματικο στρωμα σημειων στο εργο QGIS |
| Δεν υπαρχουν αριθμητικα πεδια | Το στρωμα δεν εχει αριθμητικα πεδια | Ελεγξτε τον πινακα χαρακτηριστικων του στρωματος |
| Επιπεδο βαριογραμμα | Δεδομενα χωρις χωρικη συσχετιση | Ελεγξτε τα δεδομενα, αυξηστε τη μεγιστη αποσταση |
| Αποτυχια kriging | Μη προσαρμοσμενο μοντελο βαριογραμματος | Προσαρμοστε πρωτα μοντελο στην καρτελα Βαριογραμμα |
| Κακα αποτελεσματα ML | Ανεπαρκη δεδομενα η μη ενημερωτικες μεταβλητες | Προσθεστε μεταβλητες προβλεψης η αυξηστε τα δεδομενα |
| Πινακας μη ορατος | Widget εκλεισε κατα λαθος | Ξανανοιξτε απο το μενου Εργαλεια Αναλυσης |

### Συνηθη σφαλματα

- **"Ανεπαρκη δεδομενα"**: Απαιτουνται τουλαχιστον 30 σημεια για αξιοπιστη αναλυση
- **"Μοντελο βαριογραμματος μη ορισμενο"**: Προσαρμοστε μοντελο πριν εκτελεσετε kriging
- **"Μη συμβατο CRS"**: Ολα τα στρωματα πρεπει να χρησιμοποιουν το ιδιο συστημα αναφορας

### Αποδοση

- Η αναλυση εκτελειται σε **νημα παρασκηνιου**: η διεπαφη QGIS παραμενει χρησιμοποιησιμη
- Για πολυ μεγαλα συνολα δεδομενων (>10.000 σημεια), η επεξεργασια μπορει να παρει περισσοτερο χρονο
- Η προοδος μπορει να παρακολουθηθει μεσω της γραμμης στο κατω μερος του πινακα

## Τεχνικες σημειωσεις

### Εξαρτησεις

Το GeoArchaeo χρησιμοποιει τις ακολουθες βιβλιοθηκες Python:

| Βιβλιοθηκη | Χρηση |
|-----------|-------|
| **NumPy** | Αριθμητικοι και μητρωικοι υπολογισμοι |
| **SciPy** | Βελτιστοποιηση και προσαρμογη μοντελων |
| **scikit-learn** | Αλγοριθμοι μηχανικης μαθησης |
| **Matplotlib** | Δημιουργια γραφηματων |

### Συστηματα αναφορας συντεταγμενων

- Το GeoArchaeo λειτουργει στο συστημα αναφορας του τρεχοντος εργου QGIS
- Συνισταται ενα **προβεβλημενο CRS** (σε μετρα) για γεωστατιστικη αναλυση
- Τα γεωγραφικα συστηματα (σε μοιρες) μπορει να δωσουν ανακριβη αποτελεσματα

### Εξαγωγη αποτελεσματων

Τα αποτελεσματα μπορουν να εξαχθουν σε διαφορες μορφες:

- **Στρωματα raster** (GeoTIFF) για παρεμβεβλημενες επιφανειες
- **Διανυσματικα στρωματα** (GeoPackage, Shapefile) για σημεια δειγματοληψιας
- **Γραφηματα** (PNG, SVG) για βαριογραμματα και διαγνωστικα
- **Αναφορες** (PDF, HTML, Markdown) για τεκμηριωση

### Ενσωματωση με QGIS

- Τα στρωματα εξοδου προστιθενται αυτοματα στον πινακα **Στρωματα** του QGIS
- Η μορφοποιηση στρωματων raster μπορει να προσαρμοστει μεσω των ιδιοτητων στρωματος QGIS
- Τα αποτελεσματα ειναι συμβατα με ολα τα εργαλεια χωρικης αναλυσης του QGIS
