# Εκπαιδευτικό 20: Δημοσίευση στο Διαδίκτυο με Lizmap

## Εισαγωγή

Το PyArchInit υποστηρίζει **δημοσίευση στο διαδίκτυο** αρχαιολογικών δεδομένων μέσω **Lizmap**, μιας εφαρμογής που επιτρέπει τη μετατροπή έργων QGIS σε διαδραστικές διαδικτυακές εφαρμογές.

### Τι είναι το Lizmap;

Το Lizmap αποτελείται από:
- **Πρόσθετο QGIS**: Για ρύθμιση δημοσίευσης
- **Lizmap Web Client**: Διαδικτυακή εφαρμογή για προβολή χαρτών
- **QGIS Server**: Υποδομή για εξυπηρέτηση χαρτών

### Πλεονεκτήματα Διαδικτυακής Δημοσίευσης

| Πτυχή | Περιγραφή |
|-------|-----------|
| Προσβασιμότητα | Δεδομένα προσβάσιμα από πρόγραμμα περιήγησης |
| Κοινοποίηση | Εύκολη διανομή σε ενδιαφερόμενους |
| Διαδραστικότητα | Μεγέθυνση, μετακίνηση, ερωτήματα, αναδυόμενα |
| Ενημέρωση | Συγχρονισμός με βάση δεδομένων |
| Κινητό | Πρόσβαση από smartphone/tablet |

## Προαπαιτούμενα

### Εξυπηρετητής

1. Εγκατεστημένος **QGIS Server**
2. Ρυθμισμένος **Lizmap Web Client**
3. Διαδικτυακός εξυπηρετητής (Apache/Nginx)
4. PHP 7.4+
5. PostgreSQL/PostGIS (συνιστάται)

### Πελάτης

1. **QGIS Desktop** με πρόσθετο Lizmap
2. Ρυθμισμένο **PyArchInit**
3. Αποθηκευμένο έργο QGIS

## Εγκατάσταση Πρόσθετου Lizmap

### Από το QGIS

1. **Plugins** → **Manage Plugins**
2. Αναζήτηση «Lizmap»
3. Εγκατάσταση **Lizmap**
4. Επανεκκίνηση QGIS

## Προετοιμασία Έργου

### 1. Οργάνωση Στρωμάτων

Συνιστώμενη δομή:
```
QGIS Project
├── Ομάδα: Βάση
│   ├── Ορθοφωτογραφία
│   └── CTR/Κτηματολογικό
├── Ομάδα: Ανασκαφή
│   ├── pyunitastratigrafiche
│   ├── pyunitastratigrafiche_usm
│   └── pyarchinit_quote
├── Ομάδα: Τεκμηρίωση
│   ├── Γεωαναφερμένες φωτογραφίες
│   └── Αποτυπώσεις
└── Ομάδα: Ανάλυση
    └── Harris Matrix (εικόνα)
```

### 2. Μορφοποίηση Στρωμάτων

Για κάθε στρώμα:
1. Εφαρμόστε κατάλληλη μορφοποίηση
2. Ρυθμίστε ετικέτες
3. Ορίστε ορατότητα κλίμακας

### 3. Αναδυόμενα Παράθυρα και Ιδιότητες

Ρυθμίστε αναδυόμενα παράθυρα για κάθε στρώμα:
1. Δεξί κλικ στο στρώμα → **Properties**
2. Καρτέλα **Display**
3. Ρυθμίστε **HTML Map Tip**

Παράδειγμα αναδυόμενου ΣΕ:
```html
<h3>SU [% "us_s" %]</h3>
<p><b>Area:</b> [% "area_s" %]</p>
<p><b>Type:</b> [% "tipo_us_s" %]</p>
<p><b>Definition:</b> [% "definizione" %]</p>
```

### 4. Αποθήκευση Έργου

1. Αποθηκεύστε το έργο (.qgz) στον φάκελο Lizmap
2. Χρησιμοποιήστε σχετικές διαδρομές για τα δεδομένα
3. Επαληθεύστε ότι όλα τα στρώματα είναι προσβάσιμα

## Ρύθμιση Lizmap

### Άνοιγμα Πρόσθετου

1. **Web** → **Lizmap** → **Lizmap**

### Καρτέλα Γενικά

| Πεδίο | Περιγραφή | Τιμή |
|-------|-----------|------|
| Τίτλος χάρτη | Εμφανιζόμενο όνομα | «Ανασκαφή Οδού Ρώμης» |
| Περίληψη | Περιγραφή | «Αρχαιολογική τεκμηρίωση...» |
| Εικόνα | Μικρογραφία έργου | project_thumb.png |
| Αποθετήριο | Φάκελος εξυπηρετητή | /var/www/lizmap/projects |

### Καρτέλα Στρωμάτων

Για κάθε στρώμα ρυθμίστε:

| Επιλογή | Περιγραφή |
|---------|-----------|
| Enabled | Στρώμα ορατό στο Lizmap |
| Base layer | Στρώμα υποβάθρου (ορθοφωτογραφία κ.λπ.) |
| Popup | Ενεργοποίηση αναδυόμενου στο κλικ |
| Editing | Επιτρέψτε διαδικτυακή επεξεργασία |
| Filter | Φίλτρα χρηστών |

### Καρτέλα Υπόβαθρα

Ρυθμίστε υπόβαθρα:
- OpenStreetMap
- Google Maps (απαιτεί κλειδί API)
- Bing Maps
- Προσαρμοσμένη ορθοφωτογραφία

### Καρτέλα Εντοπισμού

Αναζήτηση τοποθεσίας:
- Ρυθμίστε στρώματα για αναζήτηση
- Πεδία για αναζήτηση
- Μορφή εμφάνισης

### Καρτέλα Επεξεργασίας (Προαιρετική)

Για να επιτρέψετε διαδικτυακή επεξεργασία:
1. Επιλέξτε επεξεργάσιμα στρώματα
2. Ρυθμίστε επεξεργάσιμα πεδία
3. Ορίστε δικαιώματα

### Καρτέλα Εργαλείων

Ενεργοποίηση/απενεργοποίηση:
- Εκτύπωση
- Μετρήσεις
- Επιλογή
- Μόνιμος σύνδεσμος
- κ.λπ.

### Αποθήκευση Ρύθμισης

Κάντε κλικ στο **Save** για δημιουργία του αρχείου `.qgs.cfg`

## Δημοσίευση

### Μεταφόρτωση στον Εξυπηρετητή

1. Αντιγράψτε τα αρχεία `.qgz` και `.qgz.cfg` στον εξυπηρετητή
2. Επαληθεύστε τα δικαιώματα αρχείων
3. Ρυθμίστε τον QGIS Server

### Δομή Εξυπηρετητή

```
/var/www/lizmap/
├── lizmap/           # Εφαρμογή Lizmap
├── projects/         # Έργα QGIS
│   ├── scavo_roma.qgz
│   └── scavo_roma.qgz.cfg
└── var/              # Δεδομένα εφαρμογής
```

### Ρύθμιση QGIS Server

Αρχείο `/etc/apache2/sites-available/lizmap.conf`:
```apache
<VirtualHost *:80>
    ServerName lizmap.example.com
    DocumentRoot /var/www/lizmap/lizmap/www

    <Directory /var/www/lizmap/lizmap/www>
        AllowOverride All
        Require all granted
    </Directory>

    # QGIS Server
    ScriptAlias /cgi-bin/ /usr/lib/cgi-bin/
    <Directory "/usr/lib/cgi-bin">
        AllowOverride All
        Require all granted
    </Directory>

    FcgidInitialEnv QGIS_SERVER_LOG_FILE /var/log/qgis/qgis_server.log
    FcgidInitialEnv QGIS_SERVER_LOG_LEVEL 0
</VirtualHost>
```

## Ενσωμάτωση PyArchInit

### Δεδομένα Πραγματικού Χρόνου

Με PostgreSQL:
- Τα δεδομένα είναι πάντα ενημερωμένα
- Αλλαγές στο PyArchInit ορατές αμέσως
- Χωρίς χειρονακτικό συγχρονισμό

### Συνιστώμενα Στρώματα

| Στρώμα PyArchInit | Δημοσίευση |
|-------------------|------------|
| pyunitastratigrafiche | Ναι, με αναδυόμενο |
| pyunitastratigrafiche_usm | Ναι, με αναδυόμενο |
| pyarchinit_quote | Ναι |
| pyarchinit_siti | Ναι, ως επισκόπηση |
| Harris Matrix | Ως στατική εικόνα |

### Προηγμένη Ρύθμιση Αναδυόμενων ΣΕ

Προηγμένο πρότυπο HTML:
```html
<div class="us-popup">
    <h3 style="color:#2c3e50;">SU [% "us_s" %]</h3>
    <table>
        <tr><td><b>Site:</b></td><td>[% "scavo_s" %]</td></tr>
        <tr><td><b>Area:</b></td><td>[% "area_s" %]</td></tr>
        <tr><td><b>Type:</b></td><td>[% "tipo_us_s" %]</td></tr>
        <tr><td><b>Definition:</b></td><td>[% "definizione" %]</td></tr>
        <tr><td><b>Period:</b></td><td>[% "periodo" %]</td></tr>
    </table>
    [% if "foto_url" %]
    <img src="[% "foto_url" %]" style="max-width:200px;"/>
    [% end %]
</div>
```

## Συντήρηση

### Ενημερώσεις Έργου

1. Επεξεργασία έργου στο QGIS Desktop
2. Αναδημιουργία ρύθμισης Lizmap
3. Μεταφόρτωση στον εξυπηρετητή

### Κρυφή Μνήμη

Διαχείριση κρυφής μνήμης πλακιδίων:
```bash
# Εκκαθάριση κρυφής μνήμης
lizmap-cache-clearer.php -project scavo_roma

# Αναδημιουργία πλακιδίων
lizmap-tiles-seeder.php -project scavo_roma -bbox xmin,ymin,xmax,ymax
```

### Καταγραφή και Εντοπισμός Σφαλμάτων

```bash
# Αρχείο καταγραφής QGIS Server
tail -f /var/log/qgis/qgis_server.log

# Αρχείο καταγραφής Lizmap
tail -f /var/www/lizmap/var/log/messages.log
```

## Αντιμετώπιση Προβλημάτων

### Ο Χάρτης δεν Εμφανίζεται

**Αιτίες**:
- Μη ρυθμισμένος QGIS Server
- Λανθασμένες διαδρομές αρχείων
- Ανεπαρκή δικαιώματα

**Λύσεις**:
- Ελέγξτε το αρχείο καταγραφής QGIS Server
- Επαληθεύστε τις διαδρομές στο έργο
- Ελέγξτε τα δικαιώματα αρχείων

### Τα Αναδυόμενα δεν Λειτουργούν

**Αιτίες**:
- Το στρώμα δεν είναι ρυθμισμένο για αναδυόμενο
- Εσφαλμένο HTML στο πρότυπο

**Λύσεις**:
- Ενεργοποιήστε αναδυόμενο στο Lizmap
- Επαληθεύστε τη σύνταξη HTML

### Αργή Απόδοση

**Αιτίες**:
- Πάρα πολλά δεδομένα
- Μη ρυθμισμένη κρυφή μνήμη
- Υποδιαστασιολογημένος εξυπηρετητής

**Λύσεις**:
- Μειώστε τα ορατά δεδομένα
- Ρυθμίστε κρυφή μνήμη πλακιδίων
- Αυξήστε τους πόρους εξυπηρετητή
