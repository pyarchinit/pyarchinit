# Εκπαιδευτικό 07: Δελτίο Ατόμου

## Εισαγωγή

Το **Δελτίο Ατόμου** είναι η ενότητα του PyArchInit αφιερωμένη στην ανθρωπολογική τεκμηρίωση ανθρώπινων λειψάνων που βρέθηκαν κατά την ανασκαφή. Αυτό το δελτίο καταγράφει πληροφορίες σχετικά με το φύλο, την ηλικία, τη θέση σώματος και την κατάσταση διατήρησης του σκελετού.

### Βασικές Έννοιες

**Άτομο στο PyArchInit:**
- Ένα άτομο είναι ένα σύνολο οστεολογικών λειψάνων αποδιδόμενων σε ένα μόνο πρόσωπο
- Συνδέεται με το Δελτίο Ταφής (ταφικό πλαίσιο)
- Συνδέεται με το Δελτίο Κατασκευής (φυσική κατασκευή)
- Μπορεί να συνδεθεί με Αρχαιοζωολογία για ειδικές αναλύσεις

**Ανθρωπολογικά Δεδομένα:**
- Εκτίμηση βιολογικού φύλου
- Εκτίμηση ηλικίας κατά τον θάνατο
- Θέση σώματος και προσανατολισμός
- Κατάσταση διατήρησης και πληρότητα

## Πρόσβαση στο Δελτίο

### Μέσω Μενού
1. Μενού **PyArchInit** στη γραμμή μενού του QGIS
2. Επιλέξτε **Individual Form**

## Πεδία Ταυτοποίησης

![Πεδία ταυτοποίησης](images/07_scheda_individui/06_campi_identificativi.png)

### Θέση
**Πεδίο**: `comboBox_sito` / **Βάση δεδομένων**: `sito`

### Τομέας
**Πεδίο**: `comboBox_area` / **Βάση δεδομένων**: `area`

### ΣΜ
**Πεδίο**: `comboBox_us` / **Βάση δεδομένων**: `us`
Στρωματογραφική Μονάδα αναφοράς.

### Αριθμός Ατόμου
**Πεδίο**: `lineEdit_individuo` / **Βάση δεδομένων**: `nr_individuo`
Προοδευτικός αριθμός ατόμου. Ο επόμενος διαθέσιμος αριθμός προτείνεται αυτόματα.

**Σημειώσεις:**
- Ο συνδυασμός Θέση + Τομέας + ΣΜ + Αρ. Ατόμου πρέπει να είναι μοναδικός
- Ο αριθμός αποδίδεται αυτόματα κατά τη δημιουργία

### Σύνδεση Κατασκευής

| Πεδίο | Βάση δεδομένων | Περιγραφή |
|-------|----------------|-----------|
| Structure code | `sigla_struttura` | Κωδικός κατασκευής (π.χ., TM) |
| Structure no. | `nr_struttura` | Αριθμός κατασκευής |

## Καρτέλα Περιγραφικών Δεδομένων

![Καρτέλα Περιγραφικών Δεδομένων](images/07_scheda_individui/08_tab_descrittivi.png)

### Εκτίμηση Φύλου
**Πεδίο**: `comboBox_sesso` / **Βάση δεδομένων**: `sesso`

| Τιμή | Περιγραφή |
|------|-----------|
| Αρσενικό | Σαφή ανδρικά χαρακτηριστικά |
| Θηλυκό | Σαφή γυναικεία χαρακτηριστικά |
| Πιθανώς αρσενικό | Κυρίως ανδρικά χαρακτηριστικά |
| Πιθανώς θηλυκό | Κυρίως γυναικεία χαρακτηριστικά |
| Αδιευκρίνιστο | Μη προσδιοριζόμενο |

**Κριτήρια προσδιορισμού:**
- Μορφολογία λεκάνης
- Μορφολογία κρανίου
- Γενική σκελετική ρωμαλεότητα
- Οστικές διαστάσεις

### Εκτίμηση Ηλικίας κατά τον Θάνατο

| Πεδίο | Βάση δεδομένων | Περιγραφή |
|-------|----------------|-----------|
| Ελάχιστη ηλικία | `eta_min` | Κατώτερο όριο εκτίμησης |
| Μέγιστη ηλικία | `eta_max` | Ανώτερο όριο εκτίμησης |

**Μέθοδοι εκτίμησης:**
- Ηβική σύμφυση
- Ωτοειδής επιφάνεια
- Κρανιακές ραφές
- Οδοντική ανάπτυξη (για ανήλικα)
- Οστικές επιφύσεις (για ανήλικα)

### Ηλικιακές Κλάσεις
**Πεδίο**: `comboBox_classi_eta` / **Βάση δεδομένων**: `classi_eta`

| Κλάση | Κατά προσέγγιση ηλικία |
|-------|------------------------|
| Infans I | 0-6 έτη |
| Infans II | 7-12 έτη |
| Juvenis | 13-20 έτη |
| Adultus | 21-40 έτη |
| Maturus | 41-60 έτη |
| Senilis | >60 έτη |

### Παρατηρήσεις
**Πεδίο**: `textEdit_osservazioni` / **Βάση δεδομένων**: `osservazioni`

**Προτεινόμενο περιεχόμενο:**
- Παρατηρούμενες παθολογίες
- Τραύματα
- Δείκτες επαγγελματικής δραστηριότητας
- Σκελετικές ανωμαλίες
- Σημειώσεις για τον προσδιορισμό φύλου και ηλικίας

## Καρτέλα Προσανατολισμού και Θέσης

![Καρτέλα Προσανατολισμού](images/07_scheda_individui/09_tab_orientamento.png)

### Κατάσταση Διατήρησης

| Πεδίο | Βάση δεδομένων | Τιμές |
|-------|----------------|-------|
| Πλήρης | `completo_si_no` | Ναι / Όχι |
| Διαταραγμένο | `disturbato_si_no` | Ναι / Όχι |
| Σε σύναρθρωση | `in_connessione_si_no` | Ναι / Όχι |

**Ορισμοί:**
- **Πλήρης**: όλες οι ανατομικές περιοχές αντιπροσωπεύονται
- **Διαταραγμένο**: ενδείξεις μετα-αποθετικής διαταραχής
- **Σε σύναρθρωση**: οι αρθρώσεις διατηρούνται

### Μήκος Σκελετού
**Πεδίο**: `lineEdit_lunghezza` / **Βάση δεδομένων**: `lunghezza_scheletro`
Μετρημένο μήκος σκελετού in situ (σε εκ. ή μ.).

### Θέση Σκελετού
**Τιμές:**
- Ύπτια (σε πλάτη)
- Πρηνής (με πρόσωπο κάτω)
- Δεξιά πλάγια
- Αριστερή πλάγια
- Συνεσταλμένη
- Ακανόνιστη

### Θέση Κρανίου
**Τιμές:** Στραμμένο δεξιά, Στραμμένο αριστερά, Στραμμένο πάνω, Στραμμένο κάτω, Μη προσδιοριζόμενο

### Θέση Άνω Άκρων
**Τιμές:** Εκτεταμένα κατά μήκος, Στη λεκάνη, Στο στήθος, Σταυρωμένα στο στήθος, Μικτή, Μη προσδιοριζόμενο

### Θέση Κάτω Άκρων
**Τιμές:** Εκτεταμένα, Λυγισμένα, Σταυρωμένα, Ανοιχτά, Μη προσδιοριζόμενο

### Προσανατολισμός Άξονα
**Τιμές:** Β-Ν, Ν-Β, Α-Δ, Δ-Α, ΒΑ-ΝΔ, ΒΔ-ΝΑ κ.λπ.

### Αζιμούθιο Προσανατολισμού
Αριθμητική τιμή αζιμουθίου σε μοίρες (0-360).

## Ροή Εργασίας

### Δημιουργία Νέου Ατόμου

1. **Άνοιγμα δελτίου** - Μέσω μενού ή γραμμής εργαλείων

2. **Νέα εγγραφή** - Κάντε κλικ στο "New Record"

3. **Δεδομένα ταυτοποίησης**
   ```
   Site: Νεκρόπολη Isola Sacra
   Area: 1
   SU: 150
   Individual No.: 1
   Structure code: TM
   Structure no.: 45
   ```

4. **Ανθρωπολογικά δεδομένα** (Καρτέλα 1)
   ```
   Sex: Αρσενικό
   Min age: 35
   Max age: 45
   Age class: Adultus
   ```

5. **Προσανατολισμός και θέση** (Καρτέλα 2)
   ```
   Complete: Ναι
   Disturbed: Όχι
   In articulation: Ναι
   Position: Ύπτια
   Orientation: Α-Δ
   ```

6. **Οστεολογικά λείψανα** (Καρτέλα 3)
   - Τεκμηρίωση παρόντων ανατομικών περιοχών

7. **Αποθήκευση** - Κάντε κλικ στο "Save"

## Βέλτιστες Πρακτικές

### Προσδιορισμός Φύλου
- Χρησιμοποιήστε πολλαπλούς μορφολογικούς δείκτες
- Αναφέρετε επίπεδο αξιοπιστίας
- Προσδιορίστε κριτήρια στις παρατηρήσεις

### Εκτίμηση Ηλικίας
- Πάντα παρέχετε εύρος (ελάχ.-μέγ.)
- Αναφέρετε χρησιμοποιούμενες μεθόδους
- Για ανήλικα, προσδιορίστε οδοντική και επιφυσιακή ανάπτυξη

### Θέση και Προσανατολισμός
- Τεκμηριώστε με φωτογραφίες πριν την αφαίρεση
- Χρησιμοποιήστε αναφορές σε σημεία του ορίζοντα
- Μετρήστε αζιμούθιο με πυξίδα
