# Οδηγός 26: Εργαλεία Κεραμικής (Pottery Tools)

## Εισαγωγή

Τα **Pottery Tools** αποτελούν μια προηγμένη ενότητα για την επεξεργασία εικόνων κεραμικής. Προσφέρουν εργαλεία για εξαγωγή εικόνων από αρχεία PDF, δημιουργία πινάκων κεραμικής, επεξεργασία σχεδίων με τεχνητή νοημοσύνη (PotteryInk), καθώς και άλλες εξειδικευμένες λειτουργίες για την τεκμηρίωση κεραμικής.

### Κύρια Χαρακτηριστικά

- Εξαγωγή εικόνων από αρχεία PDF
- Δημιουργία διάταξης πινάκων κεραμικής
- Επεξεργασία εικόνων με τεχνητή νοημοσύνη
- Μετατροπή μορφής σχεδίων
- Ολοκλήρωση με τη Φόρμα Κεραμικής

## Πρόσβαση

### Από το Μενού
**PyArchInit** → **Pottery Tools**

## Καρτέλα Εξαγωγής PDF

### Λειτουργία

Αυτόματη εξαγωγή εικόνων από έγγραφα PDF που περιέχουν πίνακες κεραμικής.

### Χρήση

1. Επιλέξτε αρχείο πηγής PDF
2. Καθορίστε φάκελο προορισμού
3. Κάντε κλικ στο **"Extract"**
4. Οι εικόνες αποθηκεύονται ως ξεχωριστά αρχεία

### Επιλογές

| Επιλογή | Περιγραφή |
|---------|-----------|
| DPI | Ανάλυση εξαγωγής (150-600) |
| Format | PNG, JPG, TIFF |
| Pages | Όλες ή συγκεκριμένο εύρος |

## Καρτέλα Δημιουργού Διάταξης

### Λειτουργία

Αυτόματη δημιουργία πινάκων κεραμικής με τυποποιημένη διάταξη.

### Τύποι Διάταξης

| Διάταξη | Περιγραφή |
|---------|-----------|
| Grid | Εικόνες σε κανονικό πλέγμα |
| Sequence | Εικόνες σε αριθμημένη ακολουθία |
| Comparison | Διάταξη για σύγκριση |
| Catalog | Μορφή καταλόγου με λεζάντες |

### Χρήση

1. Επιλέξτε εικόνες προς συμπερίληψη
2. Επιλέξτε τύπο διάταξης
3. Ρυθμίστε παραμέτρους (διαστάσεις, περιθώρια)
4. Δημιουργήστε τον πίνακα

### Παράμετροι Διάταξης

| Παράμετρος | Περιγραφή |
|-----------|-----------|
| Page size | A4, A3, Προσαρμοσμένο |
| Orientation | Κατακόρυφα, Οριζόντια |
| Margins | Απόσταση περιθωρίων |
| Spacing | Απόσταση μεταξύ εικόνων |
| Captions | Κείμενο κάτω από τις εικόνες |

## Καρτέλα Επεξεργασίας Εικόνας

### Λειτουργία

Μαζική επεξεργασία εικόνων κεραμικής.

### Διαθέσιμες Λειτουργίες

| Λειτουργία | Περιγραφή |
|-----------|-----------|
| Resize | Κλιμάκωση εικόνων |
| Crop | Αυτόματη/χειροκίνητη περικοπή |
| Rotate | Περιστροφή κατά μοίρες |
| Convert | Αλλαγή μορφής |
| Optimize | Συμπίεση ποιότητας |

### Μαζική Επεξεργασία

1. Επιλέξτε φάκελο πηγής
2. Επιλέξτε τις λειτουργίες προς εφαρμογή
3. Καθορίστε τον προορισμό
4. Εκτελέστε την επεξεργασία

## Καρτέλα PotteryInk AI

### Λειτουργία

Χρησιμοποιεί τεχνητή νοημοσύνη για:
- Μετατροπή φωτογραφίας → τεχνικό σχέδιο
- Αναγνώριση κεραμικής μορφής
- Προτάσεις ταξινόμησης
- Αυτόματη μέτρηση

### Απαιτήσεις

- Ρυθμισμένο εικονικό περιβάλλον Python
- Λήψη μοντέλων τεχνητής νοημοσύνης (YOLO κ.λπ.)
- Συνιστάται GPU (αλλά δεν απαιτείται)

### Χρήση

1. Φορτώστε εικόνα κεραμικής
2. Επιλέξτε τύπο επεξεργασίας
3. Περιμένετε την επεξεργασία ΤΝ
4. Επαληθεύστε και αποθηκεύστε το αποτέλεσμα

### Τύποι Επεξεργασίας ΤΝ

| Τύπος | Περιγραφή |
|-------|-----------|
| Ink Conversion | Μετατροπή φωτογραφίας σε τεχνικό σχέδιο |
| Shape Detection | Αναγνώριση σχήματος αγγείου |
| Profile Extraction | Εξαγωγή κεραμικού τομής |
| Decoration Analysis | Ανάλυση διακόσμησης |

## Εικονικό Περιβάλλον

### Αυτόματη Εγκατάσταση

Κατά την πρώτη εκκίνηση, τα Pottery Tools:
1. Δημιουργούν εικονικό περιβάλλον στο `~/pyarchinit/bin/pottery_venv/`
2. Εγκαθιστούν τις απαιτούμενες εξαρτήσεις
3. Κατεβάζουν μοντέλα ΤΝ (εάν απαιτείται)

### Εξαρτήσεις

- PyTorch
- Ultralytics (YOLO)
- OpenCV
- Pillow

### Επαλήθευση Εγκατάστασης

Η καταγραφή εμφανίζει την κατάσταση:
```
✓ Virtual environment created
✓ Dependencies installed
✓ Models downloaded
✓ Pottery Tools initialized successfully!
```

## Ολοκλήρωση με τη Βάση Δεδομένων

### Σύνδεση με τη Φόρμα Κεραμικής

Οι επεξεργασμένες εικόνες μπορούν να:
- Συνδεθούν αυτόματα με εγγραφές Κεραμικής
- Αποθηκευτούν με κατάλληλα μεταδεδομένα
- Οργανωθούν κατά θέση/κατάλογο

## Βέλτιστες Πρακτικές

### 1. Ποιότητα Εικόνας Εισόδου

- Ελάχιστη ανάλυση: 300 DPI
- Ομοιόμορφος φωτισμός
- Ουδέτερο φόντο (λευκό/γκρι)
- Ορατή μετρική κλίμακα

### 2. Επεξεργασία ΤΝ

- Πάντα επαληθεύετε τα αποτελέσματα ΤΝ
- Διορθώστε χειροκίνητα εάν χρειάζεται
- Αποθηκεύστε πρωτότυπα και επεξεργασμένα

### 3. Οργάνωση Εξόδου

- Χρησιμοποιήστε συνεπή ονοματολογία
- Οργανώστε κατά θέση/ανασκαφική περίοδο
- Διατηρήστε ιχνηλασιμότητα

## Αντιμετώπιση Προβλημάτων

### Το Εικονικό Περιβάλλον Δεν Δημιουργήθηκε

**Αιτίες**:
- Δεν βρέθηκε η Python
- Ανεπαρκή δικαιώματα

**Λύσεις**:
- Επαληθεύστε την εγκατάσταση Python
- Ελέγξτε τα δικαιώματα φακέλου

### Αργή Επεξεργασία ΤΝ

**Αιτίες**:
- Δεν υπάρχει διαθέσιμη GPU
- Πολύ μεγάλες εικόνες

**Λύσεις**:
- Μειώστε το μέγεθος εικόνας
- Χρησιμοποιήστε CPU (πιο αργό αλλά λειτουργεί)

### Αποτυχία Εξαγωγής PDF

**Αιτίες**:
- Προστατευμένο PDF
- Μη υποστηριζόμενη μορφή

**Λύσεις**:
- Επαληθεύστε την προστασία του PDF
- Δοκιμάστε με άλλο λογισμικό PDF
