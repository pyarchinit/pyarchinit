## Εισαγωγή

Το **Δελτίο Απογραφής Υλικών** είναι το κύριο εργαλείο για τη διαχείριση αρχαιολογικών ευρημάτων στο PyArchInit. Σας επιτρέπει να καταλογογραφήσετε, να περιγράψετε και να ποσοτικοποιήσετε όλα τα υλικά που ανακαλύφθηκαν κατά τη διάρκεια της ανασκαφής, από κεραμική μέχρι μέταλλα, από γυαλί μέχρι ζωικά οστά.

### Σκοπός του Δελτίου

- Απογραφή όλων των ανακτημένων ευρημάτων
- Συσχέτιση ευρημάτων με ΣΜ προέλευσης
- Διαχείριση τυπολογικής κατάταξης
- Τεκμηρίωση φυσικών και τεχνολογικών χαρακτηριστικών
- Υπολογισμός ποσοτικοποιήσεων (ελάχιστα αγγεία, EVE, βάρος)
- Σύνδεση φωτογραφιών και σχεδίων με ευρήματα
- Δημιουργία αναφορών και στατιστικών

### Διαχειρίσιμοι Τύποι Υλικών

- **Κεραμική**: Αγγειοπλαστική, πήλινα ειδώλια, οικοδομικά υλικά
- **Μέταλλα**: Χαλκός, σίδηρος, μόλυβδος, χρυσός, ασήμι
- **Γυαλί**: Αγγεία, υαλοπίνακες
- **Οστό/Ελεφαντόδοντο**: Αντικείμενα από σκληρό ζωικό υλικό
- **Λίθος**: Λιθοτεχνία, γλυπτική
- **Νομίσματα**: Νομισματική
- **Οργανικά**: Ξύλο, υφάσματα, δέρμα

## Πρόσβαση στο Δελτίο

### Από Μενού
1. Μενού **PyArchInit** → **Archaeological record management** → **Artefact** → **Artefact form**

### Από Γραμμή Εργαλείων
1. Κάντε κλικ στο εικονίδιο **Finds** (εικονίδιο αμφορέα/αγγείου)

### Συντόμευση Πληκτρολογίου
- **Ctrl+G**: Ενεργοπ./απενεργοπ. απεικόνισης GIS τρέχοντος ευρήματος

## Κύρια Πεδία

### Πεδία Ταυτοποίησης

#### Θέση
- **Τύπος**: ComboBox (μόνο ανάγνωση μετά αποθήκευση)
- **Υποχρεωτικό**: Ναι

#### Αριθμός Απογραφής
- **Τύπος**: Αριθμητικό LineEdit
- **Υποχρεωτικό**: Ναι
- **Περιορισμός**: Μοναδικός ανά θέση

#### ΣΜ (Στρωματογραφική Μονάδα)
- **Τύπος**: LineEdit
- **Υποχρεωτικό**: Όχι (αλλά συνιστάται ιδιαίτερα)
- **Σύνδεση**: Συνδέει εύρημα με αντίστοιχο δελτίο ΣΜ

### Πεδία Κατάταξης

#### Τύπος Ευρήματος
- **Τυπικές τιμές**: Κεραμική, Μέταλλο, Γυαλί, Οστό, Λίθος, Νόμισμα κ.λπ.

#### Κλάση Υλικού (Κριτήριο Καταγραφής)
- **Παραδείγματα για κεραμική**:
  - Κοινά σκεύη
  - Ιταλική sigillata
  - Αφρικανική sigillata
  - Μελανόγλωσσα σκεύη
  - Λεπτότοιχα σκεύη
  - Αμφορείς
  - Λυχνάρια

#### Ορισμός
- **Παραδείγματα**: Πιάτο, Κύπελλο, Χύτρα, Οινοχόη, Κάλυμμα κ.λπ.

### Πεδία Κατάστασης και Διατήρησης

| Πεδίο | Τιμές |
|-------|-------|
| **Πλυμένο** | Ναι, Όχι |
| **Καταλογογραφημένο** | Ναι, Όχι |
| **Διαγνωστικό** | Ναι, Όχι |
| **Κατάσταση Διατήρησης** | Ακέραιο, Αποσπασματικό, Ελλιπές, Αποκατεστημένο |

## Καρτέλες Δελτίου

### Καρτέλα 1: Περιγραφή

#### Πεδίο Περιγραφής
**Προτεινόμενο περιεχόμενο:**
- Γενική μορφή
- Σωζόμενα τμήματα
- Τεχνικά χαρακτηριστικά
- Διακοσμήσεις
- Τυπολογικές συγκρίσεις

#### Χρονολόγηση Ευρήματος
**Μορφή**: Κείμενο (π.χ., "1ος αιώνας π.Χ.", "2ος-3ος αιώνας μ.Χ.")

### Καρτέλα 3: Ποσοτικά Δεδομένα

Θεμελιώδης καρτέλα για ποσοτική ανάλυση, ιδιαίτερα για κεραμική.

#### Πίνακας Στοιχείων Ευρήματος

| Στήλη | Περιγραφή | Παράδειγμα |
|-------|-----------|------------|
| Στοιχείο | Ανατομικό μέρος αγγείου | Χείλος, Τοίχωμα, Βάση, Λαβή |
| Τύπος ποσότητας | Κατάσταση θραύσματος | Θραύσμα, Ακέραιο |
| Ποσότητα | Αριθμός τεμαχίων | 5 |

#### Πεδία Κεραμικής Ποσοτικοποίησης

| Πεδίο | Περιγραφή |
|-------|-----------|
| **Ελάχιστα αγγεία** | Ελάχιστος Αριθμός Ατόμων (MNI) |
| **Μέγιστα αγγεία** | Μέγιστος Αριθμός Ατόμων |
| **Σύνολο θραυσμάτων** | Αυτόματος υπολογισμός από πίνακα στοιχείων |
| **Βάρος** | Βάρος σε γραμμάρια |
| **Διάμετρος χείλους** | Διάμετρος χείλους σε εκ. |
| **EVE χείλους** | Estimated Vessel Equivalent (ποσοστό σωζόμενης περιφέρειας χείλους) |

### Καρτέλα 4: Τεχνολογίες

Καταγραφή τεχνικών παραγωγής και διακόσμησης.

#### Ειδικά Πεδία Κεραμικής

| Πεδίο | Περιγραφή |
|-------|-----------|
| **Ceramic body** | Τύπος πηλού (Λεπτός, Ημίλεπτος, Αδρός) |
| **Coating** | Τύπος επικάλυψης (Επίχρισμα, Αγγόμπιο, Υάλωμα) |

### Καρτέλα 5: Βιβλιογραφικές Αναφορές

| Στήλη | Περιγραφή |
|-------|-----------|
| Συγγραφέας | Επώνυμο(α) συγγραφέα |
| Έτος | Έτος δημοσίευσης |
| Τίτλος | Συντομευμένος τίτλος |
| Σελίδα | Αναφορά σελίδας |
| Εικόνα | Αναφορά εικόνας/πίνακα |

### Καρτέλα 6: Ποσοτικοποιήσεις

#### Τύποι Ποσοτικοποίησης

| Τύπος | Περιγραφή |
|-------|-----------|
| **Ελάχιστα αγγεία** | Γράφημα MNI |
| **Μέγιστα αγγεία** | Γράφημα μέγιστου αριθμού |
| **Σύνολο θραυσμάτων** | Γράφημα αρίθμησης θραυσμάτων |
| **Βάρος** | Γράφημα βάρους |
| **EVE χείλους** | Γράφημα EVE |

## Ενσωμάτωση Τεχνητής Νοημοσύνης

### SketchGPT

Λειτουργία ΤΝ για αυτόματη δημιουργία περιγραφών ευρημάτων από εικόνες.

#### Χρήση
1. Συσχετίστε μια εικόνα με το εύρημα
2. Κάντε κλικ στο κουμπί **SketchGPT**
3. Επιλέξτε εικόνα προς ανάλυση
4. Επιλέξτε μοντέλο GPT (gpt-4-vision, gpt-4o)
5. Το σύστημα δημιουργεί αρχαιολογική περιγραφή

> **Σημείωση**: Απαιτεί ρυθμισμένο API Key του OpenAI.

## Ροή Εργασίας

### Δημιουργία Νέου Ευρήματος

#### Βήμα 1: Άνοιγμα Δελτίου
#### Βήμα 2: Νέα Εγγραφή - Κάντε κλικ στο **New record**
#### Βήμα 3: Δεδομένα Ταυτοποίησης
- Επαληθεύστε/επιλέξτε **Θέση**
- Εισαγάγετε **Αριθμό απογραφής**
- Εισαγάγετε **Τομέα** και **ΣΜ** προέλευσης

#### Βήμα 4: Κατάταξη
- Επιλέξτε **Τύπο ευρήματος**
- Επιλέξτε **Κλάση υλικού**
- Επιλέξτε/εισαγάγετε **Ορισμό**

#### Βήμα 5: Περιγραφή
- Συμπληρώστε **Περιγραφή**
- Επιλέξτε **Χρονολόγηση**

#### Βήμα 6: Ποσοτικά Δεδομένα (αν κεραμική)
- Εισαγάγετε στοιχεία στον πίνακα
- Συμπληρώστε ελάχιστα/μέγιστα αγγεία
- Εισαγάγετε βάρος και μετρήσεις

#### Βήμα 7: Αποθήκευση - Κάντε κλικ στο **Save**

## Βέλτιστες Πρακτικές

### Αρίθμηση Απογραφής
- Χρησιμοποιήστε προοδευτική αρίθμηση χωρίς κενά
- Ένας αριθμός = ένα εύρημα (ή ομοιογενής ομάδα)

### Κεραμική Ποσοτικοποίηση
1. **Ελάχιστα αγγεία (MNI)**: Μετρήστε μόνο διαγνωστικά στοιχεία
2. **EVE**: Μετρήστε ποσοστό σωζόμενης περιφέρειας
3. **Βάρος**: Ζυγίστε όλα τα θραύσματα της ομάδας

### Φωτογραφική Τεκμηρίωση
- Φωτογραφίστε όλα τα διαγνωστικά ευρήματα
- Χρησιμοποιήστε μετρική κλίμακα στις φωτογραφίες

## Αντιμετώπιση Προβλημάτων

### Διπλός αριθμός απογραφής
- **Λύση**: Επαληθεύστε τον επόμενο διαθέσιμο αριθμό με "View all"

### Οι εικόνες δεν εμφανίζονται
- **Λύση**: Επαληθεύστε ρύθμιση διαδρομής στις Ρυθμίσεις

### Κενές ποσοτικοποιήσεις
- **Λύση**: Συμπληρώστε ελάχιστα/μέγιστα αγγεία ή σύνολο θραυσμάτων
