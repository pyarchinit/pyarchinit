# Εκπαιδευτικό 10: Δελτίο Τεκμηρίωσης

## Εισαγωγή

Το **Δελτίο Τεκμηρίωσης** είναι η ενότητα του PyArchInit για τη διαχείριση της γραφικής τεκμηρίωσης ανασκαφής: κατόψεις, τομές, όψεις, αποτυπώσεις και οποιοδήποτε άλλο γραφικό αποτέλεσμα παράγεται κατά τη διάρκεια αρχαιολογικών δραστηριοτήτων.

### Τύποι Τεκμηρίωσης

- **Κατόψεις**: κατόψεις στρωμάτων, φάσεων, γενικές κατόψεις
- **Τομές**: στρωματογραφικές τομές
- **Όψεις**: όψεις τοιχοποιίας, μέτωπα ανασκαφής
- **Αποτυπώσεις**: τοπογραφικές, φωτογραμμετρικές αποτυπώσεις
- **Ορθοφωτογραφίες**: αποτελέσματα drone/φωτογραμμετρίας
- **Σχέδια ευρημάτων**: κεραμικής, μετάλλων κ.λπ.

---

## Πρόσβαση στο Δελτίο

### Μέσω Μενού
1. Μενού **PyArchInit** στη γραμμή μενού του QGIS
2. Επιλέξτε **Documentation Form**

### Μέσω Γραμμής Εργαλείων
1. Εντοπίστε τη γραμμή εργαλείων PyArchInit
2. Κάντε κλικ στο εικονίδιο **Documentation**

## Πεδία Δελτίου

### Θέση

**Πεδίο**: `comboBox_sito_doc`
**Βάση δεδομένων**: `sito`

Αρχαιολογική θέση αναφοράς.

### Όνομα Τεκμηρίωσης

**Πεδίο**: `lineEdit_nome_doc`
**Βάση δεδομένων**: `nome_doc`

Ταυτοποιητικό όνομα εγγράφου.

**Συμβάσεις ονοματοδοσίας:**
- `P` = Κάτοψη (π.χ., P001)
- `S` = Τομή (π.χ., S001)
- `E` = Όψη (π.χ., E001)
- `R` = Αποτύπωση (π.χ., R001)

### Ημερομηνία

**Πεδίο**: `dateEdit_data`
**Βάση δεδομένων**: `data`

Ημερομηνία εκτέλεσης σχεδίου/αποτύπωσης.

### Τύπος Τεκμηρίωσης

**Πεδίο**: `comboBox_tipo_doc`
**Βάση δεδομένων**: `tipo_documentazione`

**Τυπικές τιμές:**
| Τύπος | Περιγραφή |
|-------|-----------|
| Κάτοψη στρώματος | Μεμονωμένη ΣΜ |
| Κάτοψη φάσης | Πολλαπλές σύγχρονες ΣΜ |
| Γενική κάτοψη | Συνολική άποψη |
| Στρωματογραφική τομή | Κατακόρυφο προφίλ |
| Όψη | Ανωδομή τοιχοποιίας |
| Τοπογραφική αποτύπωση | Γενική πλανιμετρία |
| Ορθοφωτογραφία | Από φωτογραμμετρία |
| Σχέδιο ευρήματος | Κεραμική, μέταλλα κ.λπ. |

### Πηγή

**Πεδίο**: `comboBox_sorgente`
**Βάση δεδομένων**: `sorgente`

Πηγή/μέθοδος παραγωγής.

**Τιμές:**
- Άμεση αποτύπωση
- Φωτογραμμετρία
- Σαρωτής laser
- GPS/Γεωδαιτικός σταθμός
- Ψηφιοποίηση CAD
- Ορθοφωτογραφία drone

### Κλίμακα

**Πεδίο**: `comboBox_scala`
**Βάση δεδομένων**: `scala`

Κλίμακα αναπαράστασης.

**Συνήθεις κλίμακες:**
| Κλίμακα | Τυπική χρήση |
|---------|-------------|
| 1:1 | Σχέδια ευρημάτων |
| 1:5 | Λεπτομέρειες |
| 1:10 | Τομές, λεπτομέρειες |
| 1:20 | Κατόψεις στρωμάτων |
| 1:50 | Γενικές κατόψεις |
| 1:100 | Πλανιμετρίες |
| 1:200+ | Τοπογραφικοί χάρτες |

### Σχεδιαστής

**Πεδίο**: `comboBox_disegnatore`
**Βάση δεδομένων**: `disegnatore`

Συντάκτης σχεδίου/αποτύπωσης.

### Σημειώσεις

**Πεδίο**: `textEdit_note`
**Βάση δεδομένων**: `note`

Πρόσθετες σημειώσεις σχετικά με το έγγραφο.

## Ροή Εργασίας

### Καταχώρηση Νέας Τεκμηρίωσης

1. **Άνοιγμα δελτίου** - Μέσω μενού ή γραμμής εργαλείων

2. **Νέα εγγραφή** - Κάντε κλικ στο "New Record"

3. **Δεδομένα ταυτοποίησης**
   ```
   Site: Ρωμαϊκή Έπαυλη Settefinestre
   Name: P025
   Date: 15/06/2024
   ```

4. **Κατάταξη**
   ```
   Type: Κάτοψη στρώματος
   Source: Άμεση αποτύπωση
   Scale: 1:20
   ```

5. **Συντάκτης και σημειώσεις**
   ```
   Drawer: Μ. Ρόσι
   Notes: Κάτοψη ΣΜ 150. Επισημαίνει
   τα όρια επιφάνειας δαπέδου.
   ```

6. **Αποθήκευση** - Κάντε κλικ στο "Save"

### Αναζήτηση Τεκμηρίωσης

1. Κάντε κλικ στο "New Search"
2. Συμπληρώστε κριτήρια:
   - Θέση
   - Τύπος τεκμηρίωσης
   - Κλίμακα
   - Σχεδιαστής
3. Κάντε κλικ στο "Search"
4. Πλοηγηθείτε στα αποτελέσματα

## Βέλτιστες Πρακτικές

### Ονοματοδοσία

- Χρησιμοποιήστε συνεπείς κωδικούς σε όλο το έργο
- Προοδευτική αρίθμηση ανά τύπο
- Τεκμηριώστε τις υιοθετημένες συμβάσεις

### Οργάνωση

- Πάντα συνδέστε με θέση αναφοράς
- Αναφέρετε πραγματική κλίμακα
- Καταγράψτε ημερομηνία και συντάκτη

### Αρχειοθέτηση

- Συνδέστε ψηφιακά αρχεία μέσω διαχείρισης πολυμέσων
- Διατηρήστε αντίγραφα ασφαλείας
- Χρησιμοποιήστε τυποποιημένες μορφές (PDF, TIFF)

---

## Αντιμετώπιση Προβλημάτων

### Πρόβλημα: Τύπος τεκμηρίωσης μη διαθέσιμος

**Αιτία**: Θησαυρός μη ρυθμισμένος.

**Λύση**:
1. Ανοίξτε το Δελτίο Θησαυρού
2. Προσθέστε τους ελλείποντες τύπους για `documentazione_table`

### Πρόβλημα: Το αρχείο δεν εμφανίζεται

**Αιτία**: Λανθασμένη διαδρομή ή ελλείπον αρχείο.

**Λύση**:
1. Επαληθεύστε ότι το αρχείο υπάρχει
2. Ελέγξτε τη διαδρομή στη ρύθμιση πολυμέσων
