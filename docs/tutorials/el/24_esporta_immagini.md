# Οδηγός 24: Εξαγωγή Εικόνων

## Εισαγωγή

Η λειτουργία **Εξαγωγή Εικόνων** επιτρέπει τη μαζική εξαγωγή εικόνων που σχετίζονται με αρχαιολογικές εγγραφές, οργανώνοντάς τες αυτόματα σε φακέλους κατά περίοδο, φάση και τύπο οντότητας.

## Πρόσβαση

### Από το Μενού
**PyArchInit** → **Export Images**

## Διεπαφή

### Πίνακας Εξαγωγής

```
+--------------------------------------------------+
|           Export Images                           |
+--------------------------------------------------+
| Site: [ComboBox επιλογή θέσης]                   |
| Year: [ComboBox έτος ανασκαφής]                  |
+--------------------------------------------------+
| Τύπος Εξαγωγής:                                   |
|   [o] Όλες οι εικόνες                            |
|   [ ] Μόνο ΣΜ                                    |
|   [ ] Μόνο ευρήματα                              |
|   [ ] Μόνο κεραμική                              |
+--------------------------------------------------+
| [Open Folder]           [Export]                 |
+--------------------------------------------------+
```

### Επιλογές Εξαγωγής

| Επιλογή | Περιγραφή |
|---------|-----------|
| All images | Εξαγωγή όλου του φωτογραφικού υλικού |
| SU only | Εξαγωγή μόνο εικόνων συνδεδεμένων με ΣΜ |
| Finds only | Εξαγωγή μόνο εικόνων ευρημάτων |
| Pottery only | Εξαγωγή μόνο εικόνων κεραμικής |

## Δομή Εξόδου

### Οργάνωση Φακέλων

Η εξαγωγή δημιουργεί μια ιεραρχική δομή:

```
pyarchinit_image_export/
└── [Όνομα Θέσης] - All images/
    ├── Period - 1/
    │   ├── Phase - 1/
    │   │   ├── SU_001/
    │   │   │   ├── photo_001.jpg
    │   │   │   └── photo_002.jpg
    │   │   └── SU_002/
    │   │       └── photo_003.jpg
    │   └── Phase - 2/
    │       └── SU_003/
    │           └── photo_004.jpg
    └── Period - 2/
        └── ...
```

### Σύμβαση Ονοματοδοσίας

Τα αρχεία διατηρούν το αρχικό τους όνομα, οργανωμένα κατά:
1. **Περίοδος** - Αρχική χρονολογική περίοδος
2. **Φάση** - Αρχική χρονολογική φάση
3. **Οντότητα** - ΣΜ, Εύρημα κ.λπ.

## Διαδικασία Εξαγωγής

### Βήμα 1: Επιλογή Παραμέτρων

1. Επιλέξτε **Site** από το ComboBox
2. Επιλέξτε **Year** (προαιρετικό)
3. Επιλέξτε **Τύπο Εξαγωγής**

### Βήμα 2: Εκτέλεση

1. Κάντε κλικ στο **"Export"**
2. Περιμένετε την ολοκλήρωση
3. Μήνυμα επιβεβαίωσης

### Βήμα 3: Επαλήθευση

1. Κάντε κλικ στο **"Open Folder"**
2. Επαληθεύστε τη δημιουργηθείσα δομή
3. Ελέγξτε την πληρότητα

## Φάκελος Εξόδου

### Τυπική Διαδρομή

```
~/pyarchinit/pyarchinit_image_export/
```

### Περιεχόμενα

- Φάκελοι οργανωμένοι κατά θέση
- Υποφάκελοι κατά περίοδο/φάση
- Πρωτότυπες εικόνες (χωρίς αλλαγή μεγέθους)

## Φίλτρο Έτους

Το ComboBox **Year** επιτρέπει:
- Εξαγωγή μόνο εικόνων από συγκεκριμένη ανασκαφική περίοδο
- Οργάνωση εξαγωγής κατά έτος ανασκαφής
- Μείωση του μεγέθους εξαγωγής

## Βέλτιστες Πρακτικές

### 1. Πριν την Εξαγωγή

- Επαληθεύστε τις συνδέσεις εικόνων-οντοτήτων
- Ελέγξτε τη χρονολόγηση των ΣΜ
- Βεβαιωθείτε ότι υπάρχει επαρκής χώρος στον δίσκο

### 2. Κατά τη Διάρκεια της Εξαγωγής

- Μην διακόψετε τη διαδικασία
- Περιμένετε το μήνυμα ολοκλήρωσης

### 3. Μετά την Εξαγωγή

- Επαληθεύστε τη δομή φακέλων
- Ελέγξτε την πληρότητα εικόνων
- Δημιουργήστε αντίγραφο ασφαλείας εάν χρειάζεται

## Τυπικές Χρήσεις

### Προετοιμασία Αναφοράς

```
1. Επιλέξτε θέση
2. Εξάγετε όλες τις εικόνες
3. Χρησιμοποιήστε τη δομή για τα κεφάλαια της αναφοράς
```

### Παράδοση στην Εφορεία

```
1. Επιλέξτε θέση και έτος
2. Εξάγετε κατά τον απαιτούμενο τύπο
3. Οργανώστε σύμφωνα με τα πρότυπα
```

### Αντίγραφο Ασφαλείας Ανασκαφικής Περιόδου

```
1. Στο τέλος της ανασκαφής, εξάγετε τα πάντα
2. Αρχειοθετήστε σε εξωτερικό μέσο αποθήκευσης
3. Επαληθεύστε την ακεραιότητα
```

## Αντιμετώπιση Προβλημάτων

### Ατελής Εξαγωγή

**Αιτίες**:
- Μη συνδεδεμένες εικόνες
- Λανθασμένες διαδρομές αρχείων

**Λύσεις**:
- Επαληθεύστε τις συνδέσεις στο Media Manager
- Ελέγξτε την ύπαρξη των αρχείων πηγής

### Λανθασμένη Δομή

**Αιτίες**:
- Ελλιπής χρονολόγηση
- ΣΜ χωρίς περίοδο/φάση

**Λύσεις**:
- Συμπληρώστε τη χρονολόγηση των ΣΜ
- Αντιστοιχίστε περίοδο/φάση σε όλες τις ΣΜ
