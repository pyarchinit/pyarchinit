# Οδηγός 29: Δημιουργία Χάρτη (Make Your Map)

## Εισαγωγή

Η λειτουργία **Make Your Map** του PyArchInit δημιουργεί επαγγελματικούς χάρτες και διατάξεις εκτύπωσης απευθείας από την τρέχουσα προβολή του QGIS. Χρησιμοποιεί προκαθορισμένα πρότυπα διάταξης για τη δημιουργία τυποποιημένων χαρτογραφικών εκτυπώσεων.

### Χαρακτηριστικά

- Ταχεία δημιουργία χάρτη από την τρέχουσα προβολή
- Προκαθορισμένα πρότυπα για διάφορες μορφές
- Προσαρμογή κεφαλίδας και υπομνήματος
- Εξαγωγή σε PDF, PNG, SVG

## Πρόσβαση

### Από τη Γραμμή Εργαλείων
Εικονίδιο **"Make your Map"** (εκτυπωτής) στη γραμμή εργαλείων του PyArchInit

### Από το Μενού
**PyArchInit** → **Make your Map**

## Βασική Χρήση

### Γρήγορη Δημιουργία

1. Ρυθμίστε την επιθυμητή προβολή χάρτη στο QGIS
2. Ορίστε τη σωστή μεγέθυνση και έκταση
3. Κάντε κλικ στο **"Make your Map"**
4. Επιλέξτε το επιθυμητό πρότυπο
5. Εισάγετε τίτλο και πληροφορίες
6. Δημιουργήστε τον χάρτη

## Διαθέσιμα Πρότυπα

### Τυποποιημένες Μορφές

| Πρότυπο | Μορφή | Προσανατολισμός | Χρήση |
|---------|-------|-----------------|-------|
| A4 Κατακόρυφο | A4 | Κατακόρυφος | Τυπική τεκμηρίωση |
| A4 Οριζόντιο | A4 | Οριζόντιος | Εκτεταμένες προβολές |
| A3 Κατακόρυφο | A3 | Κατακόρυφος | Λεπτομερείς πίνακες |
| A3 Οριζόντιο | A3 | Οριζόντιος | Κατόψεις |

### Στοιχεία Προτύπου

Κάθε πρότυπο περιλαμβάνει:
- **Περιοχή χάρτη** - Κύρια προβολή
- **Κεφαλίδα** - Τίτλος και πληροφορίες έργου
- **Κλίμακα** - Γραφική κλίμακα
- **Βορράς** - Βέλος βορρά
- **Υπόμνημα** - Σύμβολα επιπέδων
- **Πινακίδα** - Τεχνικές πληροφορίες

## Προσαρμογή

### Επεξεργάσιμες Πληροφορίες

| Πεδίο | Περιγραφή |
|-------|-----------|
| Title | Όνομα χάρτη |
| Subtitle | Πρόσθετη περιγραφή |
| Site | Όνομα αρχαιολογικής θέσης |
| Area | Αριθμός τομέα |
| Date | Ημερομηνία δημιουργίας |
| Author | Όνομα δημιουργού |
| Scale | Κλίμακα αναπαράστασης |

### Στυλ Χάρτη

Πριν τη δημιουργία:
1. Ρυθμίστε τα στυλ επιπέδων στο QGIS
2. Ενεργοποιήστε/απενεργοποιήστε τα επιθυμητά επίπεδα
3. Ορίστε τις ετικέτες
4. Επαληθεύστε το υπόμνημα

## Εξαγωγή

### Διαθέσιμες Μορφές

| Μορφή | Χρήση | Ποιότητα |
|-------|-------|---------|
| PDF | Εκτύπωση, αρχείο | Διανυσματική |
| PNG | Ιστός, παρουσιάσεις | Ψηφιδωτή |
| SVG | Επεξεργασία, δημοσίευση | Διανυσματική |
| JPG | Ιστός, προεπισκόπηση | Συμπιεσμένη ψηφιδωτή |

### Ανάλυση

| DPI | Χρήση |
|-----|-------|
| 96 | Οθόνη/προεπισκόπηση |
| 150 | Δημοσίευση στον ιστό |
| 300 | Τυπική εκτύπωση |
| 600 | Εκτύπωση υψηλής ποιότητας |

## Ολοκλήρωση με Time Manager

### Δημιουργία Ακολουθίας

Σε συνδυασμό με τον Time Manager:
1. Ρυθμίστε τον Time Manager
2. Για κάθε στρωματογραφικό επίπεδο:
   - Ορίστε το επίπεδο
   - Δημιουργήστε χάρτη
   - Αποθηκεύστε με σταδιακό όνομα

### Έξοδος Κινούμενης Εικόνας

Σειρά χαρτών για:
- Παρουσιάσεις
- Βίντεο time-lapse
- Σταδιακή τεκμηρίωση

## Τυπική Ροή Εργασίας

### 1. Προετοιμασία

```
1. Φορτώστε τα απαραίτητα επίπεδα
2. Ρυθμίστε τα κατάλληλα στυλ
3. Ορίστε το σύστημα αναφοράς συντεταγμένων
4. Καθορίστε την έκταση του χάρτη
```

### 2. Ρύθμιση Προβολής

```
1. Μεγεθύνετε στην περιοχή ενδιαφέροντος
2. Ενεργοποιήστε/απενεργοποιήστε επίπεδα
3. Επαληθεύστε τις ετικέτες
4. Ελέγξτε το υπόμνημα
```

### 3. Δημιουργία

```
1. Κάντε κλικ στο Make your Map
2. Επιλέξτε πρότυπο
3. Συμπληρώστε τις πληροφορίες
4. Επιλέξτε μορφή εξαγωγής
5. Αποθηκεύστε
```

## Βέλτιστες Πρακτικές

### 1. Πριν τη Δημιουργία

- Επαληθεύστε την πληρότητα δεδομένων
- Ελέγξτε τα στυλ επιπέδων
- Ορίστε κατάλληλη κλίμακα

### 2. Πρότυπα

- Χρησιμοποιήστε συνεπή πρότυπα στο έργο
- Προσαρμόστε τις κεφαλίδες στον φορέα σας
- Τηρήστε χαρτογραφικά πρότυπα

### 3. Έξοδος

- Αποθηκεύστε σε υψηλή ανάλυση για εκτύπωση
- Διατηρήστε αντίγραφο PDF για αρχείο
- Χρησιμοποιήστε περιγραφική ονοματολογία

## Προσαρμογή Προτύπων

### Τροποποίηση Προτύπου

Τα πρότυπα QGIS μπορούν να τροποποιηθούν:
1. Ανοίξτε τον Διαχειριστή Διατάξεων στο QGIS
2. Τροποποιήστε το υπάρχον πρότυπο
3. Αποθηκεύστε ως νέο πρότυπο
4. Διαθέσιμο στο Make your Map

### Δημιουργία Προτύπου

1. Δημιουργήστε νέα διάταξη στο QGIS
2. Προσθέστε τα απαραίτητα στοιχεία
3. Ρυθμίστε μεταβλητές για δυναμικά πεδία
4. Αποθηκεύστε στον φάκελο προτύπων

## Αντιμετώπιση Προβλημάτων

### Κενός Χάρτης

**Αιτίες**:
- Κανένα ενεργό επίπεδο
- Λανθασμένη έκταση

**Λύσεις**:
- Ενεργοποιήστε ορατά επίπεδα
- Μεγεθύνετε σε περιοχή με δεδομένα

### Ατελές Υπόμνημα

**Αιτία**: Τα επίπεδα δεν είναι σωστά ρυθμισμένα

**Λύση**: Επαληθεύστε τις ιδιότητες επιπέδων στο QGIS

### Αποτυχία Εξαγωγής

**Αιτίες**:
- Μη εγγράψιμη διαδρομή
- Μη υποστηριζόμενη μορφή

**Λύσεις**:
- Επαληθεύστε τα δικαιώματα φακέλου
- Επιλέξτε διαφορετική μορφή
