# Εκπαιδευτικό 06: Δελτίο Ταφής

## Εισαγωγή

Το **Δελτίο Ταφής** είναι η ενότητα του PyArchInit αφιερωμένη στην τεκμηρίωση αρχαιολογικών ταφών. Αυτό το εργαλείο σας επιτρέπει να καταγράψετε όλες τις πτυχές ενός τάφου: από την ταφική κατασκευή μέχρι το τελετουργικό, από τα κτερίσματα μέχρι τα ενταφιασμένα άτομα.

### Βασικές Έννοιες

**Ταφή στο PyArchInit:**
- Μια ταφή είναι μια ταφική κατασκευή που περιέχει ένα ή περισσότερα άτομα
- Συνδέεται με το Δελτίο Κατασκευής (η φυσική κατασκευή της ταφής)
- Συνδέεται με το Δελτίο Ατόμου (για ανθρωπολογικά δεδομένα)
- Τεκμηριώνει τελετουργικό, κτερίσματα και χαρακτηριστικά εναπόθεσης

**Σχέσεις:**
```
Ταφή → Κατασκευή (φυσικός περιέκτης)
      → Άτομο(α) (ανθρώπινα λείψανα)
      → Κτερίσματα (συνοδευτικά αντικείμενα)
      → Απογραφή Υλικών (συσχετισμένα ευρήματα)
```

## Πεδία Ταυτοποίησης

![Πεδία ταυτοποίησης](images/06_scheda_tomba/06_campi_identificativi.png)

### Θέση
**Πεδίο**: `comboBox_sito` / **Βάση δεδομένων**: `sito`
Επιλέξτε την αρχαιολογική θέση στην οποία ανήκει.

### Τομέας
**Πεδίο**: `comboBox_area` / **Βάση δεδομένων**: `area`
Τομέας ανασκαφής εντός της θέσης.

### Αριθμός Δελτίου
**Πεδίο**: `lineEdit_nr_scheda` / **Βάση δεδομένων**: `nr_scheda_taf`
Προοδευτικός αριθμός δελτίου ταφής. Ο επόμενος διαθέσιμος αριθμός προτείνεται αυτόματα.

### Κωδικός και Αριθμός Κατασκευής

| Πεδίο | Βάση δεδομένων | Περιγραφή |
|-------|----------------|-----------|
| Structure code | `sigla_struttura` | Κωδικός κατασκευής (π.χ., TM, TB) |
| Structure no. | `nr_struttura` | Αριθμός κατασκευής |

Αυτά τα πεδία συνδέουν την ταφή με το αντίστοιχο Δελτίο Κατασκευής.

### Αριθμός Ατόμου
**Πεδίο**: `comboBox_nr_individuo` / **Βάση δεδομένων**: `nr_individuo`
Αριθμός του ενταφιασμένου ατόμου. Συνδέει την ταφή με το Δελτίο Ατόμου.

**Πεδίο**: `comboBox_tipo_sepoltura` / **Βάση δεδομένων**: `tipo_sepoltura`

**Παραδείγματα:**
- Απλός λακκοειδής τάφος
- Κιβωτιόσχημος τάφος
- Θαλαμοειδής τάφος
- Τάφος καλυβίτης (cappuccina)
- Εγχυτρισμός
- Σαρκοφάγος
- Οστεοθήκη

### Τύπος Εναπόθεσης
- Πρωτογενής (άμεση εναπόθεση)
- Δευτερογενής (ανακομιδή/μετακίνηση)
- Ταυτόχρονη πολλαπλή
- Διαδοχική πολλαπλή

### Κατάσταση Διατήρησης
- Εξαιρετική
- Καλή
- Μέτρια
- Κακή
- Πολύ κακή

### Περιγραφή
**Πεδίο**: `textEdit_descrizione` / **Βάση δεδομένων**: `descrizione_taf`

**Προτεινόμενο περιεχόμενο:**
- Σχήμα και διαστάσεις λάκκου
- Προσανατολισμός
- Βάθος
- Χαρακτηριστικά πλήρωσης
- Κατάσταση κατά τη στιγμή της ανασκαφής

## Ροή Εργασίας

### Δημιουργία Νέας Ταφής

1. **Άνοιγμα δελτίου** - Μέσω μενού ή γραμμής εργαλείων

2. **Νέα εγγραφή** - Κάντε κλικ στο "New Record"

3. **Δεδομένα ταυτοποίησης**
   ```
   Site: Νεκρόπολη Isola Sacra
   Area: 1
   Form No.: 45
   Structure code: TM
   Structure no.: 45
   ```

4. **Σύνδεση ατόμου**
   ```
   Individual No.: 1
   ```

5. **Περιγραφικά δεδομένα** (Καρτέλα 1)
   ```
   Rite: Ενταφιασμός
   Burial type: Απλός λακκοειδής τάφος
   Deposition type: Πρωτογενής
   Conservation status: Καλή
   ```

6. **Κτερίσματα** (Καρτέλα 2)
   ```
   Presence: Ναι
   Type: Προσωπικά
   ```

7. **Περιοδολόγηση**
   ```
   Initial period: II - Phase A
   Dating: 2ος αιώνας μ.Χ.
   ```

8. **Αποθήκευση** - Κάντε κλικ στο "Save"

## Σχέσεις με Άλλα Δελτία

| Δελτίο | Σχέση |
|--------|-------|
| **Δελτίο Θέσης** | Η θέση περιέχει ταφές |
| **Δελτίο Κατασκευής** | Φυσική κατασκευή της ταφής |
| **Δελτίο Ατόμου** | Ανθρώπινα λείψανα στην ταφή |
| **Δελτίο Απογραφής Υλικών** | Ευρήματα κτερισμάτων |
| **Δελτίο Περιοδολόγησης** | Χρονολογία |

### Προτεινόμενη Ροή Εργασίας

1. Δημιουργήστε **Δελτίο Θέσης** (αν δεν υπάρχει)
2. Δημιουργήστε **Δελτίο Κατασκευής** για την ταφή
3. Δημιουργήστε **Δελτίο Ταφής** συνδέοντάς το με την κατασκευή
4. Δημιουργήστε **Δελτίο Ατόμου** για κάθε άτομο
5. Καταγράψτε τα κτερίσματα στο **Δελτίο Απογραφής Υλικών**

---

## Βέλτιστες Πρακτικές

### Ονοματοδοσία
- Χρησιμοποιήστε συνεπείς κωδικούς (TM, TB, SEP)
- Προοδευτική αρίθμηση εντός θέσης

### Περιγραφή
- Περιγράψτε συστηματικά σχήμα, διαστάσεις, προσανατολισμό
- Τεκμηριώστε κατάσταση κατά τη στιγμή ανασκαφής

### Κτερίσματα
- Καταγράψτε ακριβή θέση κάθε αντικειμένου
- Συνδέστε κάθε στοιχείο με Απογραφή Υλικών

---

## Αντιμετώπιση Προβλημάτων

### Πρόβλημα: Το άτομο δεν είναι διαθέσιμο στο μενού
**Λύση**: Δημιουργήστε πρώτα το Δελτίο Ατόμου και συσχετίστε το με την ίδια κατασκευή

### Πρόβλημα: Τα κτερίσματα δεν εμφανίζονται στον πίνακα
**Λύση**: Επαληθεύστε ότι τα ευρήματα στο Δελτίο Απογραφής Υλικών έχουν τη σωστή κατασκευή
