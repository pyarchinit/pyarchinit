# Εκπαιδευτικό 13: Διαχείριση Θησαυρού

## Εισαγωγή

Ο **Θησαυρός** στο PyArchInit είναι το κεντρικό σύστημα διαχείρισης ελεγχόμενων λεξιλογίων. Σας επιτρέπει να ορίσετε και να συντηρήσετε τους καταλόγους τιμών που χρησιμοποιούνται σε όλα τα δελτία του πρόσθετου, διασφαλίζοντας ορολογική συνέπεια και διευκολύνοντας την αναζήτηση.

### Κύριες Λειτουργίες

- Διαχείριση λεξιλογίων για κάθε δελτίο
- Πολυγλωσσική υποστήριξη
- Κωδικοί και εκτεταμένες περιγραφές
- Ενσωμάτωση GPT για προτάσεις
- Εισαγωγή/εξαγωγή αρχείων CSV

---

## Πρόσβαση στον Θησαυρό

### Μέσω Μενού
1. Μενού **PyArchInit** στη γραμμή μενού του QGIS
2. Επιλέξτε **Thesaurus** (ή **Thesaurus form**)

### Μέσω Γραμμής Εργαλείων
1. Εντοπίστε τη γραμμή εργαλείων του PyArchInit
2. Κάντε κλικ στο εικονίδιο **Thesaurus** (βιβλίο/λεξικό)

---

## Πεδία Θησαυρού

### Όνομα Πίνακα

**Πεδίο**: `comboBox_nome_tabella`
**Βάση δεδομένων**: `nome_tabella`

Επιλέξτε το δελτίο για το οποίο θα ορίσετε τιμές.

**Διαθέσιμοι πίνακες:**
| Πίνακας | Περιγραφή |
|---------|-----------|
| `us_table` | Δελτίο ΣΕ/ΤΣΕ |
| `site_table` | Δελτίο Θέσης |
| `periodizzazione_table` | Περιοδολόγηση |
| `inventario_materiali_table` | Καταγραφή Υλικών |
| `pottery_table` | Δελτίο Κεραμικής |
| `campioni_table` | Δελτίο Δειγμάτων |
| `documentazione_table` | Τεκμηρίωση |
| `tomba_table` | Δελτίο Τάφου |
| `individui_table` | Δελτίο Ατόμου |
| `fauna_table` | Αρχαιοζωολογία |
| `ut_table` | Δελτίο ΤΜ |

### Κωδικός

**Πεδίο**: `comboBox_sigla`
**Βάση δεδομένων**: `sigla`

Σύντομος κωδικός/συντομογραφία για τον όρο.

**Παραδείγματα:**
- `WL` για Τοίχο (Wall)
- `SU` για Στρωματογραφική Ενότητα (Stratigraphic Unit)
- `CR` για Κεραμική (Ceramic)

### Εκτεταμένος Κωδικός

**Πεδίο**: `comboBox_sigla_estesa`
**Βάση δεδομένων**: `sigla_estesa`

Πλήρης μορφή του όρου.

**Παραδείγματα:**
- `Perimeter wall`
- `Stratigraphic Unit`
- `Common ware`

### Περιγραφή

**Πεδίο**: `textEdit_descrizione_sigla`
**Βάση δεδομένων**: `descrizione`

Λεπτομερής περιγραφή του όρου, ορισμός, σημειώσεις χρήσης.

### Τύπος Κωδικού

**Πεδίο**: `comboBox_tipologia_sigla`
**Βάση δεδομένων**: `tipologia_sigla`

Αριθμητικός κωδικός που προσδιορίζει το πεδίο προορισμού.

**Δομή κωδικού:**
```
X.Y όπου:
X = αριθμός πίνακα
Y = αριθμός πεδίου
```

**Παραδείγματα για us_table:**
| Κωδικός | Πεδίο |
|---------|-------|
| 1.1 | Στρωματογραφικός ορισμός |
| 1.2 | Τρόπος σχηματισμού |
| 1.3 | Τύπος ΣΕ |

### Γλώσσα

**Πεδίο**: `comboBox_lingua`
**Βάση δεδομένων**: `lingua`

Γλώσσα του όρου.

**Υποστηριζόμενες γλώσσες:**
- IT (Ιταλικά)
- EN_US (Αγγλικά)
- DE (Γερμανικά)
- FR (Γαλλικά)
- ES (Ισπανικά)
- AR (Αραβικά)
- CA (Καταλανικά)

## Ειδικές Λειτουργίες

### Προτάσεις GPT

Το κουμπί «Suggestions» χρησιμοποιεί το OpenAI GPT για:
- Δημιουργία αυτόματων περιγραφών
- Παροχή συνδέσμων αναφοράς στη Wikipedia
- Πρόταση ορισμών σε αρχαιολογικό πλαίσιο

**Χρήση:**
1. Επιλέξτε ή εισαγάγετε έναν όρο στον «Εκτεταμένο κωδικό»
2. Κάντε κλικ στο «Suggestions»
3. Επιλέξτε μοντέλο GPT
4. Περιμένετε τη δημιουργία
5. Ελέγξτε και αποθηκεύστε

**Σημείωση:** Απαιτεί ρυθμισμένο κλειδί API OpenAI.

### Εισαγωγή CSV

Για βάση δεδομένων SQLite, τα λεξιλόγια μπορούν να εισαχθούν από αρχεία CSV.

**Απαιτούμενη μορφή CSV:**
```csv
nome_tabella,sigla,sigla_estesa,descrizione,tipologia_sigla,lingua
us_table,WL,Wall,Wall structure,1.3,EN_US
us_table,FL,Floor,Walking surface,1.3,EN_US
```

**Διαδικασία:**
1. Κάντε κλικ στο «Import CSV»
2. Επιλέξτε αρχείο
3. Επιβεβαιώστε την εισαγωγή
4. Επαληθεύστε τα εισαγόμενα δεδομένα

## Ροή Εργασίας

### Προσθήκη Νέου Όρου

1. **Ανοίξτε τον Θησαυρό**
   - Μέσω μενού ή γραμμής εργαλείων

2. **Νέα εγγραφή**
   - Κάντε κλικ στο «New Record»

3. **Επιλογή πίνακα**
   ```
   Table name: us_table
   ```

4. **Ορισμός όρου**
   ```
   Code: WE
   Extended code: Well
   Code type: 1.3
   Language: EN_US
   ```

5. **Περιγραφή**
   ```
   Structure dug into the ground for
   water supply. Generally circular
   in shape with stone or brick lining.
   ```

6. **Αποθήκευση**
   - Κάντε κλικ στο «Save»

### Αναζήτηση Όρων

1. Κάντε κλικ στο «New Search»
2. Συμπληρώστε τα κριτήρια:
   - Όνομα πίνακα
   - Κωδικός ή εκτεταμένος κωδικός
   - Γλώσσα
3. Κάντε κλικ στο «Search»
4. Πλοηγηθείτε στα αποτελέσματα

### Τροποποίηση Υπάρχοντος Όρου

1. Αναζητήστε τον όρο προς τροποποίηση
2. Τροποποιήστε τα απαραίτητα πεδία
3. Κάντε κλικ στο «Save»

## Βέλτιστες Πρακτικές

### Ορολογική Συνέπεια

- Χρησιμοποιείτε πάντα τους ίδιους όρους για τις ίδιες έννοιες
- Αποφεύγετε μη τεκμηριωμένα συνώνυμα

- Δημιουργήστε όρους σε όλες τις απαραίτητες γλώσσες

### Ιεραρχία

- Χρησιμοποιήστε ιεραρχική δομή για σχετιζόμενους όρους
- Ορίστε σαφώς τα επίπεδα
- Τεκμηριώστε τις σχέσεις

### Συντήρηση

- Ελέγχετε περιοδικά τα λεξιλόγια
- Αφαιρέστε παρωχημένους όρους
- Ενημερώστε τις περιγραφές

## Αντιμετώπιση Προβλημάτων

### Πρόβλημα: Ο όρος δεν εμφανίζεται στα ComboBoxes

**Αιτία:** Εσφαλμένος κωδικός τύπου ή μη αντιστοιχούσα γλώσσα.

**Λύση:**
1. Επαληθεύστε τον κωδικό tipologia_sigla
2. Ελέγξτε την ορισμένη γλώσσα
3. Βεβαιωθείτε ότι η εγγραφή έχει αποθηκευτεί

### Πρόβλημα: Αποτυχία εισαγωγής CSV

**Αιτία:** Εσφαλμένη μορφή αρχείου.

**Λύση:**
1. Επαληθεύστε τη δομή CSV
2. Ελέγξτε τα διαχωριστικά (κόμμα)
3. Επαληθεύστε την κωδικοποίηση (UTF-8)

### Πρόβλημα: Οι προτάσεις GPT δεν λειτουργούν

**Αιτία:** Κλειδί API ελλιπές ή μη έγκυρο.

**Λύση:**
1. Επαληθεύστε τη ρύθμιση κλειδιού API
2. Ελέγξτε τη σύνδεση στο διαδίκτυο
3. Επαληθεύστε το πιστωτικό υπόλοιπο OpenAI
