# الدرس 06: بطاقة القبر

## المقدمة

**بطاقة القبر** هي وحدة PyArchInit المخصصة لتوثيق المدافن الأثرية. تتيح هذه الأداة تسجيل جميع جوانب القبر: من الهيكل الجنائزي إلى الطقوس، ومن الأثاث الجنائزي إلى الأفراد المدفونين.

### المفاهيم الأساسية

**القبر في PyArchInit:**
- القبر هو هيكل جنائزي يحتوي على فرد واحد أو أكثر
- مرتبط ببطاقة الهيكل (الهيكل المادي للمدفن)
- مرتبط ببطاقة الأفراد (للبيانات الأنثروبولوجية)
- يوثق الطقوس والأثاث الجنائزي وخصائص الدفن

**العلاقات:**
```
القبر → الهيكل (الحاوية المادية)
     → الفرد/الأفراد (الرفات البشرية)
     → الأثاث الجنائزي (أغراض المرافقة)
     → جرد المواد (القطع المرتبطة)
```

![حقول التعريف](images/06_scheda_tomba/06_campi_identificativi.png)

**الحقل**: `comboBox_sito`
**قاعدة البيانات**: `sito`

### المنطقة

**الحقل**: `comboBox_area`
**قاعدة البيانات**: `area`

### رقم البطاقة

**الحقل**: `lineEdit_nr_scheda`
**قاعدة البيانات**: `nr_scheda_taf`

الرقم التسلسلي لبطاقة القبر. يُقترح تلقائياً الرقم المتاح التالي.

### رمز ورقم الهيكل

| الحقل | قاعدة البيانات | الوصف |
|-------|---------------|-------|

تربط هذه الحقول القبر ببطاقة الهيكل المقابلة.

### رقم الفرد

**الحقل**: `comboBox_nr_individuo`
**قاعدة البيانات**: `nr_individuo`

رقم الفرد المدفون. يربط القبر ببطاقة الأفراد.

**ملاحظات:**
- يمكن أن يحتوي القبر على عدة أفراد (دفن متعدد)
- تعرض القائمة الأفراد المتاحين للهيكل المحدد

## سير العمل التشغيلي

### إنشاء قبر جديد

1. **فتح البطاقة**
   - عبر القائمة أو شريط الأدوات

2. **سجل جديد**
   - النقر على "سجل جديد"
   - يُقترح رقم البطاقة تلقائياً

3. **بيانات التعريف**
   ```
   الموقع: مقبرة إيزولا ساكرا
   المنطقة: 1
   رقم البطاقة: 45
   رمز الهيكل: TM
   رقم الهيكل: 45
   ```

4. **ربط الفرد**
   ```
   رقم الفرد: 1
   ```

5. **البيانات الوصفية** (التبويب 1)
   ```
   الطقس: الدفن
   نوع المدفن: قبر حفرة بسيطة
   نوع الدفن: أولي
   حالة الحفظ: جيد

   الوصف: حفرة مستطيلة بزوايا
   مستديرة، موجهة شرق-غرب...

   العلامات: نصب
   التغطية: قرميد بسقف جملوني
   ```

6. **الأثاث الجنائزي** (التبويب 2)
   ```
   الوجود: نعم
   النوع: شخصي
   الوصف: دبوس برونزي قرب
   الكتف الأيمن، عملة قرب الفم...
   ```

7. **التأريخ**
   ```
   الفترة الأولية: II - المرحلة A
   الفترة النهائية: II - المرحلة A
   التأريخ: القرن الثاني الميلادي
   ```

8. **الحفظ**
   - النقر على "حفظ"

### البحث عن القبور

1. النقر على "بحث جديد"
2. ملء معايير البحث:
   - الموقع
   - الطقس
   - نوع المدفن
   - الفترة
3. النقر على "بحث"
4. التنقل بين النتائج

## العلاقات مع البطاقات الأخرى

| البطاقة | العلاقة |
|---------|---------|
| **بطاقة الموقع** | الموقع يحتوي على القبور |
| **بطاقة الهيكل** | الهيكل المادي للقبر |
| **بطاقة الأفراد** | الرفات البشرية في القبر |
| **بطاقة جرد المواد** | قطع الأثاث الجنائزي |
| **بطاقة التأريخ** | التسلسل الزمني |

### سير العمل الموصى به

1. إنشاء **بطاقة الموقع** (إذا لم تكن موجودة)
2. إنشاء **بطاقة الهيكل** للقبر
3. إنشاء **بطاقة القبر** وربطها بالهيكل
4. إنشاء **بطاقة الأفراد** لكل فرد
5. تسجيل الأثاث الجنائزي في **بطاقة جرد المواد**

## استكشاف الأخطاء وإصلاحها

### المشكلة: الفرد غير متاح في القائمة

**السبب**: لم يُنشأ الفرد بعد أو غير مرتبط بالهيكل.

**الحل**:
1. التحقق من وجود بطاقة الأفراد
2. التحقق من ارتباط الفرد بنفس الهيكل

### المشكلة: الأثاث الجنائزي لا يظهر في الجدول

**السبب**: القطع غير مرتبطة بالهيكل.

**الحل**:
1. فتح بطاقة جرد المواد
2. التحقق من أن القطع لها الهيكل الصحيح
3. تحديث بطاقة القبر

### المشكلة: القبر غير مرئي على الخريطة

**السبب**: الهندسة غير مرتبطة.

**الحل**:
1. التحقق من وجود طبقة القبر
2. التحقق من أن الهيكل له هندسة
3. التحقق من النظام المرجعي
