# الدرس 15: بطاقة علم آثار الحيوان (الحيوانات)

## المقدمة

**بطاقة علم آثار الحيوان/الحيوانات** (بطاقة FR - Fauna Record) هي وحدة PyArchInit المخصصة لتحليل وتوثيق البقايا الحيوانية. تتيح تسجيل بيانات أثرية حيوانية مفصلة لدراسة اقتصاديات المعيشة القديمة.

### المفاهيم الأساسية

**علم آثار الحيوان:**
- دراسة البقايا الحيوانية من السياقات الأثرية
- تحليل العلاقات بين الإنسان والحيوان في الماضي
- إعادة بناء الحميات الغذائية، التربية، الصيد

**البيانات المسجلة:**
- التعريف التصنيفي (الأنواع)
- الأجزاء الهيكلية الموجودة
- NMI (الحد الأدنى لعدد الأفراد)
- حالة الحفظ
- آثار التحلل
- علامات الذبح

---

## سير العمل التشغيلي

### توثيق البقايا الحيوانية

1. **فتح البطاقة**
   - عبر القائمة أو شريط الأدوات

2. **سجل جديد**
   - النقر على "سجل جديد"

3. **بيانات التعريف**
   ```
   الموقع: الفيلا الرومانية
   المنطقة: 1000
   الوحدة الطبقية: 150
   المسؤول: ج. أخضر
   التاريخ: 20/07/2024
   ```

4. **البيانات الأثرية الحيوانية** (التبويب 2)
   ```
   السياق: نفايات/مكب
   الأنواع: Bos taurus
   NMI: 3

   الأجزاء الهيكلية:
   - العضد / أيمن / 2 / 1
   - الظنبوب / أيسر / 3 / 2
   - ميتابودي / - / 5 / 1
   ```

5. **البيانات التحللية** (التبويب 3)
   ```
   التجزئة: مجزأ
   الحفظ: جيد
   الاحتراق: غائب
   العلامات التحللية: علامات الجذور
   ```

6. **التفسير**
   ```
   نفايات غذائية.
   وجود آثار ذبح
   على بعض العظام الطويلة.
   ```

7. **الحفظ**
   - النقر على "حفظ"

## أفضل الممارسات

### التعريف

- استخدام مجموعات مرجعية
- تحديد درجة اليقين في التعريف
- تسجيل البقايا غير القابلة للتعريف أيضاً

### NMI

- الحساب لكل نوع بشكل منفصل
- مراعاة الجانب والعمر للقطع
- توثيق طريقة الحساب

### التحلل

- ملاحظة كل قطعة بشكل منهجي
- توثيق الآثار قبل الغسل
- تصوير الحالات المهمة

### السياق

- الربط دائماً بالوحدة الطبقية المصدرية
- تسجيل طريقة الاسترداد
- تدوين الارتباطات المهمة

## استكشاف الأخطاء وإصلاحها

### المشكلة: الأنواع غير متاحة

**السبب**: قائمة الأنواع غير مكتملة.

**الحل**:
1. التحقق من قاموس الحيوانات
2. إضافة الأنواع المفقودة
3. الاتصال بالمسؤول

### المشكلة: NMI غير محسوب

**السبب**: الأجزاء الهيكلية غير مسجلة.

**الحل**:
1. ملء جدول الأجزاء الهيكلية
2. تحديد الجانب والكمية
3. سيحسب النظام تلقائياً
