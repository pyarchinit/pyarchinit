# الدرس 21: بطاقة الوحدة الطبوغرافية (UT)

## المقدمة

**بطاقة الوحدة الطبوغرافية** (UT) هي وحدة PyArchInit المخصصة لتوثيق المسوحات الأثرية السطحية (survey). تتيح تسجيل البيانات المتعلقة بتركيزات المواد، وشذوذات التضاريس، والمواقع المُحددة أثناء عمليات المسح.

### المفاهيم الأساسية

**الوحدة الطبوغرافية (UT):**
- منطقة محددة ذات خصائص أثرية متجانسة
- مُحددة أثناء المسح السطحي
- معرفة بتركيز المواد أو الشذوذات المرئية

**المسح (Survey):**
- استكشاف منهجي للإقليم
- جمع بيانات عن الوجود البشري القديم
- توثيق دون حفر

## سير العمل التشغيلي

### تسجيل UT جديدة

1. **فتح البطاقة**
   - عبر القائمة أو شريط الأدوات

2. **سجل جديد**
   - النقر على "سجل جديد"

3. **بيانات التعريف**
   ```
   المشروع: مسح وادي النهر 2024
   رقم UT: 25
   ```

4. **الموقع**
   ```
   المنطقة: القاهرة
   المحافظة: الجيزة
   البلدية: الهرم
   المكان: التل الشرقي
   الإحداثيات: 29.9792° N, 31.1342° E
   الارتفاع: 125 م
   ```

5. **الوصف**
   ```
   التعريف: تركيز مواد
   الوصف: منطقة بيضاوية حوالي 50×30 م
   مع تركيز شظايا فخارية
   وآجر على منحدر تلة مواجه
   للجنوب...
   ```

6. **بيانات المسح**
   ```
   الطريقة: مسح منهجي
   الرؤية: 80%
   الحالة: محروث حديثاً
   التاريخ: 15/04/2024
   المسؤول: الفريق أ
   ```

7. **المواد والتسلسل الزمني**
   ```
   الأبعاد: 1500 م²
   القطع/م²: 5-8
   القطع المؤرخة: فخار عادي،
   سيجيلاتا إيطالية، آجر

   الفترة I: روماني
   التأريخ I: القرن I-II م
   التفسير I: فيلا ريفية
   ```

8. **الحفظ**
   - النقر على "حفظ"

## تكامل GIS

بطاقة UT متكاملة بشكل وثيق مع QGIS:

- **طبقة UT**: عرض الهندسات
- **السمات المرتبطة**: بيانات من البطاقة
- **الاختيار من الخريطة**: النقر على الهندسة يفتح البطاقة

- تقدير الكثافة لكل منطقة
- توثيق التوزيع المكاني

## استكشاف الأخطاء وإصلاحها

### المشكلة: إحداثيات غير صالحة

**السبب**: صيغة خاطئة أو نظام مرجعي.

**الحل**:
1. التحقق من الصيغة (DD أو DMS)
2. فحص النظام المرجعي
3. استخدام أداة تحويل QGIS

### المشكلة: UT غير مرئي على الخريطة

**السبب**: الهندسة غير مرتبطة.

**الحل**:
1. التحقق من وجود طبقة UT
2. فحص أن السجل له هندسة
3. التحقق من إسقاط الطبقة
