# الدرس 24: تصدير الصور

## المقدمة

وظيفة **تصدير الصور** تتيح تصدير الصور المرتبطة بالسجلات الأثرية بشكل جماعي، مع تنظيمها تلقائياً في مجلدات حسب الفترة والمرحلة ونوع الكيان.

## الوصول

### من القائمة
**PyArchInit** ← **تصدير الصور**

## الواجهة

### لوحة التصدير

```
+--------------------------------------------------+
|           تصدير الصور                             |
+--------------------------------------------------+
| الموقع: [قائمة منسدلة اختيار الموقع]             |
| السنة: [قائمة منسدلة سنة الحفر]                  |
+--------------------------------------------------+
| نوع التصدير:                                      |
|   [o] جميع الصور                                 |
|   [ ] الوحدات الطبقية فقط                        |
|   [ ] القطع فقط                                  |
|   [ ] الفخار فقط                                 |
+--------------------------------------------------+
| [فتح المجلد]           [تصدير]                   |
+--------------------------------------------------+
```

### خيارات التصدير

| الخيار | الوصف |
|--------|-------|
| جميع الصور | تصدير كل المواد الفوتوغرافية |
| الوحدات الطبقية فقط | تصدير الصور المرتبطة بالوحدات الطبقية فقط |
| القطع فقط | تصدير صور القطع فقط |
| الفخار فقط | تصدير صور السيراميك فقط |

## بنية المخرجات

### تنظيم المجلدات

التصدير يُنشئ بنية هرمية:

```
pyarchinit_image_export/
└── [اسم الموقع] - جميع الصور/
    ├── الفترة - 1/
    │   ├── المرحلة - 1/
    │   │   ├── US_001/
    │   │   │   ├── foto_001.jpg
    │   │   │   └── foto_002.jpg
    │   │   └── US_002/
    │   │       └── foto_003.jpg
    │   └── المرحلة - 2/
    │       └── US_003/
    │           └── foto_004.jpg
    └── الفترة - 2/
        └── ...
```

### اتفاقية التسمية

الملفات تحتفظ بالاسم الأصلي، منظمة حسب:
1. **الفترة** - الفترة الزمنية الأولية
2. **المرحلة** - المرحلة الزمنية الأولية
3. **الكيان** - الوحدة الطبقية، القطعة، إلخ.

## عملية التصدير

### الخطوة 1: اختيار المعاملات

1. اختيار **الموقع** من القائمة المنسدلة
2. اختيار **السنة** (اختياري)
3. اختيار **نوع التصدير**

### الخطوة 2: التنفيذ

1. النقر على **"تصدير"**
2. انتظار الاكتمال
3. رسالة التأكيد

### الخطوة 3: التحقق

1. النقر على **"فتح المجلد"**
2. التحقق من البنية المُنشأة
3. فحص الاكتمال

## مجلد المخرجات

### المسار القياسي

```
~/pyarchinit/pyarchinit_image_export/
```

### المحتوى

- مجلدات منظمة حسب الموقع
- مجلدات فرعية حسب الفترة/المرحلة
- الصور الأصلية (غير مُصغرة)

## التصفية حسب السنة

القائمة المنسدلة **السنة** تتيح:
- تصدير صور حملة محددة فقط
- تنظيم التصدير حسب سنة الحفر
- تقليل حجم التصدير

## أفضل الممارسات

### 1. قبل التصدير

- التحقق من روابط الصور-الكيانات
- فحص تأريخ الوحدات الطبقية
- التأكد من توفر مساحة كافية على القرص

### 2. أثناء التصدير

- عدم إيقاف العملية
- انتظار رسالة الاكتمال

### 3. بعد التصدير

- التحقق من بنية المجلدات
- فحص اكتمال الصور
- إنشاء نسخة احتياطية إذا لزم الأمر

## الاستخدامات النموذجية

### تحضير التقرير

```
1. اختيار الموقع
2. تصدير جميع الصور
3. استخدام البنية لفصول التقرير
```

### التسليم للجهات الرسمية

```
1. اختيار الموقع والسنة
2. التصدير حسب النوع المطلوب
3. التنظيم وفق المعايير الوزارية
```

### نسخة احتياطية للحملة

```
1. في نهاية الحملة، تصدير كل شيء
2. الأرشفة على وسيط تخزين خارجي
3. التحقق من السلامة
```

## حل المشكلات

### التصدير غير مكتمل

**الأسباب**:
- صور غير مرتبطة
- مسارات ملفات خاطئة

**الحلول**:
- التحقق من الروابط في مدير الوسائط
- فحص وجود الملفات المصدرية

### البنية غير صحيحة

**الأسباب**:
- التأريخ مفقود
- الوحدات الطبقية بدون فترة/مرحلة

**الحلول**:
- ملء تأريخ الوحدات الطبقية
- تعيين الفترة/المرحلة لجميع الوحدات
