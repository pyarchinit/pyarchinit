## Note Tecniche

### Particolarita dell'Interfaccia
- La scheda ha collegamento automatico con Struttura
- Il numero individuo viene proposto automaticamente
- I valori dei ComboBox provengono dal thesaurus

### Elementi da Evidenziare
- Collegamento Individuo-Struttura-Tomba
- Campi per determinazione sesso ed eta
- Documentazione completa della posizione
- Tab resti osteologici per inventario anatomico

### Terminologia Antropologica
Assicurarsi che gli screenshot mostrino correttamente:
- Classi di eta standard (Infans I/II, Juvenis, Adultus, Maturus, Senilis)
- Posizioni anatomiche corrette
- Orientamenti cardinali
