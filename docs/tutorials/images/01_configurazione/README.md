## Suggerimenti per Screenshot

1. **Risoluzione schermo**: Usare 1920x1080 o superiore
2. **Tema QGIS**: Usare tema chiaro per migliore leggibilit√†
3. **Dati esempio**: Popolare i campi con dati realistici
4. **Evidenziazione**: Usare frecce rosse per indicare elementi importanti
5. **Crop**: Ritagliare solo l'area rilevante, non tutto lo schermo
