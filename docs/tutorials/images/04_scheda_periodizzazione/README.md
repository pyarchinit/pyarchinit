  2. Indicatori Stato
  3. Dati Identificativi (Sito, Periodo, Fase, Codice)
  4. Dati Descrittivi
  5. Cronologia
  6. Datazione Estesa

- Mostrare i comboBox con valori selezionati
- Esempio: Periodo = 2, Fase = 1

### Per la Datazione Estesa (10)
- Catturare il comboBox aperto
- Mostrare alcune epoche storiche visibili
- Esempio: "Eta Medievale - XII secolo (1101-1200)"

### Per la Mappa GIS (14)
- Caricare le US per un periodo specifico
- Mostrare layer con legenda visibile
- Usare colori diversi per diversi periodi

- Usare dati di esempio realistici:
  - Sito: "Villa Romana di Settefinestre"
  - Periodo: 2
  - Fase: 1
  - Codice Periodo: 201
  - Cronologia: 1101-1200
  - Datazione: "Eta Medievale - XII secolo"
  - Descrizione: "Fase di abbandono e spoliazione delle strutture romane..."
