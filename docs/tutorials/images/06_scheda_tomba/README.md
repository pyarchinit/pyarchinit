## Suggerimenti

### Per la Vista Completa (04)
- Catturare l'intera finestra della scheda
- Numerare le aree principali:
  1. Toolbar DBMS
  2. DB Info
  3. Campi identificativi
  4. Campo individuo
  5. Area tab

### Per i ComboBox (09-12, 16-17)
- Mostrare i menu aperti con tutte le opzioni visibili
- Evidenziare l'opzione selezionata

### Per il Tab Corredo (15-19)
- Mostrare la tabella con almeno 3-4 elementi di corredo
- Esempio dati:
  | ID Reperto | ID Indv. | Materiale | Posizione |
  |------------|----------|-----------|-----------|
  | 1001 | 1 | Fibula bronzo | Presso spalla dx |
  | 1002 | 1 | Moneta | Presso bocca |
  | 1003 | 1 | Lucerna | Presso piedi |
  | 1004 | 1 | Unguentario | Lato sinistro |

### Per il Workflow (27-33)
- Usare dati di esempio realistici:
  - Sito: "Necropoli di Isola Sacra"
  - Area: 1
  - N. Scheda: 45
  - Sigla: TM
  - Nr struttura: 45
  - Nr Individuo: 1
  - Rito: Inumazione
  - Tipo sepoltura: Tomba a fossa semplice
  - Tipo deposizione: Primaria
  - Stato conservazione: Buono

### Per la Mappa (25)
- Tomba ben visibile e selezionata
- Mostrare il layer tomba in legenda
- Zoom adeguato

## Note Tecniche

### Particolarita dell'Interfaccia
- La scheda ha collegamento automatico con Struttura
- Il numero scheda viene proposto automaticamente
- La tabella corredo si popola dai reperti della struttura

### Elementi da Evidenziare
- Collegamento Tomba-Struttura-Individuo
- Tabella corredo con collegamento a Inventario
- Campi periodizzazione collegati alla scheda Periodizzazione
