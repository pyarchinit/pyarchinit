## Suggerimenti

### Per la Vista Completa (04)
- Catturare l'intera finestra della scheda
- Numerare le aree principali:
  1. Header con toolbar
  2. Campi identificativi
  3. Campi classificazione
  4. Area tab

### Per i Dati Quantitativi (10-11)
- Mostrare la tabella elementi con almeno 3 righe compilate
- Esempio:
  - Orlo / Frammento / 2
  - Parete / Frammento / 15
  - Fondo / Frammento / 1

### Per le Quantificazioni (18-19)
- Eseguire una ricerca con almeno 20 reperti
- Generare grafico per "Classe materiale"
- Catturare sia il pannello che il grafico risultante

### Per il Workflow (23-30)
- Usare dati di esempio realistici:
  - Sito: "Villa Romana di Settefinestre"
  - N. Inventario: 1001
  - Area: 1000
  - US: 101
  - Tipo reperto: Ceramica
  - Classe: Sigillata africana
  - Definizione: Piatto
  - Tipo: Hayes 50
