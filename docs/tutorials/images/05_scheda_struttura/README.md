## Suggerimenti

### Per la Vista Completa (04)
- Catturare l'intera finestra della scheda
- Numerare le aree principali:
  1. Toolbar DBMS
  2. DB Info
  3. Campi identificativi
  4. Campi classificazione
  5. Area tab

### Per il Tab Elementi Costruttivi (17-19)
- Mostrare entrambe le tabelle affiancate
- Inserire dati realistici:
  - Materiali: Blocchi di calcare, Malta di calce, Ciottoli
  - Elementi: Blocchi squadrati (45), Soglia (1), Base colonna (2)

### Per il Tab Rapporti (14-16)
- Mostrare almeno 3-4 rapporti inseriti
- Esempio:
  | Tipo | Sito | Sigla | Numero |
  |------|------|-------|--------|
  | Si lega a | Villa Romana | MR | 16 |
  | Si lega a | Villa Romana | MR | 17 |
  | Tagliato da | Villa Romana | ST | 5 |

### Per il Workflow (30-38)
- Usare dati di esempio realistici:
  - Sito: "Villa Romana di Settefinestre"
  - Sigla: MR
  - Numero: 15
  - Categoria: Residenziale
  - Tipologia: Muro perimetrale
  - Definizione: Muro in opera incerta
  - Periodo iniziale: I / Fase A
  - Periodo finale: II / Fase B

### Per l'Export (25-27)
- Mostrare il pannello con entrambe le checkbox
- Catturare un esempio di PDF generato
- Mostrare il dialog di conversione Word

### Per GIS (28-29)
- Struttura ben visibile e selezionata sulla mappa
- Layer in legenda con stile appropriato
- Zoom adeguato all'estensione della struttura

## Formato

- **Tipo**: PNG
- **Larghezza max**: 1200px
- **Compressione**: Ottimizzare per web
- **Annotazioni**: Usare frecce rosse o cerchi per evidenziare elementi importanti

## Note Tecniche

### Particolarita dell'Interfaccia
- La scheda ha un pannello floating per l'export (mDockWidget_export)
- Il pannello PDF to Word e separato (mDockWidget_3)
- Entrambi i pannelli sono nascosti di default
- Il tab Map contiene un QgsMapCanvas per l'anteprima

### Elementi da Evidenziare
- Pulsante "map preview" per toggle anteprima
- Pulsante GIS per caricamento layer
- Pulsante "Load all" per caricare tutte le strutture
- Drag & drop per aggiungere media
